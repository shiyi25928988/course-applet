1计算：|2 + 3i| = ______（i 为虚数单位）。
√13。|2 + 3i| = √(2² + 3²) = √13
2已知集合 A = {0, 1, 2}，B = {1, m}，若 B ⊆ A，则实数 m = ______。
0 或 2。
B ⊆ A ⇒ m ∈ A 且 m ≠ 1 ⇒ m = 0 或 m = 2
3函数 y = 2sin(3x - π/6) 的振幅为 ______。
2。正弦函数 y = A sin(ωx + φ) 的振幅为 |A|
4在 (x² + 1/x)^5 的展开式中，x⁴ 的系数为 ______（用数字作答）。
10。
通项 T_(r+1) = C(5, r) (x²)^(5-r) (1/x)^r = C(5, r) x^(10-3r)
令 10 - 3r = 4 ⇒ r = 2
系数 = C(5, 2) = 10
5已知圆锥的底面半径为 1，母线长为 3，则该圆锥的侧面积为 ______。
3π。
圆锥侧面积公式：S = πrl = π × 1 × 3 = 3π
6在等比数列 {aₙ} 中，a₁ = 1，a₄ = 8，则公比 q = ______。
2。
a₄ = a₁q³ ⇒ 8 = 1 × q³ ⇒ q = 2

三、解答题（本大题共5题，满分76分）
19本题满分14分
在 △ABC 中，角 A, B, C 所对的边分别为 a, b, c，已知 cosA = 3/5，cosB = 5/13，且 a = 1。
（1）求 sinC 的值；
（2）求 b 的值。
解：
（1）在 △ABC 中，A + B + C = π ⇒ C = π - A - B
sinC = sin(π - A - B) = sin(A + B) = sinA cosB + cosA sinB
∵ cosA = 3/5 > 0，cosB = 5/13 > 0 ∴ A, B 为锐角
sinA = √(1 - cos²A) = √(1 - 9/25) = 4/5
sinB = √(1 - cos²B) = √(1 - 25/169) = 12/13
故 sinC = (4/5)(5/13) + (3/5)(12/13) = 20/65 + 36/65 = 56/65

（2）由正弦定理 a/sinA = b/sinB
b = a · sinB / sinA = 1 × (12/13) / (4/5) = (12/13) × (5/4) = 15/13
