1导数与单调性（12分）

已知函数 f(x) = x³ - 3x² + 2。求：

(1) f(x) 的单调区间；

(2) f(x) 在区间 [-1, 3] 上的最大值和最小值。

答案：(1) 单调增区间：(-∞,0) 和 (2,+∞)，减区间：(0,2)；(2) 最大值 2，最小值 -2
解析：
(1) f'(x) = 3x² - 6x = 3x(x - 2)
令 f'(x) > 0，得 x < 0 或 x > 2（增区间）
令 f'(x) < 0，得 0 < x < 2（减区间）

(2) f(-1) = -2，f(0) = 2，f(2) = -2，f(3) = 2
∴ 最大值为 2，最小值为 -2
2恒成立问题（12分）

已知 f(x) = x² - 2x + a，若 f(x) ≥ 0 对 x ∈ [1, +∞) 恒成立，求实数 a 的取值范围。

答案：a ≥ 1
解析：
f(x) = (x - 1)² + a - 1，开口向上，对称轴 x = 1
∴ f(x) 在 [1, +∞) 上单调递增
f(x) 的最小值为 f(1) = a - 1
要使 f(x) ≥ 0 恒成立，需 f(x)min ≥ 0
即 a - 1 ≥ 0 ⇒ a ≥ 1
