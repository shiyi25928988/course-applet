一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {1, 2, 3, 4}，B = {3, 4, 5, 6}，则 A ∩ B = ______。
答案：{3, 4}
解析：两个集合的交集是它们共有的元素，A 和 B 共有的元素是 3 和 4。
2计算：(1 + i)² = ______（i 为虚数单位）。
答案：2i
解析：(1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i
3函数 f(x) = 1/√(1 - x) + ln(x + 1) 的定义域为 ______。
答案：(-1, 1)
解析：
要使函数有意义，需满足：
1 - x > 0 ⇒ x < 1；
x + 1 > 0 ⇒ x > -1
故定义域为 (-1, 1)
4在 (x + 1/x)^6 的展开式中，常数项为 ______。
答案：20
解析：
通项公式：T_(r+1) = C(6, r) x^(6-r) (1/x)^r = C(6, r) x^(6-2r)
令 6 - 2r = 0 ⇒ r = 3
常数项 = C(6, 3) = 20
5已知球的表面积为 16π，则该球的体积为 ______。
答案：32π/3
解析：
表面积公式：S = 4πr² = 16π ⇒ r² = 4 ⇒ r = 2
体积公式：V = 4πr³/3 = 4π×8/3 = 32π/3
6在等差数列 {aₙ} 中，a₁ = 2，a₁ + a₂ + a₃ = 12，则 a₆ = ______。
答案：12
解析：
等差数列性质：a₁ + a₂ + a₃ = 3a₂ = 12 ⇒ a₂ = 4
公差 d = a₂ - a₁ = 4 - 2 = 2
a₆ = a₁ + 5d = 2 + 5×2 = 12
二、选择题（本大题共4题，每题5分，满分20分）
13已知 a, b, c ∈ R，则 "a > b" 是 "ac² > bc²" 的（ ）
答案：B
解析：
充分性不成立：当 c = 0 时，ac² = bc² = 0；
必要性成立：ac² > bc² ⇒ c² > 0 ⇒ a > b。
14若函数 f(x) = x² + 2(a - 1)x + 2 在区间 (-∞, 4] 上单调递减，则 a 的取值范围是（ ）
答案：A
解析：
二次函数对称轴：x = -(a - 1) = 1 - a
开口向上，故递减区间为 (-∞, 1 - a]
由题意：4 ≤ 1 - a ⇒ a ≤ -3
