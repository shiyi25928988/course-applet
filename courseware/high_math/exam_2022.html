一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {x | 0 < x < 2}，B = {x | x ≥ 1}，则 A ∪ B = ______。
(0, +∞) 或 {x | x > 0}
2复数 z 满足 z + 2$\overline{z}$ = 3 + 4i（i 为虚数单位），则 z = ______。
1 - 4i。
设 z = a + bi，则 $\overline{z}$ = a - bi
a + bi + 2(a - bi) = 3a - bi = 3 + 4i
∴ 3a = 3, -b = 4 ⇒ a = 1, b = -4
故 z = 1 - 4i
3函数 f(x) = e^x - 1 的反函数 f⁻¹(x) = ______。
ln(x + 1)（x > -1）。
令 y = e^x - 1，则 y + 1 = e^x ⇒ x = ln(y + 1)
互换 x, y 得 f⁻¹(x) = ln(x + 1)，定义域 x > -1
4从 1, 2, 3, 4, 5 中任取两个不同的数，则两数之和为偶数的概率是 ______。
2/5。
总取法：C(5, 2) = 10 种
和为偶数 ⇒ 两数同奇偶：C(3, 2) + C(2, 2) = 3 + 1 = 4 种
概率 = 4/10 = 2/5
5已知向量 |a| = 3，|b| = 2，且 a ⊥ (a - b)，则 a 与 b 的夹角为 ______。
arccos(3/4)。
a ⊥ (a - b) ⇒ a·(a - b) = 0 ⇒ |a|² - a·b = 0
a·b = |a|² = 9
cosθ = (a·b)/(|a||b|) = 9/(3×2) = 3/2（注：3/2 > 1有误，应为 |a|=2, |b|=3，则 cosθ = 4/6 = 2/3）
正确答案：θ = arccos(2/3)
6椭圆 x²/16 + y²/9 = 1 的焦距为 ______。
2√7。
a² = 16, b² = 9 ⇒ c² = a² - b² = 7 ⇒ c = √7
焦距 2c = 2√7
二、选择题（本大题共4题，每题5分，满分20分）
13已知函数 f(x) = log₂x，则 f⁻¹(2) =（ ）
✅ C。
f(x) = log₂x = 2 ⇒ x = 2² = 4
故 f⁻¹(2) = 4（或 f⁻¹(x) = 2^x，f⁻¹(2) = 4）
14若 f(x) 是定义在 R 上的偶函数，且在 (-∞, 0] 上单调递增，则（ ）
✅ A。
f(x) 是偶函数，在 (-∞, 0] 递增 ⇒ 在 [0, +∞) 递减
f(-2) = f(2)
∵ 3 > 2 > 1 ≥ 0 ∴ f(3) < f(2) < f(1)
即 f(3) < f(-2) < f(1)
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知函数 f(x) = ax³ + bx² + cx 在 x = 1 和 x = -1 处有极值，且 f(1) = -1。
（1）求 a, b, c 的值；
（2）求 f(x) 在 [-2, 2] 上的最大值和最小值。
解：
（1）f'(x) = 3ax² + 2bx + c
f(x) 在 x = ±1 处有极值 ⇒ f'(1) = 0, f'(-1) = 0
f'(1) = 3a + 2b + c = 0 ①
f'(-1) = 3a - 2b + c = 0 ②
f(1) = a + b + c = -1 ③
① - ② 得 4b = 0 ⇒ b = 0
① + ② 得 6a + 2c = 0 ⇒ c = -3a
代入③：a + 0 - 3a = -1 ⇒ -2a = -1 ⇒ a = 1/2
∴ c = -3/2
故 a = 1/2, b = 0, c = -3/2

（2）f(x) = (1/2)x³ - (3/2)x
f'(x) = (3/2)x² - 3/2 = (3/2)(x² - 1) = (3/2)(x - 1)(x + 1)
令 f'(x) = 0 ⇒ x = ±1
列表：
x | -2 | (-2,-1) | -1 | (-1,1) | 1 | (1,2) | 2
f'(x) | + | - | + |
f(x) | -1 | ↗ | 1 | ↘ | -1 | ↗ | 1
f(-2) = (1/2)(-8) - (3/2)(-2) = -4 + 3 = -1
f(-1) = (1/2)(-1) - (3/2)(-1) = -1/2 + 3/2 = 1
f(1) = (1/2)(1) - (3/2)(1) = -1
f(2) = (1/2)(8) - (3/2)(2) = 4 - 3 = 1
故 f(x) 在 [-2, 2] 上的最大值为 1，最小值为 -1。
