一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知 i 是虚数单位，则复数 (2 + i)(1 - i) 的模为 ______。
√10。
方法一：(2 + i)(1 - i) = 2 - 2i + i - i² = 3 - i，模 |3 - i| = √(3² + (-1)²) = √10
方法二：|(2 + i)(1 - i)| = |2 + i| × |1 - i| = √5 × √2 = √10
2已知集合 A = {1, 2, 3}，B = {x | (x + 1)(x - 2) < 0, x ∈ Z}，则 A ∩ B = ______。
{1}。
(x + 1)(x - 2) < 0 ⇒ -1 < x < 2，又 x ∈ Z ⇒ x = 0, 1
故 B = {0, 1}，A ∩ B = {1}
3已知函数 f(x) = log₂(x + a)，若 f(2) = 1，则 a = ______。
0。
f(2) = log₂(2 + a) = 1 ⇒ 2 + a = 2¹ = 2 ⇒ a = 0
4从 3 名男同学和 2 名女同学中任选 2 人参加社会实践活动，则选出的 2 人中至少有 1 名女同学的概率为 ______。
7/10。
总选法：C(5, 2) = 10 种
全是男同学的选法：C(3, 2) = 3 种
至少 1 名女同学的概率 = 1 - 3/10 = 7/10
5已知向量 a = (1, 0)，b = (1, 1)，则 a 与 b 的夹角为 ______（用弧度表示）。
π/4。
cosθ = (a·b) / (|a||b|) = (1×1 + 0×1) / (1×√2) = 1/√2 = √2/2
θ ∈ [0, π]，故 θ = π/4
6若等差数列 {aₙ} 的前 n 项和为 Sₙ，且 a₁ = 2，S₃ = 12，则 a₆ = ______。
12。
S₃ = 3a₁ + 3d = 3×2 + 3d = 6 + 3d = 12 ⇒ d = 2
a₆ = a₁ + 5d = 2 + 5×2 = 12
二、选择题（本大题共4题，每题5分，满分20分）
13下列函数中，定义域是 R 且为增函数的是（ ）
✅ B。
A. y = e^(-x) = (1/e)^x 是减函数；
B. y = x³ 的导数 y' = 3x² ≥ 0，在 R 上单调递增；
C. y = ln x 的定义域为 (0, +∞)；
D. y = |x| 在 (-∞, 0) 单调递减，在 (0, +∞) 单调递增。
14若抛物线 y² = 2px（p > 0）的焦点与椭圆 x²/9 + y²/5 = 1 的右焦点重合，则 p =（ ）
4。
椭圆中 a² = 9, b² = 5，故 c² = a² - b² = 4 ⇒ c = 2
右焦点为 (2, 0)
抛物线 y² = 2px 的焦点为 (p/2, 0)
故 p/2 = 2 ⇒ p = 4
三、解答题（本大题共5题，满分76分）
19本题满分14分
如图，在长方体 ABCD-A₁B₁C₁D₁ 中，AB = BC = 2，AA₁ = 1。
（1）求直线 A₁B 与平面 BCC₁B₁ 所成角的大小；
（2）求三棱锥 B₁ - A₁BC 的体积。
解：
（1）∵ 长方体中 A₁B₁ ⊥ 平面 BCC₁B₁
∴ ∠A₁BB₁ 就是直线 A₁B 与平面 BCC₁B₁ 所成的角
在 Rt△A₁B₁B 中，A₁B₁ = AB = 2，BB₁ = AA₁ = 1
tan∠A₁BB₁ = A₁B₁ / BB₁ = 2 / 1 = 2
故直线 A₁B 与平面 BCC₁B₁ 所成角为 arctan 2

（2）V_B₁-A₁BC = V_A₁-BCB₁（换顶点）
S_△BCB₁ = 1/2 × BC × BB₁ = 1/2 × 2 × 1 = 1
A₁ 到平面 BCC₁B₁ 的距离 = A₁B₁ = 2
V = 1/3 × S_△BCB₁ × 高 = 1/3 × 1 × 2 = 2/3
故三棱锥 B₁ - A₁BC 的体积为 2/3
