第三章 导数及其应用

3.1 导数的概念与几何意义

f'(x₀) = limΔx→0 Δy/Δx = limΔx→0 [f(x₀+Δx) - f(x₀)]/Δx
导数的几何意义：函数y=f(x)在x=x₀处的导数f'(x₀)是曲线y=f(x)在点P(x₀, f(x₀))处的切线的斜率。

切线方程：y - f(x₀) = f'(x₀)(x - x₀)

3.2 导数的运算

基本初等函数的导数公式：
C' = 0（C为常数）
(xⁿ)' = nxⁿ⁻¹
(sin x)' = cos x
(cos x)' = -sin x
(eˣ)' = eˣ
(aˣ)' = aˣ ln a
(ln x)' = 1/x
(logax)' = 1/(x ln a)
导数运算法则：
[f(x) ± g(x)]' = f'(x) ± g'(x)
[f(x)·g(x)]' = f'(x)g(x) + f(x)g'(x)
[f(x)/g(x)]' = [f'(x)g(x) - f(x)g'(x)]/[g(x)]² （g(x)≠0）
复合函数求导：y'ₓ' = y'ᵤ · u'ₓ

3.3 导数在研究函数中的应用

函数的单调性：
在区间(a,b)内，f'(x) > 0 ⇒ f(x)单调递增
在区间(a,b)内，f'(x) < 0 ⇒ f(x)单调递减
函数的极值：
f'(x₀) = 0，且x₀左侧f'(x) > 0，右侧f'(x) < 0 ⇒ f(x₀)是极大值
f'(x₀) = 0，且x₀左侧f'(x) < 0，右侧f'(x) > 0 ⇒ f(x₀)是极小值
函数的最值：
求f(x)在[a,b]上的最值：
① 求f(x)在(a,b)内的极值
② 将极值与端点函数值f(a), f(b)比较
③ 最大的是最大值，最小的是最小值

3.4 定积分与微积分基本定理

微积分基本定理：∫ab f(x)dx = F(b) - F(a)，其中F'(x) = f(x)
