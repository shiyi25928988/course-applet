第一章 集合与逻辑

1.1 集合的概念与运算

集合的特性：确定性、互异性、无序性
集合的表示法：列举法、描述法、图示法（Venn图）

常见数集：N（自然数集）、N*或N₊（正整数集）、Z（整数集）、Q（有理数集）、R（实数集）
集合与元素的关系：
属于：a ∈ A
不属于：a ∉ A
集合间的基本关系：
子集：A ⊆ B 或 B ⊇ A
真子集：A ⫋ B
空集：∅，空集是任何集合的子集
若集合A有n个元素，则子集个数2ⁿ，真子集个数2ⁿ - 1

1.2 集合的基本运算

并集：A ∪ B = {x | x ∈ A 或 x ∈ B}
交集：A ∩ B = {x | x ∈ A 且 x ∈ B}
补集：∁UA = {x | x ∈ U 且 x ∉ A}
运算性质：
A ∩ A = A，A ∩ ∅ = ∅
A ∪ A = A，A ∪ ∅ = A
A ∩ ∁UA = ∅，A ∪ ∁UA = U
∁U(∁UA) = A

1.3 常用逻辑用语

充分条件与必要条件：
若 p ⇒ q，则p是q的充分条件，q是p的必要条件
若 p ⇔ q，则p是q的充要条件
全称量词与存在量词：
全称命题：∀x ∈ M，p(x)，否定：∃x ∈ M，¬p(x)
特称命题：∃x ∈ M，p(x)，否定：∀x ∈ M，¬p(x)
