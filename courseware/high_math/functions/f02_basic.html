📝 典型例题

例1已知函数 f(x) = x² - 2x + 3，求在区间 [0, 3] 上的值域。
解：配方得 f(x) = (x-1)² + 2
对称轴 x=1，在 [0,1] 递减，[1,3] 递增
f(1)=2（最小值），f(3)=6（最大值）
∴ 值域为 [2, 6]
例2设 a = log32，b = log52，c = log23，比较大小。
解：c = log23 > 1，a 和 b 都 < 1
a = 1/log23，b = 1/log25，因为 log23 < log25，所以 a > b
∴ c > a > b
