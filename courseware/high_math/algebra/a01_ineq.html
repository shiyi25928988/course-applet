不等式

不等式的性质

① 对称性：a > b ⇔ b < a
② 传递性：a > b, b > c ⇒ a > c
③ 可加性：a > b ⇒ a + c > b + c
④ 可乘性：a > b, c > 0 ⇒ ac > bc；a > b, c < 0 ⇒ ac < bc
⑤ 同向可加：a > b, c > d ⇒ a + c > b + d
⑥ 同向同正可乘：a > b > 0, c > d > 0 ⇒ ac > bd
⑦ 乘方法则：a > b > 0 ⇒ aⁿ > bⁿ（n∈N*, n≥2）
⑧ 开方法则：a > b > 0 ⇒ ⁿ√a > ⁿ√b（n∈N*, n≥2）

一元二次不等式

解法步骤：
① 化标准：ax² + bx + c > 0（a > 0）
② 算判别：Δ = b² - 4ac
③ 求方程根：解方程ax² + bx + c = 0
④ 写解集：大于0取两边，小于0取中间

基本不等式

√(ab) ≤ (a + b)/2 （a > 0, b > 0）
当且仅当a = b时，等号成立
使用条件：一正、二定、三相等

重要变形：
a² + b² ≥ 2ab
ab ≤ [(a + b)/2]²
最值问题：
积定和最小，和定积最大
x > 0, y > 0，若xy = P（定值），则x + y ≥ 2√P
若x + y = S（定值），则xy ≤ S²/4

线性规划

解题步骤：
① 画可行域：画出约束条件表示的平面区域
② 作目标函数线：平移目标函数对应的直线
③ 求最优解：确定目标函数取得最大值或最小值的点
④ 求最值：将最优解代入目标函数
