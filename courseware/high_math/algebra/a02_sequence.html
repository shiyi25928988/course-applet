数列

等差数列

定义：an+1 - an = d（常数）
通项：an = a₁ + (n - 1)d
前n项和：Sn = n(a₁ + an)/2 = na₁ + n(n - 1)d/2
性质：
① an = am + (n - m)d
② 若m + n = p + q，则am + an = ap + aq
③ Sk, S2k - Sk, S3k - S2k 仍成等差数列

等比数列

定义：an+1/an = q（常数，q≠0）
通项：an = a₁ qn-1
前n项和：Sn = a₁(1 - qn)/(1 - q) = (a₁ - anq)/(1 - q) （q≠1）
Sn = na₁ （q = 1）
性质：
① an = am qn-m
② 若m + n = p + q，则am · an = ap · aq
③ Sk, S2k - Sk, S3k - S2k 仍成等比数列（q≠-1或k为奇数）

数列求和方法

① 公式法：直接用等差、等比数列求和公式
② 分组求和：将数列拆成等差、等比或常见数列，分别求和后再合并
③ 裂项相消法：将通项拆成两项之差，正负相消
常见裂项：
1/n(n + 1) = 1/n - 1/(n + 1)
1/n(n + k) = 1/k [1/n - 1/(n + k)]
④ 错位相减法：适用于{anbn}，其中{an}等差，{bn}等比
⑤ 倒序相加法：首尾配对相加（如等差数列求和推导）

递推数列求通项

① an+1 = an + f(n)：累加法
② an+1 = an · f(n)：累乘法
③ an+1 = pan + q：构造等比数列，an+1 + t = p(an + t)
④ an+1 = pan + qn：两边同除以qn+1，转化为类型③
