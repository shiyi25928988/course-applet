复数

复数的概念

z = a + bi （a, b ∈ R）
各部分名称：
a = Re(z)：实部
b = Im(z)：虚部
i：虚数单位，满足i² = -1
分类：
实数：b = 0
虚数：b ≠ 0
纯虚数：a = 0 且 b ≠ 0
复数相等：a + bi = c + di ⇔ a = c 且 b = d
共轭复数：z = a + bi 的共轭复数为 z̄ = a - bi

复数的四则运算

加法：(a + bi) + (c + di) = (a + c) + (b + d)i
减法：(a + bi) - (c + di) = (a - c) + (b - d)i
乘法：(a + bi)(c + di) = (ac - bd) + (ad + bc)i
除法：(a + bi)/(c + di) = [(a + bi)(c - di)]/(c² + d²)

复数的几何意义

复平面：
x轴（实轴）：表示实数
y轴（虚轴）：表示纯虚数（除原点）
点Z(a, b) 一一对应 复数z = a + bi
复数的模：|z| = |a + bi| = √(a² + b²)
模的性质：
|z| = |z̄|
|z₁ · z₂| = |z₁| · |z₂|
|zⁿ| = |z|ⁿ
| |z₁| - |z₂| | ≤ |z₁ + z₂| ≤ |z₁| + |z₂|

常用结论

i的幂次周期性：
i⁴ⁿ = 1，i⁴ⁿ⁺¹ = i，i⁴ⁿ⁺² = -1，i⁴ⁿ⁺³ = -i
i⁴ⁿ + i⁴ⁿ⁺¹ + i⁴ⁿ⁺² + i⁴ⁿ⁺³ = 0
(1 + i)² = 2i，(1 - i)² = -2i
1/(1 + i) = (1 - i)/2，1/(1 - i) = (1 + i)/2
-b + ai = i(a + bi)
