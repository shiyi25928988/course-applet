一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {1, 2, 3, 4}，B = {2, 4, 6, 8}，则 A ∪ B 中元素的个数为 ______。
6。A ∪ B = {1, 2, 3, 4, 6, 8}
2复数 (1 + i)² 的实部为 ______。
0。(1 + i)² = 1 + 2i + i² = 2i，实部为 0
3函数 f(x) = √(x² - 4) 的定义域为 ______。
(-∞, -2] ∪ [2, +∞)。x² - 4 ≥ 0 ⇒ x ≥ 2 或 x ≤ -2
4已知 tanα = 2，则 tan(α + π/4) = ______。
-3。tan(α + π/4) = (tanα + tanπ/4)/(1 - tanα·tanπ/4) = (2 + 1)/(1 - 2×1) = -3
5若双曲线 x²/a² - y²/9 = 1（a > 0）的渐近线方程为 y = ±(3/2)x，则 a = ______。
2。双曲线渐近线方程为 y = ±(3/a)x，故 3/a = 3/2 ⇒ a = 2
6等比数列 {aₙ} 中，a₁ = 1，a₃ = 4，则公比 q = ______。
±2。a₃ = a₁q² ⇒ 4 = q² ⇒ q = ±2
二、选择题（本大题共4题，每题5分，满分20分）
13函数 f(x) = ln(x² - 2x - 8) 的单调递增区间是（ ）
✅ D。
首先定义域：x² - 2x - 8 > 0 ⇒ (x - 4)(x + 2) > 0 ⇒ x < -2 或 x > 4。
令 t = x² - 2x - 8，则 y = lnt 单调递增。
t = x² - 2x - 8 的递增区间为 (1, +∞)，结合定义域得递增区间为 (4, +∞)。
14已知向量 a = (1, 2)，b = (2, -2)，c = (1, λ)，若 c ∥ (2a + b)，则 λ =（ ）
1/2。
2a + b = 2(1, 2) + (2, -2) = (4, 2)
c ∥ (2a + b) ⇒ 4λ - 2×1 = 0 ⇒ λ = 1/2
三、解答题（本大题共5题，满分76分）
19本题满分14分
如图，在正方体 ABCD-A₁B₁C₁D₁ 中，E, F 分别是 AB, BC 的中点。
（1）求证：EF ∥ 平面 A₁C₁D；
（2）求异面直线 EF 与 AD₁ 所成角的大小。
解：
（1）证明：连接 AC
∵ E, F 分别是 AB, BC 的中点 ∴ EF ∥ AC
又正方体中 AA₁ ∥ CC₁ 且 AA₁ = CC₁ ∴ ACC₁A₁ 是平行四边形
∴ AC ∥ A₁C₁ ∴ EF ∥ A₁C₁
又 EF ⊄ 平面 A₁C₁D，A₁C₁ ⊂ 平面 A₁C₁D
∴ EF ∥ 平面 A₁C₁D

（2）∵ EF ∥ A₁C₁，∴ ∠D₁A₁C₁ 或其补角是异面直线 EF 与 AD₁ 所成的角
设正方体棱长为 a，则 A₁D₁ = a，A₁C₁ = √2a，D₁C₁ = a
△A₁D₁C₁ 中，A₁D₁ = D₁C₁ = a，∠A₁D₁C₁ = 90°
∴ ∠D₁A₁C₁ = 45°
即异面直线 EF 与 AD₁ 所成角为 45°（或 π/4）
