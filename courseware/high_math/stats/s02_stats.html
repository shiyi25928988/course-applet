1. 随机抽样

三种基本抽样方法：

抽样方法特点适用范围
简单随机抽样从总体中逐个抽取，每个个体被抽到的概率相等总体个数较少
系统抽样将总体分成均衡的几部分，按预先制定的规则抽取总体个数较多
分层抽样将总体分成互不交叉的层，按比例从各层抽取总体由差异明显的几部分组成
💡 共同点：三种抽样都是等概率抽样，每个个体被抽到的概率都是 n/N，其中 n 是样本容量，N 是总体容量

2. 用样本估计总体

(1) 频率分布直方图：

  • 求极差：最大值 - 最小值
  • 决定组距与组数
  • 将数据分组
  • 列频率分布表
  • 画频率分布直方图
频率 = 频数 / 样本容量
频率分布直方图中，各小长方形的面积 = 频率
各小长方形面积之和 = 1

(2) 数字特征：

平均数：x̄ = (x₁ + x₂ + ... + xₙ) / n

中位数：将数据按大小排列，位于中间位置的数

方差：s² = [(x₁-x̄)² + (x₂-x̄)² + ... + (xₙ-x̄)²] / n

标准差：s = √s²
💡 方差的意义：方差（标准差）越大，数据的离散程度越大，越不稳定；方差越小，数据越集中，越稳定。

3. 变量的相关性

(1) 散点图：表示两个变量之间关系的图形，可以直观判断两个变量是否相关。

(2) 正相关与负相关：

  • 正相关：点散布在从左下角到右上角的区域
  • 负相关：点散布在从左上角到右下角的区域

(3) 线性回归方程：

ŷ = bx + a

其中：
b = [Σ(xᵢ - x̄)(yᵢ - ȳ)] / Σ(xᵢ - x̄)²
a = ȳ - b x̄
💡 重要性质：回归直线必过样本中心点 (x̄, ȳ)

(4) 相关系数 r：

  • r > 0 表示正相关，r < 0 表示负相关
  • |r| 越接近 1，线性相关程度越强
  • |r| 越接近 0，线性相关程度越弱

4. 独立性检验

利用随机变量 K²（卡方）来判断"两个分类变量有关系"的方法称为独立性检验。

2×2 列联表：

y₁y₂总计
x₁aba + b
x₂cdc + d
总计a + cb + dn = a+b+c+d
K² = n(ad - bc)² / [(a+b)(c+d)(a+c)(b+d)]

临界值表（常用）：

P(K² ≥ k)0.0500.0250.0100.001
k3.8415.0246.63510.828
💡 判断规则：
若 K² ≥ 3.841，则有 95% 的把握认为"两个变量有关系"
若 K² ≥ 6.635，则有 99% 的把握认为"两个变量有关系"
