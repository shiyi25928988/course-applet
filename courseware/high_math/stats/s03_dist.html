1. 离散型随机变量的期望与方差

设离散型随机变量 X 的分布列为：

P(X = xᵢ) = pᵢ，i = 1, 2, ..., n
数学期望（均值）：
E(X) = x₁p₁ + x₂p₂ + ... + xₙpₙ = Σxᵢpᵢ
方差：
D(X) = Σ(xᵢ - E(X))² pᵢ
标准差：
σ(X) = √D(X)

期望与方差的性质：

E(aX + b) = aE(X) + b
D(aX + b) = a²D(X)

2. 二项分布

在 n 次独立重复试验中，设事件 A 发生的次数为 X，每次试验中 A 发生的概率为 p，则 X 服从二项分布，记为 X ~ B(n, p)。

二项分布 B(n, p)

分布列：P(X = k) = C(n, k) p^k (1-p)^(n-k)，k = 0, 1, ..., n

期望：E(X) = np

方差：D(X) = np(1-p)

💡 二项分布的识别：n 次独立重复试验，每次只有两种结果，每次概率相同

3. 正态分布

正态分布是最常见、最重要的连续型概率分布，许多自然现象和测量误差都服从正态分布。

正态分布 N(μ, σ²)

参数含义：μ 是均值（期望），σ² 是方差（σ > 0）

概率密度函数：

f(x) = [1/(σ√2π)] · e^[-(x-μ)²/(2σ²)]，x ∈ ℝ

正态曲线的性质：

  • 曲线位于 x 轴上方，与 x 轴不相交
  • 曲线是单峰的，关于直线 x = μ 对称
  • 曲线在 x = μ 处达到峰值
  • 曲线与 x 轴之间的面积为 1
  • σ 越小，曲线越"瘦高"；σ 越大，曲线越"矮胖"
3σ 原则：
P(μ-σ < X ≤ μ+σ) ≈ 68.27%
P(μ-2σ < X ≤ μ+2σ) ≈ 95.45%
P(μ-3σ < X ≤ μ+3σ) ≈ 99.73%
💡 标准正态分布：N(0, 1)，即 μ = 0，σ = 1 的正态分布

4. 超几何分布

在含有 M 件次品的 N 件产品中，任取 n 件，其中恰有 X 件次品，则 X 服从超几何分布。

超几何分布 H(n, M, N)

分布列：P(X = k) = C(M, k) C(N-M, n-k) / C(N, n)

k = 0, 1, 2, ..., m，其中 m = min(M, n)

期望：E(X) = n · M/N

💡 与二项分布的关系：当 N 很大时，超几何分布可以用二项分布近似，其中 p = M/N
