🎲 概率

1. 随机事件的概率

基本概念：

  • 必然事件：在一定条件下必然发生的事件，记为 Ω
  • 不可能事件：在一定条件下不可能发生的事件，记为 ∅
  • 随机事件：在一定条件下可能发生也可能不发生的事件
概率的范围：0 ≤ P(A) ≤ 1
P(Ω) = 1（必然事件），P(∅) = 0（不可能事件）

事件的关系与运算：

  • 包含关系：A ⊆ B 表示 A 发生则 B 必发生
  • 相等关系：A = B 表示 A ⊆ B 且 B ⊆ A
  • 并事件（和事件）：A ∪ B 表示 A 或 B 至少一个发生
  • 交事件（积事件）：A ∩ B 表示 A 且 B 同时发生
  • 互斥事件：A ∩ B = ∅，即 A、B 不能同时发生
  • 对立事件：A ∩ B = ∅ 且 A ∪ B = Ω，即 A、B 必有一个且仅有一个发生
💡 重要性质：若 A 与 B 对立，则 P(A) + P(B) = 1，即 P(A) = 1 - P(B)

2. 古典概型

特点：① 试验中所有可能出现的基本事件只有有限个；② 每个基本事件出现的可能性相等。

P(A) = 事件 A 包含的基本事件个数 / 基本事件的总数 = n(A) / n(Ω)
例题：掷两枚均匀的硬币，求一正一反的概率

解：基本事件有 {正正, 正反, 反正, 反反}，共 4 个

一正一反的事件有 {正反, 反正}，共 2 个

所以 P = 2/4 = 1/2

互斥事件的概率加法公式：

若 A、B 互斥，则 P(A ∪ B) = P(A) + P(B)
一般地，P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

3. 几何概型

特点：① 试验中所有可能出现的结果（基本事件）有无限多个；② 每个基本事件出现的可能性相等。

P(A) = 构成事件 A 的区域长度（面积或体积） / 试验的全部结果所构成的区域长度（面积或体积）
例题：在 [0, 10] 中任取一数，求该数大于 6 的概率

解：总区间长度 = 10 - 0 = 10

事件区域长度 = 10 - 6 = 4

所以 P = 4/10 = 0.4

4. 条件概率

设 A、B 为两个事件，且 P(A) > 0，称 P(B|A) 为在事件 A 发生的条件下，事件 B 发生的条件概率。

P(B|A) = P(A ∩ B) / P(A)
例题：已知某家庭有两个孩子，其中一个是女孩，求另一个也是女孩的概率

解：设 A = {至少有一个女孩}，B = {两个都是女孩}

基本事件：{男男, 男女, 女男, 女女}

P(A) = 3/4，P(A ∩ B) = P(B) = 1/4

所以 P(B|A) = (1/4) / (3/4) = 1/3

5. 事件的独立性

设 A、B 为两个事件，若 P(A ∩ B) = P(A)P(B)，则称事件 A 与事件 B 相互独立。

💡 性质：若 A 与 B 相互独立，则 A 与 B̅、Ā 与 B、Ā 与 B̅ 也都相互独立

n 次独立重复试验（伯努利试验）：

  • 每次试验只有两种结果：发生或不发生
  • 各次试验中的事件是相互独立的
  • 每次试验中，事件发生的概率相同
二项分布：在 n 次独立重复试验中，设事件 A 发生的次数为 X，每次试验中 A 发生的概率为 p，则
P(X = k) = C(n, k) p^k (1-p)^(n-k)，k = 0, 1, 2, ..., n
记为 X ~ B(n, p)
