一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {x | x < 2}，B = {x | x > -1}，则 A ∩ B = ______。
(-1, 2) 或 {x | -1 < x < 2}
2若复数 z 满足 z(1 + i) = 1 - i（i 为虚数单位），则 z = ______。
-i。z = (1 - i)/(1 + i) = [(1 - i)²]/[(1 + i)(1 - i)] = (-2i)/2 = -i
3函数 f(x) = lg(x - 1) 的定义域是 ______。
(1, +∞)。x - 1 > 0 ⇒ x > 1
4若函数 f(x) = 2sin(ωx + π/4)（ω > 0）的最小正周期为 2π/3，则 ω = ______。
3。T = 2π/ω = 2π/3 ⇒ ω = 3
5已知球的体积为 36π，则该球的表面积为 ______。
36π。V = 4πr³/3 = 36π ⇒ r³ = 27 ⇒ r = 3，S = 4πr² = 36π
6等差数列 {aₙ} 中，a₁ = 1，a₃ = 5，则公差 d = ______。
2。a₃ = a₁ + 2d ⇒ 5 = 1 + 2d ⇒ d = 2
二、选择题（本大题共4题，每题5分，满分20分）
13若 a > 0，b > 0，则 "a + b ≤ 4" 是 "ab ≤ 4" 的（ ）
✅ A。由基本不等式，a + b ≤ 4 ⇒ 2√(ab) ≤ a + b ≤ 4 ⇒ √(ab) ≤ 2 ⇒ ab ≤ 4；
反之，ab ≤ 4 时，取 a = 4，b = 1，则 a + b = 5 > 4，故必要性不成立。
14下列函数中，既是偶函数又在 (0, +∞) 上单调递增的是（ ）
✅ B。
A. y = x³ 是奇函数；
B. y = |x| + 1 是偶函数，x > 0 时 y = x + 1 单调递增；
C. y = -x² + 1 在 (0, +∞) 单调递减；
D. y = 2^(-|x|) = (1/2)^|x| 在 (0, +∞) 单调递减。
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知 △ABC 中，角 A, B, C 的对边分别为 a, b, c，且 a = 2，b = √2，∠A = 45°。
（1）求 ∠B 的大小；
（2）求边长 c。
解：
（1）由正弦定理 a/sinA = b/sinB
sinB = b·sinA/a = √2 × sin45° / 2 = √2 × (√2/2) / 2 = 1/2
∵ b < a，∴ ∠B < ∠A = 45°，∴ ∠B = 30°

（2）∠C = 180° - 45° - 30° = 105°
sin105° = sin(60° + 45°) = sin60°cos45° + cos60°sin45° = (√6 + √2)/4
由正弦定理 c/sinC = a/sinA
c = a·sinC/sinA = 2 × [(√6 + √2)/4] / (√2/2) = 2 × (√6 + √2)/(2√2) = √3 + 1
