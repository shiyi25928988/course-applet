1. 已知集合 A = {x | 0 < x < 3}，B = {x | 1 ≤ x ≤ 4}，则 A ∩ B =（　　）
A. (0, 1)
B. (1, 3)
C. [1, 3)
D. (0, 4]
【答案】C
【解析】
集合的交集是两个集合的公共元素组成的集合。

A = (0, 3)，B = [1, 4]
画数轴可知，两集合的公共部分是从1（含）到3（不含）。

所以 A ∩ B = [1, 3)，选C。

【考点】集合的交集运算

5. 设集合 A = {1, 2, 3}，B = {x | x² - 4x + 3 = 0}，则 A ∪ B =  
【答案】{1, 2, 3}
【解析】
首先解方程求集合B：
x² - 4x + 3 = 0
(x - 1)(x - 3) = 0
所以 x = 1 或 x = 3

因此 B = {1, 3}

集合的并集是两个集合所有元素组成的集合（重复元素只写一次）：
A ∪ B = {1, 2, 3} ∪ {1, 3} = {1, 2, 3}

3. 已知 a = 20.3，b = 0.32，c = log20.3，则 a, b, c 的大小关系是（　　）
A. a < b < c
B. c < b < a
C. c < a < b
D. b < c < a
【答案】B
【解析】
利用中间值0和1比较大小：

对于 a = 20.3：
因为指数函数 y = 2x 是增函数，0.3 > 0，所以 20.3 > 20 = 1
即 a > 1

对于 b = 0.32：
0.32 = 0.09，所以 0 < b < 1

对于 c = log20.3：
因为对数函数 y = log2x 是增函数，0.3 < 1，所以 log20.3 < log21 = 0
即 c < 0

综上：c < 0 < b < 1 < a，即 c < b < a，选B。

【考点】指数函数、对数函数的单调性，比较大小问题。

📐 第五章 三角函数 中档 共15题 · 高考重点

1. sin 15° + cos 15° 的值为（　　）
A. √6/2
B. √6/2
C. √2/2
D. √3/2
【答案】A（√6/2）
【解析】
方法一：和角公式
sin 15° + cos 15° = √2 (√2/2 sin15° + √2/2 cos15°)
= √2 (sin15°cos45° + cos15°sin45°)
= √2 sin(15°+45°) = √2 sin60°
= √2 × (√3/2) = √6/2

方法二：先平方再开方
(sin15° + cos15°)² = sin²15° + 2sin15°cos15° + cos²15°
= 1 + sin30° = 1 + 1/2 = 3/2
因为 sin15° + cos15° > 0，所以开方得 √(3/2) = √6/2

【考点】三角恒等变换，辅助角公式。
10. 函数 f(x) = 2 sin(2x + π/3) 的最小正周期为  ， 当 x ∈ [0, π/2] 时，f(x) 的最大值为  。
【答案】π；2
【解析】
(1) 最小正周期：
对于 y = A sin(ωx + φ)，最小正周期 T = 2π/|ω|
这里 ω = 2，所以 T = 2π/2 = π

(2) 求最大值：
当 x ∈ [0, π/2] 时：
2x ∈ [0, π]
2x + π/3 ∈ [π/3, 4π/3]

令 t = 2x + π/3，则 t ∈ [π/3, 4π/3]
求 f(t) = 2 sin t 在 t ∈ [π/3, 4π/3] 上的最大值。

sin t 在 [π/3, π/2] 上递增，在 [π/2, 4π/3] 上递减，
所以当 t = π/2 时，sin t 取得最大值 1。

因此 f(x) 的最大值为 2 × 1 = 2。

【考点】三角函数的周期性和最值，是高考高频考点。

➡️ 第六章 平面向量 中档 共10题

5. 已知向量 a = (1, 2)，b = (m, -1)，若 a ⊥ b，则实数 m 的值为（　　）
A. -2
B. 2
C. -1/2
D. 1/2
【答案】B
【解析】
两个向量垂直的充要条件是它们的数量积为0：
a ⊥ b ⇔ a · b = 0
已知 a = (x₁, y₁)，b = (x₂, y₂)，则：
a · b = x₁x₂ + y₁y₂ = 0
代入 a = (1, 2)，b = (m, -1)：
1 × m + 2 × (-1) = 0
m - 2 = 0
m = 2

【考点】向量垂直的坐标表示，数量积的运算。

📦 第八章 立体几何初步 提高 共12题 · 空间想象

8. 已知正方体 ABCD-A₁B₁C₁D₁ 的棱长为 2，E 为 CC₁ 的中点，则三棱锥 E-ABD 的体积为  。
【答案】2/3
【解析】
三棱锥体积公式：V = (1/3) × 底面积 × 高

选择 △ABD 为底面：
S△ABD = (1/2) × AB × AD = (1/2) × 2 × 2 = 2

点 E 到平面 ABD 的距离就是 E 到底面 ABCD 的距离，即 EC = 1

所以体积：
V = (1/3) × S△ABD × 高 = (1/3) × 2 × 1 = 2/3

【考点】空间几何体的体积计算，换顶点法是常用技巧。

📉 第五章 一元函数的导数及其应用 提高 压轴题专项

12. 已知函数 f(x) = x³ - 3x，求：
(1) 函数 f(x) 的单调区间；
(2) 函数 f(x) 在区间 [-2, 2] 上的最大值和最小值。
【答案】(1) 递增(-∞,-1),(1,+∞)，递减(-1,1)；(2) 最大值2，最小值-2
【解析】
函数 f(x) 的定义域为 R。

(1) 求单调区间：
先求导数：f'(x) = 3x² - 3 = 3(x² - 1) = 3(x + 1)(x - 1)

令 f'(x) > 0，即 3(x + 1)(x - 1) > 0，解得 x < -1 或 x > 1
所以 f(x) 的单调递增区间为 (-∞, -1) 和 (1, +∞)

令 f'(x) < 0，即 3(x + 1)(x - 1) < 0，解得 -1 < x < 1
所以 f(x) 的单调递减区间为 (-1, 1)

(2) 求在[-2, 2]上的最值：
先求极值点和端点的函数值：
f(-2) = (-2)³ - 3×(-2) = -8 + 6 = -2
f(-1) = (-1)³ - 3×(-1) = -1 + 3 = 2（极大值）
f(1) = 1³ - 3×1 = 1 - 3 = -2（极小值）
f(2) = 2³ - 3×2 = 8 - 6 = 2

比较得：最大值为 2，最小值为 -2。

【考点】利用导数研究函数单调性和最值，是高考导数大题的基础题型。
