⭕ 圆锥曲线专项突破

椭圆 · 双曲线 · 抛物线定义性质 · 直线与圆锥曲线综合 · 定点定值 · 共 40 道精选

1椭圆基础（12分）

已知椭圆 C: x²/a² + y²/b² = 1 (a > b > 0) 的离心率为 1/2，右焦点为 F(1, 0)。求：

(1) 椭圆 C 的标准方程；

(2) 椭圆的长轴长和短轴长。

答案：(1) x²/4 + y²/3 = 1；(2) 长轴长 4，短轴长 2√3
解析：
(1) 由题意 c = 1，e = c/a = 1/2 ⇒ a = 2
b² = a² - c² = 4 - 1 = 3
∴ 椭圆方程：x²/4 + y²/3 = 1

(2) 长轴长 2a = 4，短轴长 2b = 2√3
2抛物线综合（12分）

已知抛物线 y² = 4x，过焦点 F 作斜率为 1 的直线交抛物线于 A、B 两点，求 |AB| 的长度。

答案：|AB| = 8
解析：
抛物线焦点 F(1, 0)，直线方程：y = x - 1
联立 { y = x - 1, y² = 4x } ⇒ (x - 1)² = 4x ⇒ x² - 6x + 1 = 0
设 A(x₁, y₁), B(x₂, y₂)，则 x₁ + x₂ = 6
由抛物线定义：|AB| = x₁ + x₂ + p = 6 + 2 = 8
