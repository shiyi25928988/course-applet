一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1计算：(1 + 2i)/(1 - 2i) = ______（i 为虚数单位）。
(-3 + 4i)/5。
(1 + 2i)/(1 - 2i) = [(1 + 2i)²]/[(1 - 2i)(1 + 2i)] = (1 + 4i + 4i²)/5 = (-3 + 4i)/5
2函数 f(x) = x² - 2x + 3 的值域为 ______（用区间表示）。
[2, +∞)。
f(x) = (x - 1)² + 2 ≥ 2，故值域为 [2, +∞)
3已知幂函数 y = kx^α 的图像过点 (2, √2)，则 k + α = ______。
3/2。
幂函数 ⇒ k = 1
过点 (2, √2) ⇒ √2 = 2^α ⇒ 2^(1/2) = 2^α ⇒ α = 1/2
故 k + α = 1 + 1/2 = 3/2
4在 △ABC 中，若 a = 3，b = 5，sinA = 1/3，则 sinB = ______。
5/9。
由正弦定理 a/sinA = b/sinB ⇒ 3/(1/3) = 5/sinB ⇒ 9 = 5/sinB ⇒ sinB = 5/9
5已知直线 l₁: ax + 2y - 3 = 0 与 l₂: 3x + (a + 1)y - a = 0 平行，则 a = ______。
-3。
平行 ⇒ 斜率相等，且不重合
a(a + 1) = 2×3 ⇒ a² + a - 6 = 0 ⇒ (a + 3)(a - 2) = 0 ⇒ a = -3 或 a = 2
a = 2 时，l₁: 2x + 2y - 3 = 0，l₂: 3x + 3y - 2 = 0，平行
a = -3 时，l₁: -3x + 2y - 3 = 0，l₂: 3x - 2y + 3 = 0，重合，舍去
故 a = 2（注：原答案-3有误，应为2）
6若 (x + 1/x)^n 展开式的二项式系数之和为 64，则 n = ______。
6。
二项式系数之和为 2^n = 64 = 2^6 ⇒ n = 6
二、选择题（本大题共4题，每题5分，满分20分）
13若 a > b > 0，c < d < 0，则下列不等式正确的是（ ）
✅ B。
举特例：a = 2, b = 1, c = -2, d = -1
A. ac = -4, bd = -1 ⇒ -4 > -1 错误；
B. ac = -4 < bd = -1 正确；
C. a/d = -2, b/c = -1/2 ⇒ -2 > -1/2 错误；
D. a/d = -2 < b/c = -1/2 正确。
严谨证明：c < d < 0 ⇒ -c > -d > 0
又 a > b > 0 ⇒ a(-c) > b(-d) ⇒ -ac > -bd ⇒ ac < bd，故 B 正确。
14函数 f(x) = sinx + cosx 的最小正周期为（ ）
✅ C。
f(x) = sinx + cosx = √2 sin(x + π/4)
最小正周期 T = 2π/1 = 2π
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知数列 {aₙ} 是等差数列，a₁ = 1，且 a₁, a₂, a₅ 成等比数列。
（1）求数列 {aₙ} 的通项公式；
（2）若 aₙ < 0，数列 {bₙ} 满足 bₙ = 2^(aₙ)，求数列 {bₙ} 的前 n 项和 Sₙ。
解：
（1）设等差数列公差为 d，则 a₂ = 1 + d，a₅ = 1 + 4d
∵ a₁, a₂, a₅ 成等比数列 ∴ a₂² = a₁·a₅
(1 + d)² = 1 × (1 + 4d) ⇒ 1 + 2d + d² = 1 + 4d ⇒ d² - 2d = 0 ⇒ d = 0 或 d = 2
当 d = 0 时，aₙ = 1
当 d = 2 时，aₙ = 1 + (n - 1) × 2 = 2n - 1

（2）∵ aₙ < 0，而 a₁ = 1 > 0，故只能 d = 0 时 aₙ = 1 不满足，
说明题目条件应为 aₙ > 0（否则无解），取 d = 2，aₙ = 2n - 1
则 bₙ = 2^(aₙ) = 2^(2n-1) = (1/2) × 4^n
{bₙ} 是首项 b₁ = 2^(1) = 2，公比 q = 4 的等比数列
Sₙ = b₁(1 - qⁿ)/(1 - q) = 2(1 - 4ⁿ)/(1 - 4) = (2/3)(4ⁿ - 1)
