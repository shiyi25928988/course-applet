一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1设全集 U = R，集合 A = {x | x ≥ 2}，B = {x | 0 ≤ x < 5}，则 (∁_U A) ∩ B = ______。
[0, 2)。
∁_U A = {x | x < 2}，故 (∁_U A) ∩ B = {x | 0 ≤ x < 2} = [0, 2)
2若复数 z 满足 z(1 + i) = 2i（i 为虚数单位），则 |z| = ______。
√2。
|z(1 + i)| = |2i| ⇒ |z| × √2 = 2 ⇒ |z| = √2
或 z = 2i/(1 + i) = i(1 - i) = 1 + i，|z| = √2
3函数 f(x) = x³ - 3x 的极大值为 ______。
2。
f'(x) = 3x² - 3 = 3(x + 1)(x - 1)
f'(x) = 0 ⇒ x = ±1
x < -1 时 f'(x) > 0，-1 < x < 1 时 f'(x) < 0
故 x = -1 时取极大值 f(-1) = -1 + 3 = 2
4从 1, 2, 3, 4, 5, 6 中任取两个不同的数，其乘积为偶数的概率是 ______。
4/5。
总取法：C(6, 2) = 15 种
乘积为奇数 ⇒ 两数均为奇数：C(3, 2) = 3 种
乘积为偶数的概率 = 1 - 3/15 = 1 - 1/5 = 4/5
5已知 a = (1, 2)，b = (-2, 3)，则 a 在 b 方向上的投影为 ______。
4√13/13。
投影 = (a·b)/|b| = [1×(-2) + 2×3]/√(4 + 9) = (-2 + 6)/√13 = 4/√13 = 4√13/13
6抛物线 y² = 8x 的焦点到准线的距离是 ______。
4。
y² = 2px 中，p 就是焦点到准线的距离
2p = 8 ⇒ p = 4
二、选择题（本大题共4题，每题5分，满分20分）
13已知 a > b，则下列不等式恒成立的是（ ）
✅ B。
A. a = 1, b = -2 时不成立；
B. y = 2^x 是增函数，故 a > b ⇒ 2^a > 2^b，恒成立；
C. a = 1, b = -1 时不成立；
D. c = 0 时等号成立。
14函数 f(x) = sin(2x + π/4) 的图像向左平移 φ（φ > 0）个单位后关于 y 轴对称，则 φ 的最小值为（ ）
✅ A。
平移后得 f(x + φ) = sin(2(x + φ) + π/4) = sin(2x + 2φ + π/4)
关于 y 轴对称 ⇒ 偶函数 ⇒ 2φ + π/4 = kπ + π/2, k ∈ Z
2φ = kπ + π/4 ⇒ φ = kπ/2 + π/8
φ > 0，取 k = 0 得最小值 φ = π/8
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知数列 {aₙ} 满足 a₁ = 1，a_(n+1) = 2aₙ + 1（n ∈ N*）。
（1）证明：{aₙ + 1} 是等比数列；
（2）求数列 {aₙ} 的通项公式。
解：
（1）证明：
∵ a_(n+1) = 2aₙ + 1
∴ a_(n+1) + 1 = 2aₙ + 2 = 2(aₙ + 1)
又 a₁ + 1 = 2 ≠ 0
∴ {aₙ + 1} 是以 2 为首项，2 为公比的等比数列。

（2）由（1）知 aₙ + 1 = 2 × 2^(n-1) = 2^n
故 aₙ = 2^n - 1（n ∈ N*）
