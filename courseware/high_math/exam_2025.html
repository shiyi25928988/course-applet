一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {-1, 0, 1}，B = {x | x² - x = 0}，则 A ∩ B = ______。
{0, 1}。
x² - x = 0 ⇒ x(x - 1) = 0 ⇒ x = 0 或 x = 1 ⇒ B = {0, 1}
A ∩ B = {0, 1}
2复数 z 满足 |z| = 1，且 z ≠ ±i，则 z + 1/z = ______（实数/虚数）。
实数。
设 z = cosθ + i sinθ，则 1/z = cosθ - i sinθ
z + 1/z = 2cosθ ∈ R
3函数 f(x) = √(4 - x²) + ln(x + 1) 的定义域为 ______。
(-1, 2]。
4 - x² ≥ 0 ⇒ -2 ≤ x ≤ 2
x + 1 > 0 ⇒ x > -1
定义域：(-1, 2]
4从编号为 1, 2, 3, 4 的 4 张卡片中随机抽取 2 张，则取出的 2 张卡片编号之和为奇数的概率是 ______。
2/3。
总取法：C(4, 2) = 6 种
和为奇数 ⇒ 一奇一偶：C(2, 1) × C(2, 1) = 4 种
概率 = 4/6 = 2/3
5已知向量 a = (2, 1)，b = (1, -1)，若 a - kb 与 a 垂直，则 k = ______。
5。
a - kb = (2 - k, 1 + k)
与 a 垂直 ⇒ (a - kb)·a = 0
2(2 - k) + 1(1 + k) = 0 ⇒ 4 - 2k + 1 + k = 0 ⇒ 5 - k = 0 ⇒ k = 5
6等差数列 {aₙ} 中，a₁ + a₅ = 10，a₄ = 7，则公差 d = ______。
2。
a₁ + a₅ = 2a₃ = 10 ⇒ a₃ = 5
d = a₄ - a₃ = 7 - 5 = 2

13若 f(x) 是 R 上的增函数，则下列不等式一定成立的是（ ）
✅ C。
增函数定义：x₁ > x₂ ⇔ f(x₁) > f(x₂)，故 C 正确；
A. 应为 f(0) < f(1)；
B. a = 0 时 f(0) = f(0)，不满足 >；
D. 取 f(x) = x，则 f(a) + f(-a) = 0，不满足 >。
14双曲线 x²/4 - y²/5 = 1 的渐近线方程为（ ）
✅ A。
双曲线 x²/a² - y²/b² = 1 的渐近线为 y = ±(b/a)x
此处 a² = 4, b² = 5 ⇒ a = 2, b = √5
渐近线：y = ±(√5/2)x
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知正四棱锥 P - ABCD 的底面边长为 2，侧棱长为 √5。
（1）求该四棱锥的高；
（2）求该四棱锥的表面积。
解：
（1）连接 AC, BD 交于 O，连接 PO，则 PO ⊥ 底面 ABCD，PO 就是高
底面正方形边长为 2 ⇒ 对角线 AC = 2√2 ⇒ AO = √2
在 Rt△POA 中，PA = √5，AO = √2
PO = √(PA² - AO²) = √(5 - 2) = √3
故四棱锥的高为 √3

（2）侧面是 4 个全等的等腰三角形
取 AB 中点 E，连接 PE，则 PE ⊥ AB，PE 是斜高
在 Rt△PAE 中，PA = √5，AE = 1
PE = √(PA² - AE²) = √(5 - 1) = 2
每个侧面面积 = 1/2 × 2 × 2 = 2
侧面积 = 4 × 2 = 8
底面积 = 2 × 2 = 4
表面积 = 8 + 4 = 12
