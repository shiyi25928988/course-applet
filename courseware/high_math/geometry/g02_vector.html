平面向量与空间向量

平面向量的线性运算

向量加法：三角形法则、平行四边形法则
向量减法：三角形法则
数乘运算：λa，长度|λ||a|，方向：λ>0同向，λ<0反向
共线向量定理：
向量a(a≠0)与b共线 ⇔ 存在唯一实数λ，使得b = λa
坐标运算：设a = (x₁, y₁), b = (x₂, y₂)
a + b = (x₁ + x₂, y₁ + y₂)
a - b = (x₁ - x₂, y₁ - y₂)
λa = (λx₁, λy₁)

平面向量的数量积

a · b = |a||b| cos θ
坐标形式：a · b = x₁x₂ + y₁y₂
重要性质：
a ⊥ b ⇔ a · b = 0 ⇔ x₁x₂ + y₁y₂ = 0
|a| = √(x₁² + y₁²)
cos θ = (a · b)/(|a||b|)

空间向量

设a = (x₁, y₁, z₁), b = (x₂, y₂, z₂)
a ± b = (x₁ ± x₂, y₁ ± y₂, z₁ ± z₂)
λa = (λx₁, λy₁, λz₁)
a · b = x₁x₂ + y₁y₂ + z₁z₂
|a| = √(x₁² + y₁² + z₁²)
空间向量的叉乘：
a × b = (y₁z₂ - y₂z₁, z₁x₂ - z₂x₁, x₁y₂ - x₂y₁)
|a × b| = |a||b| sin θ
空间中的平行与垂直：
a // b ⇔ a = λb ⇔ x₁/x₂ = y₁/y₂ = z₁/z₂ （b≠0）
a ⊥ b ⇔ a · b = 0 ⇔ x₁x₂ + y₁y₂ + z₁z₂ = 0
空间中的夹角与距离：
线线角：cos θ = |a · b|/(|a||b|)
点面距：d = |AB · n| / |n| （n为平面法向量）
