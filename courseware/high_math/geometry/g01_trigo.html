三角函数

任意角与弧度制

角度与弧度换算：
360° = 2π rad，180° = π rad
1° = π/180 rad，1 rad = (180/π)°
扇形公式：
弧长：l = |α|r
面积：S = ½ lr = ½ |α| r²

三角函数定义

sin α = y/r，cos α = x/r，tan α = y/x
其中r = √(x² + y²)
各象限符号：
第一象限：sin +，cos +，tan +
第二象限：sin +，cos -，tan -
第三象限：sin -，cos -，tan +
第四象限：sin -，cos +，tan -

记忆口诀：一全正，二正弦，三正切，四余弦
同角三角函数基本关系：
sin²α + cos²α = 1
tan α = sin α / cos α

诱导公式

记忆口诀：奇变偶不变，符号看象限

sin(π + α) = -sin α，cos(π + α) = -cos α
sin(π - α) = sin α，cos(π - α) = -cos α
sin(-α) = -sin α，cos(-α) = cos α
sin(π/2 + α) = cos α，cos(π/2 + α) = -sin α
sin(π/2 - α) = cos α，cos(π/2 - α) = sin α

三角恒等变换

和角公式：
sin(α ± β) = sin α cos β ± cos α sin β
cos(α ± β) = cos α cos β ∓ sin α sin β
tan(α ± β) = (tan α ± tan β) / (1 ∓ tan α tan β)
二倍角公式：
sin 2α = 2 sin α cos α
cos 2α = cos²α - sin²α = 2cos²α - 1 = 1 - 2sin²α
tan 2α = 2 tan α / (1 - tan²α)
降幂公式：
sin²α = (1 - cos 2α)/2
cos²α = (1 + cos 2α)/2

解三角形

正弦定理：a/sin A = b/sin B = c/sin C = 2R
余弦定理：
a² = b² + c² - 2bc cos A
b² = a² + c² - 2ac cos B
c² = a² + b² - 2ab cos C
面积公式：S = ½ ab sin C = ½ bc sin A = ½ ac sin B
