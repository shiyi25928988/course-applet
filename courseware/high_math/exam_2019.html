一、填空题（本大题共12题，满分54分，第1-6题每题4分，第7-12题每题5分）
1已知集合 A = {x | x² - 4 < 0}，则 A = ______（用区间表示）。
(-2, 2)。x² - 4 < 0 ⇒ (x - 2)(x + 2) < 0 ⇒ -2 < x < 2
2计算：lim┬(n→∞)⁡〖(2n+1)/(3n-2)〗 = ______。
2/3。lim┬(n→∞)⁡〖(2n+1)/(3n-2)〗 = lim┬(n→∞)⁡〖(2+1/n)/(3-2/n)〗 = 2/3
3若 f(x) = e^x + ae^(-x) 是奇函数，则 a = ______。
-1。f(-x) = e^(-x) + ae^x = -f(x) = -e^x - ae^(-x)
比较系数得 a = -1
4在 (x - 2/x)^6 的二项展开式中，常数项为 ______。
-160。
通项 T_(r+1) = C(6, r) x^(6-r) (-2/x)^r = C(6, r) (-2)^r x^(6-2r)
令 6 - 2r = 0 ⇒ r = 3
常数项 = C(6, 3) (-2)^3 = 20 × (-8) = -160
5已知圆 x² + y² = 4，则过点 (1, √3) 的切线方程为 ______。
x + √3 y - 4 = 0。
点 (1, √3) 在圆上，圆心 (0,0)，半径 2
半径斜率 = √3/1 = √3，故切线斜率 = -1/√3
切线方程：y - √3 = -1/√3 (x - 1) ⇒ √3 y - 3 = -x + 1 ⇒ x + √3 y - 4 = 0
6若 sinθ + cosθ = 1/5，则 sin2θ = ______。
-24/25。
(sinθ + cosθ)² = 1 + 2sinθcosθ = 1 + sin2θ = (1/5)² = 1/25
∴ sin2θ = 1/25 - 1 = -24/25
二、选择题（本大题共4题，每题5分，满分20分）
13已知 a, b ∈ R，则 "a² > b²" 是 "|a| > |b|" 的（ ）
✅ C。
∵ a² = |a|²，b² = |b|²
∴ a² > b² ⇔ |a|² > |b|² ⇔ |a| > |b|（因为绝对值非负）
故互为充要条件。
14若 0 < a < b < 1，则下列不等式正确的是（ ）
✅ C。
举特例：a = 1/4, b = 1/2
A. log_(1/4)(1/2) = 1/2, log_(1/2)(1/4) = 2，故 1/2 < 2，A 错误；
B. (1/4)^(1/2) = 1/2, (1/2)^(1/4) ≈ 0.84，故 1/2 < 0.84，B 错误；
C. 设 f(x) = ln x / x (0 < x < 1)，f'(x) = (1 - ln x)/x² > 0，f(x) 递增
∵ a < b ∴ f(a) < f(b) ⇒ ln a / a < ln b / b ⇒ b ln a < a ln b ⇒ a ln b > b ln a，C 正确；
D. y = e^x 单调递增，故 e^a < e^b，D 错误。
三、解答题（本大题共5题，满分76分）
19本题满分14分
已知 f(x) = 2sin(2x + π/3)。
（1）求 f(x) 的振幅、最小正周期和初相；
（2）求 f(x) 在区间 [-π/6, π/4] 上的最大值和最小值。
解：
（1）f(x) = 2sin(2x + π/3) 是正弦型函数 y = A sin(ωx + φ)
振幅 A = 2
最小正周期 T = 2π/ω = 2π/2 = π
初相 φ = π/3

（2）∵ x ∈ [-π/6, π/4]
∴ 2x + π/3 ∈ [2×(-π/6) + π/3, 2×π/4 + π/3] = [0, 5π/6]
当 2x + π/3 = π/2 即 x = π/12 时，f(x) 取得最大值：2 sin(π/2) = 2
当 2x + π/3 = 0 即 x = -π/6 时，f(x) = 2 sin 0 = 0
当 2x + π/3 = 5π/6 即 x = π/4 时，f(x) = 2 sin(5π/6) = 2 × 1/2 = 1
故 f(x) 在 [-π/6, π/4] 上的最大值为 2，最小值为 0。
