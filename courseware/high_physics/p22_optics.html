光学

几何光学

n = sinθ₁/sinθ₂ = c/v
折射定律：入射角的正弦与折射角的正弦成正比，比例常数为折射率。
sinC = 1/n
全反射：光从光密介质射入光疏介质时，当入射角增大到某一角度，折射角达到90°，折射光完全消失，只剩下反射光。C为临界角。

光的干涉

Δx = Lλ/d
双缝干涉：相邻亮条纹（或暗条纹）之间的距离公式。

亮纹条件：路程差Δr = kλ（k = 0, ±1, ±2...）
暗纹条件：路程差Δr = (2k+1)λ/2（k = 0, ±1, ±2...）

光的衍射和偏振

衍射：光离开直线路径绕到障碍物阴影里的现象。产生明显衍射的条件：障碍物或孔的尺寸可以跟波长相比甚至比波长更小。
偏振：横波只沿某一特定方向振动的现象。光的偏振证明光是横波。

电磁波谱

按波长从大到小排列：无线电波 → 红外线 → 可见光（红橙黄绿蓝靛紫）→ 紫外线 → X射线 → γ射线
