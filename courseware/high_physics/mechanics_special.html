1牛顿第二定律应用（12分）

质量为 2kg 的物体放在水平面上，在水平拉力 F = 10N 的作用下由静止开始运动，物体与地面间的动摩擦因数 μ = 0.3。求：

(1) 物体的加速度；

(2) 2s 末物体的速度和位移。（g = 10m/s²）

答案：(1) 2m/s²；(2) v = 4m/s，x = 4m
解析：
(1) 摩擦力 f = μmg = 0.3×2×10 = 6N
由牛顿第二定律：F - f = ma ⇒ a = (10 - 6)/2 = 2m/s²

(2) v = at = 2×2 = 4m/s
x = ½at² = ½×2×4 = 4m
2动量守恒定律（12分）

质量为 m₁ = 2kg 的小球以 v₁ = 5m/s 的速度向右运动，与质量为 m₂ = 3kg 的静止小球发生弹性正碰。求碰撞后两球的速度。

答案：v₁' = -1m/s（向左），v₂' = 4m/s（向右）
解析：
弹性碰撞满足动量守恒和动能守恒：
m₁v₁ = m₁v₁' + m₂v₂'
½m₁v₁² = ½m₁v₁'² + ½m₂v₂'²

解得：v₁' = (m₁-m₂)v₁/(m₁+m₂) = (-1)×5/5 = -1m/s
v₂' = 2m₁v₁/(m₁+m₂) = 4×5/5 = 4m/s
