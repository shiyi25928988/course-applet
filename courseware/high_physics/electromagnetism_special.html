⚡ 高中物理电磁学综合专项

电场强度电势 · 带电粒子运动 · 安培力洛伦兹力 · 电磁感应综合 · 共 50 道精选

1电场强度计算（12分）

真空中有两个点电荷 Q₁ = +4×10⁻⁸C 和 Q₂ = -1×10⁻⁸C，相距 0.3m。求：

(1) 在两点电荷连线上，电场强度为零的位置距 Q₁ 的距离；

(2) 该位置的电势（取无穷远电势为零）。（k = 9×10⁹N·m²/C²）

答案：(1) 0.6m；(2) 300V
解析：
(1) 设距 Q₁ 为 x，则距 Q₂ 为 x - 0.3
kQ₁/x² + kQ₂/(x-0.3)² = 0 ⇒ 4/x² = 1/(x-0.3)²
解得 x = 0.6m（x = 0.2m 在中间方向相同，舍去）

(2) φ = kQ₁/x + kQ₂/(x-0.3) = 9×10⁹×(4×10⁻⁸/0.6 - 1×10⁻⁸/0.3) = 300V
2带电粒子在磁场中运动（12分）

一质子以 v = 3×10⁶m/s 的速度垂直进入磁感应强度 B = 0.5T 的匀强磁场中，求：

(1) 洛伦兹力的大小；

(2) 圆周运动的轨道半径和周期。（e = 1.6×10⁻¹⁹C，mₚ = 1.67×10⁻²⁷kg）

答案：(1) 2.4×10⁻¹³N；(2) r = 6.26×10⁻²m，T = 1.31×10⁻⁷s
解析：
(1) F = evB = 1.6×10⁻¹⁹×3×10⁶×0.5 = 2.4×10⁻¹³N

(2) evB = mv²/r ⇒ r = mv/(eB) = (1.67×10⁻²⁷×3×10⁶)/(1.6×10⁻¹⁹×0.5) ≈ 6.26×10⁻²m
T = 2πr/v = 2πm/(eB) ≈ 1.31×10⁻⁷s
