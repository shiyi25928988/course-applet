一、选择题（本大题共12题，每题3分，满分36分）
1下列关于电场线的说法正确的是（ ）
答案：C
解析：电场线是为形象描述电场而假想的曲线，并非客观存在；电场线越密的地方场强越大；沿电场线方向电势逐渐降低；正电荷沿电场线方向运动时，电场力做正功，电势能减少。
2一定质量的理想气体，温度不变时，体积增大则压强（ ）
答案：B
解析：由玻意耳定律：pV = C（常数），温度不变时，体积增大则压强减小。
二、填空题（本大题共5题，每题4分，满分20分）
13某单色光在真空中的波长为 600 nm，该光的频率为 ______ × 10¹⁴ Hz（保留1位小数）。
答案：5.0
解析：由 c = λf，f = c/λ = 3×10⁸ / (600×10⁻⁹) = 5.0×10¹⁴ Hz
三、计算题（本大题共3题，满分44分）
19本题满分12分
如图所示，匀强磁场磁感应强度 B = 0.5 T，一质量 m = 0.1 kg、电阻 R = 1 Ω 的金属棒 ab 可沿光滑导轨无摩擦滑动，导轨间距 L = 0.4 m。现给棒一水平初速度 v₀ = 2 m/s，求： （1）棒刚开始运动时的感应电动势大小； （2）棒刚开始运动时的加速度大小。
答案：(1) 0.4 V；(2) 0.8 m/s²
解析：

（1）感应电动势 E = BLv₀ = 0.5×0.4×2 = 0.4 V

（2）感应电流 I = E/R = 0.4/1 = 0.4 A
安培力 F = BIL = 0.5×0.4×0.4 = 0.08 N
加速度 a = F/m = 0.08/0.1 = 0.8 m/s²
