第四章 曲线运动

4.1 曲线运动

速度方向：曲线运动中，质点在某一点的速度方向沿曲线在这一点的切线方向。
曲线运动条件：合外力方向与速度方向不在同一直线上。

4.2 平抛运动

水平：vx = v₀，x = v₀t
竖直：vy = gt，y = ½gt²
特点：水平方向匀速直线运动，竖直方向自由落体运动，两个分运动独立进行。
合速度：v = √(vx² + vy²)，方向tanθ = vy/vx

4.3 圆周运动

线速度：v = Δs/Δt = 2πr/T
角速度：ω = Δθ/Δt = 2π/T = 2πf
关系：v = ωr
向心加速度：a = v²/r = ω²r，方向始终指向圆心，时刻变化。
向心力：F = mv²/r = mω²r，是效果力，由某一个力或几个力的合力提供。
