✍️ 写作 Writing

Task 1 小作文 + Task 2 大作文 · 总时长 60分钟

📋 写作结构说明

Task 1 小作文

  • 字数：至少 150 词
  • 建议用时：20分钟
  • 占分比重：约 33%
  • 学术类：图表描述
  • 培训类：书信写作

Task 2 大作文

  • 字数：至少 250 词
  • 建议用时：40分钟
  • 占分比重：约 67%
  • 议论文 / 说明文
  • 四大评分标准

💡 四大评分标准

TR/TA（任务完成）→ CC（连贯与衔接）→ LR（词汇丰富度）→ GRA（语法多样性与准确性）

📊 Task 1 小作文（学术类）

📈 动态图表（数据随时间变化）

线图 Line Graph 柱状图 Bar Chart 表格 Table

核心词汇：上升（increase, rise, grow）、下降（decrease, fall, drop）、波动（fluctuate）、稳定（remain stable）、峰值（peak）、谷值（bottom out）

📊 静态图表（数据不随时间变化）

饼图 Pie Chart 柱状图 Bar Chart 表格 Table

核心词汇：占比（account for, make up, constitute）、排名（rank, lead）、最大/最小（largest, smallest）、比较（compare, contrast）

🗺️ 其他类型

流程图 Flow Chart 地图 Map 多种图 Combination

💬 Task 1 小作文（培训类 - 书信）

书信类型

投诉信 Complaint 道歉信 Apology 请求信 Request 建议信 Suggestion 感谢信 Thanks 邀请信 Invitation

📝 Task 2 大作文题型

⚖️ Argumentation 议论文（占比约 70%）

  • Agree / Disagree：同意与否
  • Discuss both views：双边讨论
  • Advantages / Disadvantages：利弊分析

🔍 Report 说明文（占比约 30%）

  • Cause + Solution：原因 + 解决办法
  • Cause + Effect：原因 + 影响
  • Problem + Solution：问题 + 解决

🎯 大作文高频话题

教育 Education 科技 Technology 环境 Environment 社会 Society 文化 Culture 政府 Government 媒体 Media 犯罪 Crime 旅游 Tourism 全球化 Globalization

✨ 高分句型示例

开头段句型

It is often argued that... / Many people believe that... / There is a growing debate about...
While some people hold the view that..., others argue that...

主体段句型

The primary reason for this is that... / Another factor to consider is...
For example... / For instance... / A case in point is...

结尾段句型

In conclusion... / To sum up... / In summary...
After analyzing both perspectives, it is clear that...

⚠️ 常见失分点

  • ❌ 字数不够（小作文不足150，大作文不足250，直接扣分！）
  • ❌ 偏题、跑题（一定要仔细审题，抓住核心关键词）
  • ❌ 逻辑混乱、衔接不足（多用连接词：however, therefore, furthermore）
  • ❌ 重复用词、词汇匮乏（积累同义词替换）
  • ❌ 语法错误过多（简单句为主，适当使用复杂句）
  • ❌ 模板痕迹过重（模板要活用，不要死记硬背）
  • ❌ 书写不清晰、有修改痕迹（工整书写很重要！）
