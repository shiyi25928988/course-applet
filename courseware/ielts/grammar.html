📐 语法精讲

时态语态 · 从句 · 虚拟语气 · 倒装 · 非谓语动词

💡 语法备考建议

雅思不直接考语法，但听说读写都要用对语法 · 写作GRA和口语GRA各占25% · 重点掌握：时态、从句、非谓语、虚拟语气、倒装 · 不求复杂，但求准确

⏰ 时态 Tenses

✅ 一般现在时 Present Simple

  • 客观真理、科学事实
  • 习惯性动作、经常性状态
✓ Water boils at 100°C.
✓ I usually go to school by bus.

✅ 现在完成时 Present Perfect

  • 过去发生，对现在有影响
  • 经历、经验
  • 从过去持续到现在的动作
✓ Technology has transformed our lives. (写作常用)
✓ I have never been abroad. (口语常用)

⚠️ 常见时态错误

✗ I am study English every day.
✓ I study English every day.

✗ I have seen him yesterday.
✓ I saw him yesterday.

📦 从句 Clauses

📌 定语从句 Relative Clauses（写作必备！）

who 指人 · which 指物 · that 指人或物 · where 指地点 · when 指时间 · whose 表所属

✓ Students who study abroad can gain a broader perspective.
✓ The internet, which was invented decades ago, has revolutionized communication.
✓ There are many reasons why people choose to live in cities.

📌 状语从句 Adverbial Clauses

原因：because, since, as · 让步：although, though, even though · 条件：if, unless, provided that · 时间：when, while, as soon as

✓ Although technology brings many benefits, it also has drawbacks. (万能句型！)
✓ Since the cost of living is high, many people struggle to make ends meet.

📌 名词从句 Noun Clauses

that / whether / what / how / why 引导的从句作主语或宾语

✓ That education plays a vital role in development is undeniable.
✓ Many people believe that government should invest more in healthcare.
✓ It is important that we protect the environment.

🔄 非谓语动词 Non-finite Verbs

📌 分词作状语 Participle Phrases（提升写作逼格！）

现在分词表主动、进行 · 过去分词表被动、完成

✓ Working from home, people can enjoy a more flexible schedule.
✓ Compared with the past, life today is much more convenient.
✓ Having studied the data, researchers drew an important conclusion.

📌 不定式作主语/宾语 Infinitive

✓ To solve this problem, we need to take effective measures.
✓ It is essential to protect endangered species.

🔀 虚拟语气 Subjunctive Mood

📌 If 条件句虚拟

类型 If 从句 主句
与现在相反 did / were would / could + do
与过去相反 had done would / could + have done
与将来相反 did / should do / were to do would / could + do
✓ If I had more time, I would learn a new language. (口语常用)
✓ If governments had acted earlier, the crisis would have been prevented.

🔄 倒装 Inversion

📌 否定词前置倒装（写作加分句型！）

Never / Rarely / Seldom / Hardly / Only + 状语 / Not only...but also...

✓ Never in history has technology advanced so rapidly.
✓ Only in this way can we solve this problem effectively.
✓ Not only does technology bring convenience, but it also creates new jobs.

⚠️ 雅思写作常见语法错误

❌ 主谓不一致

✗ Studying abroad have many advantages.
✓ Studying abroad has many advantages.

❌ 句子残缺（碎片句）

✗ Because the cost of living is high. So many people struggle.
✓ Because the cost of living is high, many people struggle.

❌ 逗号连接句（Comma Splice）

✗ Technology is useful, it also has problems.
✓ Technology is useful, but it also has problems.
✓ Although technology is useful, it also has problems.

❌ 冠词错误

✗ The internet has changed the life.
✓ The internet has changed life.
✓ The internet has changed people's lives.
