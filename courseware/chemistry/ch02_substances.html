第2章 身边的化学物质
🎯学习目标
  • 了解空气的组成，掌握氧气和二氧化碳的性质
  • 掌握实验室制取氧气和二氧化碳的方法

2.1 性质活泼的氧气

氧气的物理性质

无色无味气体，密度比空气略大（ρ=1.429g/L）。不易溶于水。液态和固态氧为淡蓝色。

氧气的化学性质

化学性质比较活泼，具有氧化性（助燃性）。能与多种物质反应：

  • C + O₂ → CO₂（碳在氧气中剧烈燃烧，发出白光）
  • S + O₂ → SO₂（硫燃烧产生蓝紫色火焰）
  • 3Fe + 2O₂ → Fe₃O₄（铁在氧气中剧烈燃烧，火星四射）
  • 4P + 5O₂ → 2P₂O₅（磷燃烧产生大量白烟）

氧气的实验室制法

原理：用含氧化合物分解制取。

2H₂O₂ →(MnO₂) 2H₂O + O₂↑ 2KMnO₄ →(加热) K₂MnO₄ + MnO₂ + O₂↑

催化剂（MnO₂）：改变化学反应速率，本身质量和化学性质不变。

收集方法：排水法（不易溶于水）或向上排空气法（密度比空气大）。

验满：将带火星木条放在集气瓶口，木条复燃则满。

2.2 奇妙的二氧化碳

CO₂的性质

无色无味气体，密度比空气大（约1.5倍），能溶于水。

化学性质：不能燃烧，也不支持燃烧。能与水反应：CO₂ + H₂O → H₂CO₃

能使澄清石灰水变浑浊（检验CO₂的方法）：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O

温室效应：大气中CO₂含量增加导致全球气候变暖。

干冰（固态CO₂）升华吸热，可用于人工降雨、舞台烟雾。

CO₂的实验室制法

原理：用石灰石（或大理石）与稀盐酸反应。

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

收集方法：向上排空气法（密度比空气大，能溶于水）。

验满：将燃着木条放瓶口，木条熄灭则满。

2.3 自然界中的水

水的组成

电解水实验证明水由氢元素和氧元素组成。

2H₂O →(通电) 2H₂↑ + O₂↑

正极产生O₂（体积小），负极产生H₂（体积大）。V(H₂) : V(O₂) = 2 : 1。

氢气燃烧：2H₂ + O₂ →(点燃) 2H₂O（产生淡蓝色火焰，验证了水的组成）。

水的净化

净化方法：沉淀 → 过滤 → 吸附 → 蒸馏（净化程度递增）。

硬水与软水：含较多钙镁化合物的水为硬水。用肥皂水鉴别：泡沫多的是软水，泡沫少、有浮渣的是硬水。硬水软化的方法：煮沸或蒸馏。

🔬 基础实验

基础实验1：氧气的制取与性质

用过氧化氢溶液和二氧化锰制取氧气，用排水法收集。验证氧气的助燃性：将带火星木条伸入集气瓶，木条复燃。

基础实验2：二氧化碳的制取与性质

用大理石和稀盐酸制取CO₂。将CO₂通入紫色石蕊溶液（变红→碳酸的酸性），再加热（变回紫色→碳酸分解）。通入澄清石灰水验证。

📝 章节练习题

1检验二氧化碳常用的试剂是____。
✅ B。CO₂使澄清石灰水变浑浊（生成CaCO₃沉淀），这是检验CO₂的特征反应。
2电解水时，正极产生的气体是____。
✅ B。电解水：正氧负氢，V(H₂):V(O₂)=2:1。
3实验室制取CO₂不用稀硫酸的原因是____。
✅ B。稀硫酸与CaCO₃反应生成微溶的CaSO₄，覆盖在大理石表面，阻止反应继续进行。
4收集氧气不能用____。
✅ C。O₂密度比空气大，不能用向下排空气法。可用排水法和向上排空气法。
5区别硬水和软水可用____。
✅ B。加肥皂水：泡沫多的是软水，泡沫少且有浮渣的是硬水。
