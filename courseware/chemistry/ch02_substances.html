2.1 空气的组成

空气的成分

空气是一种混合物，主要成分按体积分数：

成分体积分数主要用途
氮气 N₂约 78%保护气（食品充氮防腐）、制氨、液氮冷冻
氧气 O₂约 21%呼吸、助燃、炼钢、医疗急救
稀有气体约 0.94%氩气作保护气、氦气充气球、氖气制霓虹灯
二氧化碳 CO₂约 0.03%光合作用、灭火、制冷（干冰）
其他气体和杂质约 0.03%水蒸气、尘埃等
💡 记忆口诀：氮七八、氧二一、稀有零点九四、二氧化碳零点零三。

拉瓦锡测定空气组成的实验

拉瓦锡（1777年）通过将汞在密闭容器中加热 12 天，发现空气不是单一物质，而是由氧气（约占 1/5）和氮气（约占 4/5）组成的。

现代测定法——红磷燃烧法：在密闭容器中燃烧红磷，消耗氧气，水进入容器约占 1/5 体积，证明 O₂ 约占空气体积的 1/5。

4P + 5O₂ —点燃→ 2P₂O₅

空气污染与保护

主要污染物

SO₂（煤燃烧）→ 酸雨

NO₂（汽车尾气）→ 酸雨、光化学烟雾

CO（不完全燃烧）→ 有毒

PM2.5 / PM10 → 雾霾

O₃（地面臭氧）→ 刺激呼吸道

防治措施

使用清洁能源（太阳能、风能）

汽车加装催化转化器

工业废气处理达标后排放

植树造林，增加绿化面积

减少化石燃料的使用

2.2 性质活泼的氧气

氧气的物理性质

通常状况下：无色无味气体，密度比空气略大（ρ = 1.429 g/L），不易溶于水（1L 水约溶解 30mL O₂）。

在 101kPa、-183°C 时液化为淡蓝色液态氧，-218°C 时凝固为淡蓝色固态氧。

氧气的化学性质 — 活泼的氧化剂

氧气的化学性质比较活泼，具有氧化性（助燃性），能与多种物质发生反应。

反应物化学方程式实验现象注意事项
碳 C C + O₂ → CO₂ 在空气中：红热；在氧气中：剧烈燃烧，发出白光 生成气体使澄清石灰水变浑浊
硫 S S + O₂ → SO₂ 在空气中：淡蓝色火焰；在氧气中：明亮蓝紫色火焰 生成刺激性气味气体 SO₂，集气瓶底留少量水吸收
铁 Fe 3Fe + 2O₂ → Fe₃O₄ 剧烈燃烧，火星四射，生成黑色固体 集气瓶底必须留水或铺沙（防止高温熔融物炸裂瓶底）
磷 P 4P + 5O₂ → 2P₂O₅ 产生大量白烟（P₂O₅ 固体小颗粒） 白烟是五氧化二磷固体，不是白雾
镁 Mg 2Mg + O₂ → 2MgO 发出耀眼白光，生成白色粉末 用于照明弹、烟花
⚠️ 易错提醒：氧气具有助燃性，但不是可燃物！不要把"助燃性"和"可燃性"混淆。氧气本身不能燃烧，它帮助其他物质燃烧。

氧气的实验室制法

🧪 方法一：H₂O₂ 分解

原理：

2H₂O₂ —MnO₂→ 2H₂O + O₂↑

优点：不需加热，操作简便

MnO₂ 是催化剂：改变化学反应速率，自身的质量和化学性质在反应前后不变

🔥 方法二：KMnO₄ 加热

原理：

2KMnO₄ —加热→ K₂MnO₄ + MnO₂ + O₂↑

注意：试管口要略向下倾斜（防止冷凝水回流炸裂试管）；导管口放一团棉花（防止 KMnO₄ 粉末进入导管）

收集与验满

收集方法：

① 排水法（O₂ 不易溶于水）—— 收集到的气体更纯净

② 向上排空气法（O₂ 密度比空气大）—— 操作更简便，但纯度略低

验满：将带火星的木条放在集气瓶口，木条复燃则满。

放置：集满 O₂ 的集气瓶应正放（O₂ 密度比空气大），盖上玻璃片。

催化剂与催化作用

催化剂（Catalyst）：在化学反应中能改变其他物质的反应速率，而自身的质量和化学性质在反应前后不变的物质。

💡 关键理解：
① 催化剂改变速率（可加快，也可减慢），不是"增加"速率
② 催化剂不参与最终产物的组成，但可能参与中间过程
③ 催化剂不能改变反应的产率和平衡
④ 生物体内的催化剂叫做酶（如消化酶）

2.3 奇妙的二氧化碳

物理性质

无色无味气体

密度比空气大（约 1.5 倍）

能溶于水（1 体积水约溶 1 体积 CO₂）

固态 CO₂ 叫"干冰"，升华时吸热，用于人工降雨、舞台烟雾

化学性质

不能燃烧，也不支持燃烧（可用于灭火）

与水反应：CO₂ + H₂O → H₂CO₃

与石灰水反应：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O（检验 CO₂）

无毒，但含量过高会使人窒息

CO₂ 的实验室制法

原理：石灰石（或大理石，主要成分 CaCO₃）与稀盐酸反应

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
🔍 为什么不用其他酸？
• 不用稀硫酸：生成微溶的 CaSO₄ 覆盖在 CaCO₃ 表面，阻止反应继续进行
• 不用浓盐酸：浓盐酸有挥发性，使收集的 CO₂ 不纯（混有 HCl 气体）
• 不用碳酸钠：反应速率太快，不便于控制

收集方法：向上排空气法（CO₂ 密度比空气大，且能溶于水，不能用排水法）

验满：将燃着的木条放在瓶口，木条熄灭则满。

自然界中的碳循环

CO₂ 的产生

① 动植物的呼吸作用

② 化石燃料的燃烧

③ 微生物的分解作用

④ 火山喷发

CO₂ 的消耗

① 植物的光合作用

② 海洋的吸收溶解

③ 形成碳酸盐岩（石灰岩）

🌍 温室效应：大气中 CO₂ 含量增加 → 全球气候变暖 → 冰川融化、海平面上升、极端天气增多。应对措施：节能减排、使用清洁能源、植树造林。

2.4 自然界中的水

水的组成 — 电解水实验

电解水实验证明：水由氢元素和氧元素组成。

2H₂O —通电→ 2H₂↑ + O₂↑
电极产生气体体积比检验方法
正极（+）氧气 O₂1带火星木条复燃
负极（-）氢气 H₂2气体燃烧产生淡蓝色火焰
💡 记忆口诀：正氧负氢，氢二氧一（V(H₂):V(O₂) = 2:1）。正极产生氧气，负极产生氢气。

氢气

物理性质：无色无味气体，密度最小的气体（约空气的 1/14），难溶于水。

化学性质：可燃性（燃烧产物只有水，是清洁能源）

2H₂ + O₂ —点燃→ 2H₂O
⚠️ 氢气验纯：点燃氢气前必须验纯！不纯的氢气点燃会发生爆炸。验纯方法：用排水法收集一小试管氢气，移近火焰，听到轻微的"噗"声则纯，尖锐的爆鸣声则不纯。

水的净化

方法除去杂质类型净化程度
沉淀较大不溶性固体颗粒低
过滤不溶性固体中
吸附（活性炭）色素、异味、部分可溶性杂质较高
蒸馏几乎所有杂质（包括可溶性矿物质）最高

净化程度：沉淀 → 过滤 → 吸附 → 蒸馏（逐步提高）

硬水与软水

定义

硬水：含有较多可溶性钙、镁化合物的水

软水：不含或含较少可溶性钙、镁化合物的水

鉴别与软化

鉴别：加肥皂水振荡——泡沫多的是软水，泡沫少、有浮渣的是硬水

软化：① 煮沸（生活常用）② 蒸馏（实验室）

硬水的危害：洗衣服浪费肥皂、锅炉结垢易爆炸

水资源保护

节约用水

地球水资源总量虽大，但淡水资源仅占 2.53%，其中可利用的河水、湖泊水和浅层地下水不到1%

节水措施：使用节水器具、一水多用、工业循环用水

防治水污染

污染源：工业废水、农业农药化肥、生活污水

防治：工业废水处理达标后排放、合理使用化肥农药、生活污水集中处理

🔬 基础实验

基础实验1：氧气的制取与性质实验

📋 实验步骤

① 检查装置气密性

② 加药品（H₂O₂ + MnO₂ 或 KMnO₄）

③ 用排水法或向上排空气法收集

④ 用带火星木条验满

⑤ 进行氧气性质实验（木炭、硫、铁丝燃烧）

⚠️ 注意事项

用 KMnO₄ 时试管口略向下倾斜

导管口放棉花防堵塞

实验结束先撤导管再撤酒精灯（防倒吸）

铁丝燃烧时瓶底留水或铺沙

基础实验2：二氧化碳的制取与性质实验

📋 实验步骤

① 加大理石于锥形瓶

② 从长颈漏斗加入稀盐酸

③ 用向上排空气法收集 CO₂

④ 验满：燃着木条放瓶口

🧪 性质验证

① 通入紫色石蕊溶液 → 变红（H₂CO₃ 呈酸性）→ 加热 → 变回紫（H₂CO₃ 分解）

② 通入澄清石灰水 → 变浑浊（CaCO₃ 沉淀）

③ 倒入燃着蜡烛的烧杯 → 蜡烛熄灭（CO₂ 不助燃）

📝 章节练习题

1空气中体积分数约为 21% 的气体是____。
✅ B。空气成分（体积分数）：N₂ 78%，O₂ 21%，稀有气体 0.94%，CO₂ 0.03%。
2检验二氧化碳常用的试剂是____。
✅ B。CO₂ 使澄清石灰水变浑浊（Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O），这是检验 CO₂ 的特征反应。
3电解水时，正极产生的气体是____。
✅ B。正氧负氢，氢二氧一。V(H₂):V(O₂) = 2:1。
4实验室制取 CO₂ 不用稀硫酸的原因是____。
✅ B。稀硫酸与 CaCO₃ 反应生成微溶的 CaSO₄，覆盖在大理石表面，阻止反应继续进行。
5收集氧气不能用的方法是____。
✅ C。O₂ 密度比空气大，用向上排空气法；不易溶于水，可用排水法。不能用向下排空气法。
6区别硬水和软水可用____。
✅ B。加肥皂水：泡沫多的是软水，泡沫少且有浮渣的是硬水。
7铁丝在氧气中燃烧时，集气瓶底要留少量水或铺沙，其目的是____。
✅ B。铁丝燃烧生成高温熔融的 Fe₃O₄，溅落到瓶底可能导致集气瓶炸裂，所以需要留水或铺沙。
8关于催化剂的说法正确的是____。
✅ B。催化剂改变速率（可加快或减慢），自身的质量和化学性质不变。催化剂不能改变生成物的量，也不是所有反应都需要催化剂。
