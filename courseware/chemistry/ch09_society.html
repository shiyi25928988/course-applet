9.1 能源的综合利用

能源的分类

按是否可以再生分类：

  • 不可再生能源——煤、石油、天然气（三大化石燃料）、核能（铀矿）。用一点少一点，储量有限。
  • 可再生能源——太阳能、风能、水能、地热能、潮汐能、生物质能。可以不断再生，取之不尽。

按使用类型分类：常规能源（煤、石油、天然气、水能）和新能源（太阳能、风能、核能、氢能、地热能、潮汐能、生物质能）。

三大化石燃料详细对比表

🔥 燃烧热值对比：天然气（约35~50MJ/m³）> 石油（约42MJ/kg）> 煤（约20~30MJ/kg）。天然气热值高且燃烧产物主要是CO₂和H₂O，几乎不含SO₂，是最清洁的化石燃料。

煤的综合利用

煤不仅是燃料，更是重要的化工原料。通过化学加工可以将煤转化为更有价值的产品：

  • 煤的干馏（焦化）——将煤隔绝空气加强热，分解得到焦炭（用于炼铁）、煤焦油（含数百种有机物，是重要化工原料）、焦炉煤气（含H₂、CH₄、CO等，可用作燃料）。属于化学变化。
  • 煤的气化——煤与水蒸气在高温下反应生成水煤气（CO + H₂的混合气体），是重要的气体燃料和化工原料。
  • 煤的液化——将煤转化为液体燃料（类似石油产品），如煤制油（直接液化）和煤间接液化（先气化再合成）。
C + H₂O →(高温) CO + H₂（水煤气的生成）

石油的分馏

石油是复杂的液态混合物，主要含多种碳氢化合物。利用各成分沸点不同，通过加热分离的方法叫做分馏（属于物理变化）。

分馏产品沸点范围碳原子数主要用途
石油气低于40℃C₁~C₄液化石油气（LPG）、化工原料
汽油40~200℃C₅~C₁₂汽车燃料
煤油175~275℃C₁₂~C₁₆航空燃料、煤油灯
柴油250~400℃C₁₅~C₁₈卡车、船舶、发电机燃料
润滑油350℃以上C₁₆~C₂₀机械润滑
沥青残留物C₂₀以上铺路、防水材料

可燃冰（天然气水合物）

可燃冰是甲烷分子被水分子包裹形成的笼状晶体化合物，外观似冰，遇火可燃。主要分布于海底沉积物和永久冻土带。

特点：储量大（全球有机碳储量约为煤、石油、天然气总和的两倍）、能量密度高（1m³可燃冰≈164m³天然气）、燃烧污染少。

挑战：开采技术难度大，可能引发海底滑坡、甲烷泄漏（甲烷温室效应是CO₂的20多倍）等环境风险。2017年中国在南海成功试采可燃冰，成为全球首个实现海域可燃冰稳定开采的国家。

各种新能源优缺点对比

能源类型优点缺点利用方式

氢能——理想的绿色能源

氢气（H₂）作为能源具有三大优势：

  • 热值高——约142MJ/kg，约为汽油的3倍、焦炭的4.5倍
  • 零污染——燃烧产物只有水：2H₂ + O₂ →(点燃) 2H₂O
  • 来源广——水是氢的"仓库"（电解水：2H₂O →(通电) 2H₂↑ + O₂↑）

当前制约氢能推广的主要因素：制氢成本高（电解水耗电大）、储运困难（氢气密度小，需高压或低温液化）、基础设施不足（加氢站少）。

🇨🇳 中国的能源战略：我国能源结构正在从"以煤为主"向多元化、清洁化转型。
- 2030年前实现碳达峰（碳排放量达到峰值后不再增长）
- 2060年前实现碳中和（碳排放量=碳吸收量，净零排放）
- 大力发展光伏、风电、核电等清洁能源，逐步降低煤炭消费占比
- 推广新能源汽车，逐步淘汰燃油车

9.2 新型材料的研制

材料分类树状图

材料
├── 金属材料——纯金属（Fe、Cu、Al、Au）、合金（钢、黄铜、铝合金、钛合金）
├── 无机非金属材料——玻璃、水泥、陶瓷、耐火材料
├── 有机高分子材料
│   ├── 天然高分子——棉花、羊毛、蚕丝、天然橡胶、淀粉
│   └── 合成高分子——塑料、合成纤维、合成橡胶
└── 复合材料——玻璃钢、碳纤维复合材料、钢筋混凝土

无机材料

金属材料：包括纯金属和合金。合金是两种或两种以上金属（或金属与非金属）熔合而成，具有金属特性的物质。合金的硬度一般大于纯金属，熔点一般低于纯金属，性能更优。

无机非金属材料：

  • 玻璃——主要成分：Na₂SiO₃、CaSiO₃、SiO₂。普通玻璃是钠钙玻璃，钢化玻璃强度更高，防弹玻璃为多层复合。
  • 水泥——主要成分：硅酸三钙(3CaO·SiO₂)、硅酸二钙(2CaO·SiO₂)、铝酸三钙(3CaO·Al₂O₃)。与水混合后发生水合反应，硬化成坚硬固体。
  • 陶瓷——由黏土等烧制而成，耐高温、耐腐蚀、绝缘性好。从日用陶瓷到航天陶瓷，应用广泛。

有机合成材料——塑料

塑料的分类：

类型结构特点代表物特性
热塑性塑料链状结构聚乙烯（PE）、聚氯乙烯（PVC）、聚丙烯（PP）、聚苯乙烯（PS）受热软化、冷却变硬，可反复加工
热固性塑料网状结构酚醛塑料（电木）、脲醛塑料、环氧树脂成型后受热不再软化，不可重复加工
🔥 塑料的鉴别——燃烧法：
聚乙烯（PE）：燃烧时火焰上黄下蓝，有蜡味（像蜡烛燃烧），熔融滴落。常用于食品袋、保鲜膜。
聚氯乙烯（PVC）：燃烧时火焰上黄下绿，有刺激性气味（HCl气体），离火即灭。有毒，不可用于食品包装！
聚酯（涤纶）：燃烧时冒黑烟，有芳香气味，残余物为黑色硬块。

天然纤维与合成纤维的鉴别

纤维种类燃烧现象气味残余物
棉（植物纤维）易燃，火焰黄色烧纸气味灰白色灰烬，手捻成粉末
羊毛/蚕丝（动物纤维）燃烧缓慢，火焰小烧焦羽毛气味黑色脆块，手捻成粉末
涤纶（合成纤维）先熔后燃，冒黑烟芳香气味黑色硬块，不易捻碎
锦纶/尼龙（合成纤维）边熔边燃芹菜气味浅褐色硬块

"白色污染"与应对

白色污染是指废弃塑料制品对环境造成的污染。塑料在自然界中难以降解（普通塑料在土壤中可存在数百年），造成严重的环境问题。

应对措施：

  • 减少使用——少用一次性塑料制品，自带购物袋、水杯
  • 回收利用——塑料分类回收，可再生造粒制成新制品
  • 开发可降解塑料——如聚乳酸（PLA）、聚羟基脂肪酸酯（PHA）等生物可降解塑料
  • 焚烧发电——在严格控污条件下焚烧塑料，利用热能发电

复合材料

将两种或两种以上不同性质的材料通过物理或化学方法复合而成，兼具各组分优点，性能优于单一材料。

复合材料组成特点应用
玻璃钢玻璃纤维 + 塑料（树脂）质轻、强度高、耐腐蚀船体、管道、储罐、汽车外壳
碳纤维复合材料碳纤维 + 树脂/金属/陶瓷强度极高、质轻、耐高温飞机、F1赛车、高端球拍、钓鱼竿
钢筋混凝土钢筋 + 混凝土抗压+抗拉、耐久建筑、桥梁、大坝
轮胎橡胶 + 钢丝/纤维帘布耐磨、弹性好、强度高汽车、飞机轮胎

前沿新材料

  • 纳米材料——指在三维空间中至少有一维处于纳米尺度（1~100nm）的材料。纳米材料具有小尺寸效应、表面效应等特殊性质。应用：纳米涂层（不沾锅）、纳米药物载体、纳米催化剂、纳米传感器。
  • 形状记忆合金——能在特定温度下恢复原有形状的合金。如镍钛合金（Nitinol），用于航空航天展开天线、医疗支架、牙齿矫正丝、眼镜架等。
  • 功能高分子材料——具有特殊功能的高分子。如高吸水性树脂（尿不湿中的吸水材料，可吸收自身重量数百倍的水）、导电高分子、光敏高分子等。

9.3 环境污染的防治

酸雨

定义：pH< 5.6的降水（正常雨水因溶有CO₂，pH约为5.6）。

酸雨的形成过程详解：

化石燃料燃烧产生SO₂和NOₓ，排入大气后：

SO₂ + H₂O → H₂SO₃（亚硫酸） 2H₂SO₃ + O₂ → 2H₂SO₄（硫酸——酸雨的主要成分） 2NO + O₂ → 2NO₂， 3NO₂ + H₂O → 2HNO₃ + NO

酸雨的危害：

  • 腐蚀建筑物和雕塑——大理石（CaCO₃）与酸反应：CaCO₃ + H₂SO₄ → CaSO₄ + H₂O + CO₂↑
  • 酸化土壤——土壤中Ca²⁺、Mg²⁺等养分流失，Al³⁺溶出毒害植物根系
  • 酸化水体——鱼类等水生生物大量死亡
  • 破坏森林——树木叶片受损，生长受阻甚至死亡
  • 危害人体健康——酸雾刺激呼吸道

防治措施：使用清洁能源、燃煤脱硫、汽车尾气净化（三元催化转化器）。

温室效应

大气中的CO₂、CH₄等温室气体允许太阳短波辐射穿过，却吸收地面长波辐射，使热量滞留在大气层中，导致全球气温升高。

主要温室气体：CO₂（贡献最大）、CH₄、N₂O、O₃、氟利昂（CFCs）。

危害：全球变暖 → 冰川融化 → 海平面上升 → 淹没沿海低地；极端天气增多；生态失衡。

应对措施：减少化石燃料使用、开发新能源、植树造林（增加碳汇）、节能减排。

PM2.5与空气污染

PM2.5是指大气中直径≤2.5微米的可吸入颗粒物。由于粒径小，可以深入肺泡甚至进入血液循环，对人体健康危害极大。

来源：化石燃料燃烧、汽车尾气、工业排放、建筑扬尘、二次反应生成（SO₂、NOₓ转化）。

危害：引发呼吸系统疾病（哮喘、支气管炎）、心血管疾病，甚至肺癌。PM2.5也是雾霾的主要成因。

防治：使用清洁能源、机动车限行、工业废气治理、工地扬尘管控、佩戴口罩（N95）。

空气质量指数（AQI）综合了PM2.5、PM10、SO₂、NO₂、O₃、CO六项指标，分为优（0~50）、良（51~100）、轻度污染、中度污染、重度污染、严重污染六个等级。

水体富营养化

工业废水、生活污水和农业化肥中含大量氮（N）、磷（P）元素，排入水体后，造成藻类和水生植物疯狂繁殖，消耗水中溶解氧，导致鱼类等水生生物死亡，水质恶化。

典型现象：水华（淡水中的蓝藻爆发）和赤潮（海水中的藻类爆发）。

防治：污水处理达标排放、合理使用化肥、推广无磷洗衣粉。

重金属污染

汞（Hg）、镉（Cd）、铅（Pb）、铬（Cr）、砷（As）等重金属及其化合物，通过工业废水、废渣排入环境，造成严重污染。

重金属在环境中不可降解，通过食物链富集放大，最终危害人体健康。

著名案例：

  • 日本水俣病——有机汞污染导致严重神经系统疾病
  • 日本痛痛病——镉污染导致骨骼软化、剧痛
  • 血铅超标——铅污染影响儿童智力发育
⚠️ 重金属中毒急救（重复强调）：误食重金属盐后，应立即喝大量牛奶、蛋清或豆浆，使蛋白质与重金属离子结合而沉淀，减少人体吸收，然后尽快就医。这利用了蛋白质遇重金属盐变性的原理。

绿色化学

绿色化学又称"环境友好化学"，其核心思想是在化学品的生产和使用过程中，从源头上减少或消除污染。

绿色化学12条原则（核心要点）：

  1. 防止污染优于治理污染（源头预防）
  2. 提高原子经济性——尽量让所有反应物转化为产物
  3. 尽量使用和生成无毒无害物质
  4. 设计更安全的化学品
  5. 使用安全的溶剂和助剂
  6. 提高能源效率——常温常压下反应
  7. 使用可再生原料
  8. 减少不必要的衍生化步骤
  9. 使用催化剂（优于化学计量试剂）
  10. 设计可降解产物
  11. 实时分析以防止污染
  12. 使用更安全的化学工艺，防止事故发生
🌍 碳达峰与碳中和：
碳达峰：某个地区或行业的年度CO₂排放量达到历史最高值，之后进入持续下降的过程。中国承诺2030年前实现碳达峰。
碳中和：企业、团体或个人通过节能减排、植树造林、购买碳信用等方式，抵消自身产生的CO₂排放量，实现"净零排放"。中国承诺2060年前实现碳中和。
实现路径：能源转型（清洁能源替代化石燃料）、工业减排、绿色交通、建筑节能、碳捕集利用与封存（CCUS）、生态碳汇。

📝 章节练习题

1下列属于可再生能源的是____。
✅ D。煤、石油、天然气是三大化石燃料，属于不可再生能源。太阳能、风能、水能等是可再生能源，取之不尽用之不竭。
2造成酸雨的主要气体是____。
✅ B。SO₂和NOₓ是酸雨的主要成因。SO₂在空气中被氧化最终生成H₂SO₄。CO₂引起温室效应，CO有毒，O₂是助燃气体。
3"白色污染"主要指____。
✅ B。白色污染指废弃塑料制品对环境造成的污染。塑料在自然界中难以降解，大量废弃塑料严重污染土壤和海洋。
4天然气的主要成分是____。
✅ C。天然气主要成分是甲烷（CH₄），属于三大化石燃料之一，也是较清洁的化石燃料，燃烧产物主要是CO₂和H₂O。
5下列不属于合成高分子材料的是____。
✅ C。天然橡胶属于天然高分子材料，从橡胶树中采集。塑料、合成纤维（涤纶、锦纶等）、合成橡胶是三大合成高分子材料。
6下列做法不符合"绿色化学"理念的是____。
✅ B。大量使用化肥农药会造成水体富营养化、土壤污染等环境问题，不符合绿色化学"从源头减少污染"的理念。其他选项均有利于环境保护。
7鉴别羊毛和棉布最简便的方法是____。
✅ C。灼烧法是最简便的鉴别方法。羊毛是蛋白质，灼烧有烧焦羽毛气味；棉布是纤维素，灼烧有烧纸气味。合成纤维灼烧则先熔后燃，残余物为硬块。
8关于碳达峰和碳中和，下列说法错误的是____。
✅ D。碳中和不是"零排放"，而是指排放的CO₂量与通过各种方式（植树造林、CCUS技术等）吸收的CO₂量相等，实现"净零排放"。人类活动不可能完全不排放CO₂。
