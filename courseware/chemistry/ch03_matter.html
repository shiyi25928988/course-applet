第3章 物质构成的奥秘
🎯学习目标
  • 了解分子、原子、离子的概念
  • 掌握常见元素的符号和名称
  • 了解元素周期表的基本结构
  • 学会书写化学式，计算化合价和相对分子质量

3.1 构成物质的基本微粒

分子

分子是保持物质化学性质的最小粒子。

分子的特性：①质量体积都很小 ②不断运动（温度越高运动越快）③分子间有间隔 ④同种分子性质相同，不同种分子性质不同。

由分子构成的物质：水(H₂O)、二氧化碳(CO₂)、氧气(O₂)等。

原子

原子是化学变化中的最小粒子。

原子由原子核（质子+中子）和核外电子构成。

核电荷数 = 质子数 = 核外电子数（原子不显电性）。

相对原子质量：以一种碳原子质量的1/12为标准，其他原子质量与它的比值。

相对原子质量 ≈ 质子数 + 中子数。

离子

带电的原子或原子团叫做离子。

阳离子：原子失去电子，带正电。如Na⁺、Mg²⁺。

阴离子：原子得到电子，带负电。如Cl⁻、O²⁻。

离子化合物：由阴阳离子相互作用形成的化合物。如NaCl。

3.2 组成物质的化学元素

元素

元素：具有相同质子数（核电荷数）的一类原子的总称。

地壳中含量前四位：氧(O) > 硅(Si) > 铝(Al) > 铁(Fe)。

人体中含量最多的是氧(O)，其次是碳、氢。

空气中含量最多的是氮(N)，其次是氧。

元素周期表

共有7个周期（横行）、16个族（纵列）。

原子序数 = 质子数 = 核电荷数。

元素分为：金属元素（如Na、Fe）、非金属元素（如O、C）、稀有气体元素（如He、Ne）。

3.3 物质的组成

化合价

化合价表示原子之间相互化合的数目。

常见化合价口诀：一价钾钠氯氢银，二价氧钙钡镁锌；三铝四硅五价磷，二三铁、二四碳；铜汞二价最常见。

化合物中：正负化合价代数和为零。单质中元素化合价为零。

化学式的计算

相对分子质量：化学式中各原子的相对原子质量之和。

如：H₂O的相对分子质量 = 1×2 + 16 = 18。

元素质量比：H₂O中 m(H):m(O) = (1×2):16 = 1:8。

元素质量分数：ω(H) = 2/18 × 100% ≈ 11.1%。

📝 章节练习题

1保持水的化学性质的最小粒子是____。
✅ C。分子是保持物质化学性质的最小粒子，水的化学性质由水分子保持。
2决定元素种类的是____。
✅ B。质子数（核电荷数）决定元素种类。质子数相同的一类原子属同种元素。
3地壳中含量最多的金属元素是____。
✅ B。地壳中含量：O>Si>Al>Fe>Ca。Al是含量最多的金属元素。
4H₂O中氢元素和氧元素的质量比是____。
✅ C。m(H):m(O)=(1×2):16=2:16=1:8。
