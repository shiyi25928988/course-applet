4.1 常见的化学反应——燃烧

燃烧的定义与条件

燃烧：可燃物与氧气发生的一种发光、放热的剧烈的氧化反应。

燃烧必须同时具备三个条件（缺一不可）：

  1. 可燃物——如木材、煤、天然气、酒精、汽油等
  2. 与氧气（或空气）接触——氧气是助燃剂，本身不燃烧
  3. 温度达到着火点——着火点是物质燃烧所需的最低温度，是物质的固有属性，不可改变

注意：着火点是物质本身的性质，一般不会改变。不同物质的着火点不同，如白磷着火点约40°C，红磷着火点约240°C。

灭火的原理与方法

灭火的根本原理：破坏燃烧的任一条件，使燃烧反应停止。

灭火原理具体方法生活实例
清除可燃物将可燃物与火源隔离森林火灾开辟隔离带、关闭燃气阀门
隔绝氧气使可燃物与空气隔绝CO₂灭火器灭火、油锅着火盖上锅盖、用湿棉被覆盖
降温至着火点以下用水等冷却燃烧物用水灭火、森林火灾用飞机洒水

燃烧与灭火的辩证关系

燃烧与灭火是一对矛盾，二者对立统一：

  • 燃烧为人类提供能量和光明——烹饪、取暖、发电、工业炼钢等
  • 火灾则造成巨大的生命财产损失
  • 控制燃烧条件的核心在于：利用燃烧时三条件同时满足，灭火时破坏任一条件
  • 人类对火的认识经历了：钻木取火 → 保留火种 → 火柴/打火机 → 电子点火的进化
💡 思考：为什么蜡烛火焰一吹就灭，而煤炉火越吹越旺？
答：蜡烛火焰小，吹气使温度降至着火点以下；煤炉火大，吹气增加氧气供应，反而促进燃烧。

易燃易爆物安全知识

爆炸：可燃物在有限空间内急速燃烧，产生大量气体和热量，气体体积迅速膨胀而引起。

易燃易爆物：易燃液体（汽油、酒精、油漆）、易燃气体（氢气、天然气、液化石油气）、易燃固体（红磷、镁粉、硫磺）。

⚠️ 安全警示：
  • 家中燃气泄漏：立即关闭阀门、开窗通风，切勿开关电器或使用明火！
  • 加油站、面粉厂、煤矿坑道：严禁烟火，粉尘爆炸的危险性极大
  • 油锅着火：盖上锅盖（隔绝氧气），切勿用水浇——油会浮在水面继续燃烧，甚至飞溅
  • 电器着火：先切断电源，再用干粉灭火器或CO₂灭火器，不可用水或泡沫灭火器

常见灭火器类型及适用范围：

灭火器类型灭火原理适用范围不适用范围
干粉灭火器粉末覆盖隔绝氧气一般火灾、油、气、电器精密仪器（粉末有腐蚀性）
CO₂灭火器CO₂密度大，隔绝氧气并降温图书、档案、精密仪器、电器镁等活泼金属火灾（CO₂与Mg反应）
泡沫灭火器泡沫覆盖隔绝氧气油类、木材等一般火灾电器火灾（泡沫含水导电）
水基灭火器降温并隔绝氧气木材、纸张、布料油类、电器、活泼金属

完全燃烧与不完全燃烧

碳的完全燃烧（氧气充足）：C + O₂ →(点燃) CO₂

碳的不完全燃烧（氧气不足）：2C + O₂ →(点燃) 2CO

含碳燃料不完全燃烧会产生一氧化碳（CO），有毒！CO与血红蛋白的结合能力比O₂强约200倍，造成组织缺氧。

⚠️ CO中毒症状：头痛、眩晕、恶心、昏迷，严重时死亡。冬季用煤炉取暖需保持通风，燃气热水器不能安装在浴室内。发现中毒者应立即转移到通风处并拨打120。

缓慢氧化与自燃

缓慢氧化：进行得非常缓慢、甚至不易察觉的氧化反应。如：铁生锈、食物腐败、呼吸作用、金属腐蚀。

自燃：由缓慢氧化引起的自发燃烧。当缓慢氧化产生的热量不能及时散发，使温度逐渐升高达到着火点，就会引发自燃。

常见自燃现象：

  • 煤堆自燃——煤在空气中缓慢氧化，热量积聚升温
  • 白磷自燃——白磷着火点仅40°C，在空气中缓慢氧化放热即能自燃，故白磷保存在水中
  • 浸油的棉纱自燃——油在棉纱中缓慢氧化，棉纱隔热导致热量积聚
💡 对比：燃烧（剧烈氧化）vs 缓慢氧化 vs 自燃（缓慢氧化积累导致）
本质相同——都是氧化反应，区别在于反应的剧烈程度和热量散失速度。

4.2 化学反应中的质量关系

质量守恒定律

质量守恒定律：参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和。

微观解释：化学反应前后，原子的种类、数目、质量都不变。化学反应的本质是原子重新组合的过程。

质量守恒定律的验证实验——红磷燃烧

实验目的：验证化学反应前后物质的总质量是否改变。

实验原理：4P + 5O₂ →(点燃) 2P₂O₅

实验步骤：

  1. 在锥形瓶底部铺一层细沙（防止红磷燃烧时锥形瓶受热不均炸裂）
  2. 在锥形瓶口的橡皮塞上安装一根玻璃管，其上端系牢一个小气球（缓冲气压，防止瓶塞被冲开）
  3. 将锥形瓶放在托盘天平上，用砝码平衡，使天平平衡
  4. 取下锥形瓶，将橡皮塞上的玻璃管加热，引燃红磷
  5. 待锥形瓶冷却后，重新放到托盘天平上，观察天平是否平衡

实验现象：红磷燃烧，产生大量白烟（P₂O₅固体小颗粒），气球先膨胀后变瘪，冷却后天平仍保持平衡。

🔬 实验关键点：
  • 锥形瓶底部铺细沙——防止红磷燃烧时锥形瓶炸裂
  • 气球的作用——缓冲瓶内气压变化，保证装置密闭
  • 必须冷却后称量——热气体膨胀使浮力改变，影响测量准确性
  • 整个装置必须密闭——使反应前后物质都在同一容器内称量

质量守恒定律的"六不变两变一可能变"

这是质量守恒定律最核心的考点框架，务必牢记：

类别内容具体说明
六个不变
（宏观）		元素种类不变反应前后元素种类不变，如水通电分解仍只含H和O
元素质量不变各元素的总质量反应前后相等
物质总质量不变参加反应的各物质质量总和 = 生成的各物质质量总和
六个不变
（微观）		原子种类不变化学反应中原子不改变种类
原子数目不变反应前后每种原子的个数相等
原子质量不变每个原子的质量不发生变化
两个一定变物质种类变反应物和生成物是不同物质
分子种类变反应物分子变为生成物分子
一个可能变分子总数可能变如2H₂+O₂→2H₂O: 3分子→2分子(减少)；H₂+Cl₂→2HCl: 2分子→2分子(不变)

镁条燃烧"增重"之谜

问题：镁条在空气中燃烧后，生成物的质量比镁条的质量大，这是否违反质量守恒定律？

实验：2Mg + O₂ →(点燃) 2MgO

分析：镁条燃烧后质量增加，不违反质量守恒定律！

  • 镁燃烧是镁与空气中的氧气发生了化合反应
  • 参加反应的物质是镁 + 氧气，生成物是氧化镁
  • m(Mg) + m(O₂) = m(MgO)，而m(O₂)没有被计入"反应前"的质量
  • 如果在一个密闭容器中称量镁燃烧前后的总质量，质量不变
💡 关键结论：判断质量守恒定律时，必须考虑所有参加反应和生成的气体的质量。任何"增重"或"减重"的假象，都是因为有气体参与或生成而未被计入。

4.3 化学方程式的书写与应用

化学方程式的书写原则与步骤

书写原则：

  • ① 以客观事实为依据——不能凭空臆造不存在的反应
  • ② 遵守质量守恒定律——反应前后各原子种类和数目必须相等

书写步骤（写→配→注→等）：

  1. 写——写出反应物和生成物的化学式（反应物在左，生成物在右）
  2. 配——配平化学方程式，使反应前后各原子种类和数目相等
  3. 注——注明反应条件（点燃、加热△、高温、催化剂等）和生成物状态（↑气体、↓沉淀）
  4. 等——将短线改为等号

配平方法：

  • 最小公倍数法——找出出现次数多且原子个数相差大的元素，求最小公倍数
  • 观察法——从较复杂的化学式入手，逐步配平
  • 奇数配偶法——当某元素原子个数在反应物和生成物中一奇一偶时使用

化学方程式配平示例

例1：最小公倍数法配平 KClO₃ → KCl + O₂

氧原子在右边为2个，左边为3个，最小公倍数为6。
右边配3：KClO₃ → KCl + 3O₂
左边配2：2KClO₃ → KCl + 3O₂
右边配2：2KClO₃ →(MnO₂,△) 2KCl + 3O₂↑

例2：观察法配平 Fe₂O₃ + CO → Fe + CO₂

观察：每个CO夺走一个O变成CO₂，Fe₂O₃中有3个O原子，需要3个CO。
Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂

例3：奇数配偶法配平 C₂H₂ + O₂ → CO₂ + H₂O

H₂O中O为奇数，先配2：C₂H₂ + O₂ → CO₂ + 2H₂O
配C：C₂H₂ + O₂ → 2CO₂ + 2H₂O
配O：右边2×2+2=6个O，左边需3个O₂
2C₂H₂ + 5O₂ →(点燃) 4CO₂ + 2H₂O

根据化学方程式的计算——详细步骤

计算依据：反应物和生成物各物质间的质量比是固定的（由化学方程式中各物质的相对分子质量之比决定）。

计算步骤（6步法）：

  1. 设——设未知量（设某物质的质量为x）
  2. 写——写出正确的化学方程式并配平
  3. 找——找出相关物质的相对分子质量和已知量、未知量
  4. 列——列出比例式
  5. 解——求解
  6. 答——简明作答

计算实例——完整演示

例题：实验室用碳酸钙与稀盐酸反应制取二氧化碳，若需制取8.8g CO₂，至少需要多少克碳酸钙？

解：

① 设至少需要碳酸钙的质量为x。

② 写化学方程式：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

③ 找相关量：

物质CaCO₃CO₂
相对分子质量40+12+16×3 = 10012+16×2 = 44
质量x8.8g

④ 列比例式：100/44 = x/8.8g

⑤ 解：x = 100 × 8.8g / 44 = 20g

⑥ 答：至少需要20g碳酸钙。

💡 常见错误提醒：
  • 化学方程式未配平就计算——这是最常见的错误！
  • 相对分子质量计算错误——注意化学式中的下标
  • 代入数据时单位不统一——所有质量单位必须一致
  • 已知的是不纯物质的质量——需先换算为纯物质质量

4.4 四大基本反应类型

四大基本反应类型总览

化学反应按反应物和生成物的种类与数量，可分为四大基本反应类型：

反应类型通式特征典型例子
化合反应A + B → AB
多变一		两种或多种物质生成一种物质C + O₂ →(点燃) CO₂
CaO + H₂O → Ca(OH)₂
分解反应AB → A + B
一变多		一种物质分解成两种或多种物质2H₂O →(通电) 2H₂↑ + O₂↑
CaCO₃ →(高温) CaO + CO₂↑
置换反应A + BC → AC + B
单换单		单质+化合物→新单质+新化合物Fe + CuSO₄ → FeSO₄ + Cu
Zn + 2HCl → ZnCl₂ + H₂↑
复分解反应AB + CD → AD + CB
双交换		两种化合物互相交换成分
(需有沉淀/气体/水生成)		NaOH + HCl → NaCl + H₂O
AgNO₃ + NaCl → AgCl↓ + NaNO₃

各反应类型深度解析

1. 化合反应（多变一）

  • 金属+氧气→金属氧化物：3Fe + 2O₂ →(点燃) Fe₃O₄
  • 非金属+氧气→非金属氧化物：S + O₂ →(点燃) SO₂
  • 金属氧化物+水→碱（可溶碱）：CaO + H₂O → Ca(OH)₂
  • 非金属氧化物+水→酸：CO₂ + H₂O → H₂CO₃

2. 分解反应（一变多）

  • 不溶性碱受热分解：Cu(OH)₂ →(△) CuO + H₂O
  • 碳酸盐受热/加酸分解：CaCO₃ →(高温) CaO + CO₂↑
  • 含氧酸盐受热分解：2KMnO₄ →(△) K₂MnO₄ + MnO₂ + O₂↑

3. 置换反应（单换单）

  • 金属+酸→盐+氢气（氢前金属）：Fe + H₂SO₄ → FeSO₄ + H₂↑
  • 金属+盐→新金属+新盐（活泼置换不活泼）：Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag
  • 碳/氢气还原金属氧化物：C + 2CuO →(高温) 2Cu + CO₂↑

4. 复分解反应（双交换，价不变）

  • 酸+碱→盐+水（中和反应）：HCl + NaOH → NaCl + H₂O
  • 酸+盐→新酸+新盐：2HCl + CaCO₃ → CaCl₂ + H₂O + CO₂↑
  • 碱+盐→新碱+新盐：2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄
  • 盐+盐→两种新盐：NaCl + AgNO₃ → AgCl↓ + NaNO₃
📝 复分解反应发生的条件：生成物中至少要有沉淀、气体或水中的一种，否则反应不能发生。此为初中化学判断复分解反应能否发生的"三条件"。

氧化反应与还原反应（交叉分类）

氧化反应和还原反应是从得氧/失氧的角度分类，与四大基本反应类型不属于同一分类标准。

  • 氧化反应：物质与氧发生的反应（得氧）。如：C + O₂ → CO₂
  • 还原反应：含氧化合物中的氧被夺走的反应（失氧）。如：CuO + H₂ → Cu + H₂O
  • 氧化反应和还原反应同时发生，合称氧化还原反应
  • 化合反应不一定是氧化反应（如CaO+H₂O），置换反应一定是氧化还原反应

🔬 基础实验3：物质燃烧的条件

实验：探究燃烧需要的条件

实验目的：通过对比实验，验证燃烧必须同时具备的三个条件。

实验一：探究可燃物是否为必要条件

  • 操作：用镊子夹取一小块石块和一根木条，分别放在酒精灯火焰上加热
  • 现象：木条燃烧，石块不燃烧
  • 结论：燃烧需要可燃物，石块不是可燃物

实验二：探究氧气是否为必要条件

  • 操作：点燃两支蜡烛，用烧杯罩住其中一支
  • 现象：被罩住的蜡烛逐渐熄灭，另一支正常燃烧
  • 结论：燃烧需要氧气，烧杯内氧气耗尽后燃烧停止

实验三：探究温度达到着火点是否为必要条件

  • 操作：用纸折一个小船，盛适量水后放在酒精灯火焰上加热
  • 现象：水沸腾但纸船不燃烧
  • 结论：水不断吸收热量，使纸的温度始终低于其着火点，因此纸不燃烧
  • 进一步实验：不装水的纸船迅速燃烧——说明温度达到着火点是燃烧的必要条件
💡 实验总结：燃烧必须同时具备三个条件——可燃物、氧气、温度达到着火点。三者缺一不可。灭火只需破坏其中任一条件。

📝 章节练习题

1燃烧必须同时具备的三个条件是____。
✅ B。燃烧三条件：可燃物、与氧气（或空气）接触、温度达到着火点。三者缺一不可。
2质量守恒定律适用于____。
✅ B。质量守恒定律只适用于化学变化。物理变化（如物质的三态变化、形状改变）不涉及新物质的生成，不适用质量守恒定律。
3碳不完全燃烧时产生的有毒气体是____。
✅ B。碳不完全燃烧产生一氧化碳（CO），有毒。CO与血红蛋白结合能力比O₂强约200倍，造成组织缺氧，严重时致死。
4化学方程式配平的依据是____。
✅ B。配平的根本依据是质量守恒定律——反应前后各原子种类和数目不变。配平的本质是使化学方程式两边各元素的原子个数相等。
5下列反应中，属于置换反应的是____。
✅ C。置换反应：单质+化合物→新单质+新化合物。A是化合反应，B是分解反应，D是复分解反应（中和反应）。
6镁条在空气中燃烧后，生成物质量比镁条质量大，原因是____。
✅ B。不违反质量守恒定律！镁与空气中O₂反应生成MgO，m(MgO) = m(Mg) + m(O₂)，所以质量增加。若在密闭容器中称量总质量不变。
7家中燃气泄漏时，正确的做法是____。
✅ B。燃气泄漏时应立即关闭阀门、开窗通风。A、C、D都可能产生电火花或明火，引燃燃气与空气的混合物，造成爆炸！
8实验室制取8.8g CO₂，至少需要CaCO₃的质量为____。(Ca-40, C-12, O-16)
✅ C。CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑。CaCO₃相对分子质量100，CO₂为44。设需CaCO₃为x，100/44=x/8.8g，x=20g。
