一、物质与氧气的反应（燃烧） 10个

#化学方程式实验现象备注
1 C + O₂ —点燃→ CO₂ 在空气中红热；在氧气中发出白光 生成气体使石灰水变浑浊
2 4P + 5O₂ —点燃→ 2P₂O₅ 产生大量白烟（P₂O₅固体颗粒） 白烟不是白雾；用于测定空气中O₂含量
3 S + O₂ —点燃→ SO₂ 空气中：淡蓝色火焰；氧气中：明亮蓝紫色火焰 生成刺激性气味气体；集气瓶底留水吸收SO₂
4 3Fe + 2O₂ —点燃→ Fe₃O₄ 剧烈燃烧，火星四射，生成黑色固体 瓶底必须留水或铺沙（防炸裂）
5 2Mg + O₂ —点燃→ 2MgO 发出耀眼白光，生成白色粉末 用于照明弹、烟花
6 2H₂ + O₂ —点燃→ 2H₂O 产生淡蓝色火焰 点燃前必须验纯；产物只有水（清洁能源）
7 2CO + O₂ —点燃→ 2CO₂ 产生蓝色火焰 CO有毒，注意通风
8 2C + O₂ —点燃→ 2CO — 不完全燃烧，产生有毒CO
9 4Al + 3O₂ —点燃→ 2Al₂O₃ 剧烈燃烧，发出耀眼白光 铝在常温下表面形成致密氧化膜
10 2Cu + O₂ —加热→ 2CuO 红色固体变为黑色 铜在空气中加热

二、氧气的制取 4个

#化学方程式说明备注
1 2H₂O₂ —MnO₂→ 2H₂O + O₂↑ 过氧化氢分解（不需加热） MnO₂作催化剂
2 2KMnO₄ —加热→ K₂MnO₄ + MnO₂ + O₂↑ 高锰酸钾加热分解 试管口略向下倾斜；导管口放棉花
3 2KClO₃ —MnO₂/加热→ 2KCl + 3O₂↑ 氯酸钾分解（MnO₂作催化剂） —
4 2HgO —加热→ 2Hg + O₂↑ 氧化汞分解（拉瓦锡实验） 历史上最早制取O₂的方法之一

三、二氧化碳相关反应 6个

#化学方程式说明备注
1 CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ 实验室制CO₂ 用稀盐酸，不用稀硫酸
2 CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O CO₂使澄清石灰水变浑浊 检验CO₂的特征反应
3 CO₂ + H₂O → H₂CO₃ CO₂溶于水生成碳酸 使石蕊溶液变红
4 H₂CO₃ —加热→ CO₂↑ + H₂O 碳酸不稳定，受热分解 石蕊由红变回紫
5 CaCO₃ —高温→ CaO + CO₂↑ 石灰石高温分解 工业制CO₂和生石灰
6 CO₂ + 2NaOH → Na₂CO₃ + H₂O CO₂与氢氧化钠反应 用于吸收CO₂（NaOH溶液）

四、水的相关反应 4个

#化学方程式说明备注
1 2H₂O —通电→ 2H₂↑ + O₂↑ 电解水：证明水由H、O元素组成 正氧负氢，氢二氧一
2 2H₂ + O₂ —点燃→ 2H₂O 氢气燃烧生成水 验证了水的组成
3 CaO + H₂O → Ca(OH)₂ 生石灰与水反应（放热） CaO可作干燥剂
4 SO₂ + H₂O → H₂SO₃ SO₂溶于水生成亚硫酸 酸雨的形成原因之一

五、金属与金属氧化物 9个

#化学方程式说明备注
1 Fe₂O₃ + 3CO —高温→ 2Fe + 3CO₂ 高炉炼铁原理 红色粉末→黑色；石灰水变浑浊
2 Fe + CuSO₄ → FeSO₄ + Cu 湿法炼铜（置换反应） 铁钉表面有红色物质析出；溶液由蓝色→浅绿色
3 Zn + H₂SO₄ → ZnSO₄ + H₂↑ 锌与稀硫酸反应 实验室制H₂
4 Zn + 2HCl → ZnCl₂ + H₂↑ 锌与稀盐酸反应 实验室制H₂
5 Fe + 2HCl → FeCl₂ + H₂↑ 铁与稀盐酸反应 溶液变为浅绿色（Fe²⁺）
6 2Al + 6HCl → 2AlCl₃ + 3H₂↑ 铝与稀盐酸反应 反应剧烈
7 Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag 铜与硝酸银溶液反应 铜表面析出银白色固体；溶液变蓝
8 Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O 铁锈与稀盐酸反应（除锈） 铁锈溶解，溶液变黄（Fe³⁺）
9 CuO + 2HCl → CuCl₂ + H₂O 氧化铜与稀盐酸反应 黑色固体溶解，溶液变蓝绿色

六、酸的性质（酸的通性） 6个

#化学方程式说明备注
1 Zn + 2HCl → ZnCl₂ + H₂↑ 酸 + 活泼金属 → 盐 + H₂ 金属活动性须排在H之前
2 Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O 酸 + 金属氧化物 → 盐 + 水 用于除铁锈
3 CuO + H₂SO₄ → CuSO₄ + H₂O 酸 + 金属氧化物 → 盐 + 水 黑色固体溶解，溶液变蓝
4 NaOH + HCl → NaCl + H₂O 酸 + 碱 → 盐 + 水（中和反应） 实质：H⁺ + OH⁻ → H₂O
5 Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ 酸 + 碳酸盐 → 盐 + 水 + CO₂ 用于检验碳酸盐
6 NaHCO₃ + HCl → NaCl + H₂O + CO₂↑ 酸 + 碳酸氢盐 → 盐 + 水 + CO₂ 小苏打治疗胃酸过多

七、碱的性质（碱的通性） 6个

#化学方程式说明备注
1 CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O 碱 + 非金属氧化物 → 盐 + 水 检验CO₂；石灰水变浑浊
2 CO₂ + 2NaOH → Na₂CO₃ + H₂O 碱 + 非金属氧化物 → 盐 + 水 NaOH溶液吸收CO₂
3 SO₂ + 2NaOH → Na₂SO₃ + H₂O 碱 + 非金属氧化物 → 盐 + 水 NaOH溶液吸收SO₂
4 CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄ 碱 + 盐 → 新碱 + 新盐 产生蓝色沉淀Cu(OH)₂
5 FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl 碱 + 盐 → 新碱 + 新盐 产生红褐色沉淀Fe(OH)₃
6 Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH 碱 + 盐 → 新碱 + 新盐 制取NaOH（纯碱制烧碱）

八、盐的反应（置换反应） 5个

#化学方程式说明备注
1 Fe + CuSO₄ → FeSO₄ + Cu 金属 + 盐 → 新金属 + 新盐 活动性强的金属置换活动性弱的
2 Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag 金属 + 盐 → 新金属 + 新盐 Cu 置换出 Ag
3 Zn + CuSO₄ → ZnSO₄ + Cu 金属 + 盐 → 新金属 + 新盐 Zn 置换出 Cu
4 2Al + 3CuSO₄ → Al₂(SO₄)₃ + 3Cu 金属 + 盐 → 新金属 + 新盐 Al 置换出 Cu
5 Cu + Hg(NO₃)₂ → Cu(NO₃)₂ + Hg 金属 + 盐 → 新金属 + 新盐 Cu 置换出 Hg

九、复分解反应 8个

🔍 复分解反应发生条件：生成物中有沉淀、气体或水（至少满足一个条件）。
#化学方程式说明备注
1 AgNO₃ + NaCl → AgCl↓ + NaNO₃ 生成白色沉淀AgCl 用于检验Cl⁻
2 BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl 生成白色沉淀BaSO₄ 用于检验SO₄²⁻（加稀HNO₃酸化）
3 Na₂CO₃ + CaCl₂ → CaCO₃↓ + 2NaCl 生成白色沉淀CaCO₃ 用于检验CO₃²⁻
4 CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ 生成气体和水 实验室制CO₂
5 Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ 生成气体和水 检验碳酸盐
6 NaOH + HCl → NaCl + H₂O 生成水（中和反应） 实质：H⁺ + OH⁻ → H₂O
7 Ca(OH)₂ + 2HCl → CaCl₂ + 2H₂O 生成水 中和酸性土壤
8 Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O 生成水 胃药中和胃酸

十、有机物反应 4个

#化学方程式说明备注
1 CH₄ + 2O₂ —点燃→ CO₂ + 2H₂O 甲烷燃烧（天然气主要成分） 产生蓝色火焰
2 C₂H₅OH + 3O₂ —点燃→ 2CO₂ + 3H₂O 乙醇（酒精）燃烧 可再生能源
3 6CO₂ + 6H₂O —光/叶绿素→ C₆H₁₂O₆ + 6O₂ 光合作用 将光能转化为化学能
4 C₆H₁₂O₆ + 6O₂ —酶→ 6CO₂ + 6H₂O 呼吸作用 释放能量供生命活动

十一、分解反应 5个

#化学方程式说明备注
1 2H₂O₂ —MnO₂→ 2H₂O + O₂↑ 过氧化氢分解 MnO₂催化
2 2KMnO₄ —加热→ K₂MnO₄ + MnO₂ + O₂↑ 高锰酸钾加热分解 —
3 CaCO₃ —高温→ CaO + CO₂↑ 石灰石高温分解 工业制生石灰
4 H₂CO₃ → CO₂↑ + H₂O 碳酸分解 碳酸不稳定
5 Cu₂(OH)₂CO₃ —加热→ 2CuO + H₂O + CO₂↑ 铜绿（碱式碳酸铜）分解 绿色→黑色

十二、其他重要反应 5个

#化学方程式说明备注
1 4P + 5O₂ —点燃→ 2P₂O₅ 红磷燃烧测定空气中O₂含量 消耗O₂，水进入约1/5体积
2 2Mg + CO₂ —点燃→ 2MgO + C 镁在CO₂中燃烧 CO₂不能扑灭镁火
3 SO₂ + H₂O → H₂SO₃ SO₂溶于水 酸雨形成
4 2H₂SO₃ + O₂ → 2H₂SO₄ 亚硫酸被氧化 酸雨酸性增强
5 C + CO₂ —高温→ 2CO 碳还原CO₂ 高炉中产生还原剂CO
📊 总统计：共收录 60+ 个化学反应方程式，覆盖12大类别。
🔥 反应条件汇总：点燃（燃烧反应）、加热（△）、高温、通电（电解）、催化剂（MnO₂等）、光照/叶绿素（光合作用）
📝 四大基本反应类型：化合反应（多变一）、分解反应（一变多）、置换反应（单+化→新单+新化）、复分解反应（化+化→新化+新化）
