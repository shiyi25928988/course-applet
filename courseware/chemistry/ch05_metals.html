5.1 金属的性质和利用

金属的物理性质

大多数金属具有以下共同物理性质：

  • 金属光泽——表面有光泽，但块状金属通常因表面氧化而失去光泽
  • 导电性——良好的导电体，银>铜>金>铝>铁（银最好，铜最常用）
  • 导热性——良好的导热体，常用于炊具和散热器
  • 延展性——可拉成细丝（延性）或锤成薄片（展性），金可打成0.001mm厚的金箔
  • 常温下除汞（Hg）为液态外，其余金属均为固态

注意：铜是紫红色的，金是金黄色的，其余金属大多为银白色。铁粉为黑色（光学原因），铁片为银白色。

合金——金属的"升级版"

合金：在金属中加热熔合某些金属或非金属，形成的具有金属特性的物质。合金属于混合物。

合金的优越性（与组分金属相比）：

  • 硬度更大——合金的硬度一般大于其组分金属（如纯铁较软，钢很硬）
  • 熔点更低——合金的熔点一般低于其组分金属（如焊锡熔点低于纯锡和纯铅）
  • 抗腐蚀性更强——如不锈钢比纯铁更耐腐蚀
  • 机械性能更好——强度、韧性等综合性能提高

常见合金及其应用

金属的化学性质

1. 金属与氧气反应

  • 大多数金属能与氧气反应生成金属氧化物：3Fe + 2O₂ →(点燃) Fe₃O₄
  • 铝在空气中形成致密的氧化铝（Al₂O₃）保护膜，阻止进一步氧化
  • 金（Au）即使在高温下也不与氧气反应——"真金不怕火炼"
  • 铜在潮湿空气中生成铜绿：2Cu + O₂ + H₂O + CO₂ → Cu₂(OH)₂CO₃

2. 金属与酸反应

  • 活泼金属（氢前）能与稀盐酸、稀硫酸反应，产生氢气
  • Zn + 2HCl → ZnCl₂ + H₂↑
  • Fe + H₂SO₄ → FeSO₄ + H₂↑（铁与稀硫酸反应生成硫酸亚铁，溶液浅绿色）
  • 铜不与稀盐酸、稀硫酸反应——铜在氢之后

3. 金属与盐溶液反应

  • 活泼金属能从较不活泼金属的盐溶液中置换出该金属
  • Fe + CuSO₄ → FeSO₄ + Cu（铁钉表面析出红色铜，溶液蓝色变浅绿色）
  • Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag（铜表面析出银白色银，溶液变蓝色）

金属活动性顺序深度解析

K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

→ 金属活动性由强到弱 →

三大应用规则：

规则一：排在氢前面的金属能置换出稀盐酸或稀硫酸中的氢，生成氢气。
例外：K、Ca、Na太活泼，与酸反应过于剧烈甚至爆炸，通常不在实验室中使用。

规则二：排在前面的金属能把排在后面的金属从其盐溶液中置换出来。
注意：K、Ca、Na太活泼，会先与水反应，不能从盐溶液中置换其他金属。

规则三：多种金属与同一种盐溶液反应时，活动性差距越大，反应越优先。
如：将Fe和Cu同时放入AgNO₃溶液中，Fe先与AgNO₃反应。

金属与酸反应的快慢比较

在金属活动性顺序中，位置越靠前，金属与酸反应越剧烈（产生气泡的速率越快）。

金属与稀盐酸反应现象反应速率化学方程式

Zn反应较快，产生气泡★★★☆☆Zn + 2HCl → ZnCl₂ + H₂↑

🔬 实验设计要点：比较不同金属与酸反应的快慢，需控制变量——酸的种类、浓度、温度和金属的表面积（大小、形状）必须相同，以保证实验的公平性。

置换反应

定义：一种单质与一种化合物反应，生成另一种单质和另一种化合物的反应。通式：A + BC → AC + B

经典实验：铁与硫酸铜溶液反应

  • 操作：将洁净的铁钉浸入硫酸铜溶液中
  • Fe + CuSO₄ → FeSO₄ + Cu
  • 现象：铁钉表面有红色物质（铜）析出，溶液由蓝色逐渐变为浅绿色
  • 应用：古代"曾青得铁则化为铜"——湿法炼铜的原理

5.2 金属矿物 铁的冶炼

常见的铁矿石

矿石名称主要成分化学式铁元素含量颜色
赤铁矿氧化铁Fe₂O₃约70%红棕色
磁铁矿四氧化三铁Fe₃O₄约72.4%黑色
菱铁矿碳酸亚铁FeCO₃约48.3%灰白色
黄铁矿二硫化亚铁FeS₂约46.7%金黄色

黄铁矿含硫量高，炼铁会产生大量SO₂污染环境，且铁含量低，一般不用于炼铁。

高炉炼铁——工业原理

主要原料：

  • 铁矿石——提供铁元素（赤铁矿Fe₂O₃或磁铁矿Fe₃O₄）
  • 焦炭——提供热量（燃烧放热）和还原剂（生成CO）
  • 石灰石——造渣，将矿石中的脉石（主要成分SiO₂）转化为炉渣CaSiO₃除去
  • 空气——提供氧气使焦炭燃烧

高炉内的主要化学反应：

① 焦炭燃烧提供热量：C + O₂ →(点燃) CO₂

② CO₂被焦炭还原生成CO：CO₂ + C →(高温) 2CO

③ 一氧化碳还原铁矿石（核心反应——炼铁原理）：Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂

④ 石灰石分解与造渣：CaCO₃ →(高温) CaO + CO₂↑；CaO + SiO₂ →(高温) CaSiO₃

高炉产物：

  • 上方出口：高炉煤气（含CO、CO₂、N₂等）——可回收利用作燃料
  • 中间出口：炉渣（CaSiO₃等）——可用于制造水泥
  • 底部出口：生铁（含碳量2%~4.3%，硬而脆）——需进一步炼钢

炼钢与炼铁的区别

比较项目炼铁炼钢
原料铁矿石、焦炭、石灰石生铁、废钢、氧气
原理还原——将Fe₂O₃还原为Fe氧化——将生铁中过量的C、S、P等氧化除去
核心反应Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂C + O₂ →(高温) CO₂
产品含碳量2%~4.3%（生铁）0.03%~2%（钢）
产品性质硬而脆，可铸不可锻硬而韧，有弹性，可铸可锻
设备高炉转炉、电炉
💡 一句话总结：炼铁是还原过程（铁的化合价降低），炼钢是氧化过程（降低碳含量）。生铁和钢的本质区别在于含碳量不同。

实验室：一氧化碳还原氧化铁

实验原理：Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂

实验装置要点：

  • 硬质玻璃管中放入红棕色的Fe₂O₃粉末
  • 通入CO气体，用酒精喷灯加热
  • 尾气用气球收集或点燃处理（CO有毒，不能直接排放！）

实验步骤（关键顺序）：

  1. 先通CO，排尽装置内的空气（防止CO与空气混合加热发生爆炸）
  2. 点燃酒精喷灯加热
  3. 观察现象：红棕色粉末逐渐变为黑色（铁粉），澄清石灰水变浑浊
  4. 停止加热后，继续通CO至冷却（防止高温下的铁粉被重新氧化）
  5. 熄灭酒精喷灯，冷却后停止通CO
⚠️ 实验安全注意事项：CO有毒，实验必须在通风橱中进行。尾气必须处理（点燃或用气球收集），严禁直接排放！实验前必须检验CO的纯度，防止爆炸。

5.3 金属防护和废金属回收

铁锈蚀条件的探究实验

铁生锈的条件：同时与水和氧气接触，两者缺一不可。

控制变量实验设计：

试管编号条件操作方法现象结论
①有H₂O + 有O₂铁钉+水（部分浸入，暴露空气）铁钉生锈（水与空气交界处最严重）铁生锈需水和氧气
②有H₂O + 无O₂铁钉+煮沸冷却的水+植物油封层铁钉不生锈仅有水无氧气不生锈
③无H₂O + 有O₂铁钉+干燥剂（CaCl₂或硅胶）密封铁钉不生锈仅有氧气无水不生锈

实验②中水需煮沸以排出溶解的氧气，植物油封层隔绝空气。实验③中干燥剂吸收空气中的水分。

🔬 实验设计的关键：控制变量法——每次只改变一个条件，其他条件保持不变。通过对比①②③三支试管，分别得出水、氧气各自的作用。

铁锈的成分与特点

铁锈的主要成分：Fe₂O₃·xH₂O（氧化铁的水合物）

铁锈的特点：疏松多孔，不能阻止内部的铁继续与氧气和水接触，因此铁制品一旦生锈，会越来越严重，直至全部锈蚀。

对比：铝表面形成的Al₂O₃薄膜致密，能阻止铝进一步氧化，起到保护作用。

防锈方法——六大措施

防锈原理：隔绝水或氧气（或两者都隔绝），破坏铁生锈的条件。

方法原理实例
① 保持干燥隔绝水刀具擦干存放、精密仪器放干燥剂
② 涂油隔绝水和氧气机械零件涂防锈油、枪械擦油
③ 刷漆隔绝水和氧气铁门刷漆、汽车喷漆、桥梁涂装
④ 电镀隔绝水和氧气镀锌（白铁皮/镀锌板）、镀铬（水龙头）、镀锡（马口铁罐头）
⑤ 制成合金改变内部结构不锈钢（Fe+Cr+Ni），抗腐蚀性极强
⑥ 牺牲阳极法利用原电池原理船体连接锌块（锌比铁活泼，先被腐蚀，保护铁）
💡 常见防锈方法对比：
  • 镀锌板（白铁皮）——锌比铁活泼，即使镀层破损，锌仍优先腐蚀保护铁
  • 镀锡板（马口铁）——锡不如铁活泼，一旦镀层破损，铁反而加速腐蚀
  • 不锈钢——含铬（Cr）≥12%，表面形成致密Cr₂O₃保护膜，抗腐蚀性极强

废金属回收的经济和环境效益

废金属回收是金属资源循环利用的重要途径，具有多方面的重大意义：

经济效益：

  • 回收1吨废钢铁可炼出0.85吨好钢，相当于节约1.5吨铁矿石和0.5吨焦炭
  • 回收铝的能耗仅为从矿石冶炼铝的5%——铝的回收利用具有极高的经济价值
  • 减少采矿、运输、冶炼等环节的成本

环境效益：

  • 减少矿山开采对植被和地表的破坏
  • 减少冶炼过程中SO₂、CO₂等有害气体和温室气体的排放
  • 减少固体废弃物（废金属）对环境的污染
  • 回收利用比从矿石冶炼可减少70%以上的能耗
📊 数据：全球每年回收的废钢铁超过6亿吨，占钢铁总产量的40%以上。回收利用废金属是节能减排和可持续发展的重要措施。

📝 章节练习题

1下列金属活动性最强的是____。
✅ C。金属活动性顺序：Zn > Fe > (H) > Cu > Ag。Zn最活泼。
2铁生锈的条件是____。
✅ C。铁生锈需要同时接触水和氧气，两者缺一不可。控制变量实验已证明此结论。
3炼铁高炉中，还原剂主要是____。
✅ B。炼铁的还原剂是CO（由焦炭与CO₂反应生成），核心反应：Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂。
4下列反应属于置换反应的是____。
✅ C。置换反应：单质+化合物→新单质+新化合物。A化合、B分解、D复分解（中和反应）。
5生铁和钢的本质区别在于____。
✅ C。生铁含碳量2%~4.3%，钢含碳量0.03%~2%。含碳量不同导致性质差异：生铁硬而脆，钢硬而韧。
6下列防锈方法中，利用"牺牲阳极法"原理的是____。
✅ C。牺牲阳极法：连接更活泼的金属（锌），锌比铁先腐蚀，保护铁。A是隔绝法，B是保持干燥，D是改变内部结构。
7实验室用CO还原Fe₂O₃，实验操作的先后顺序正确的是____。
✅ B。先通CO排尽空气（防爆炸），再加热；停止加热后继续通CO至冷却（防止高温铁粉被重新氧化）。
8下列关于合金的说法，正确的是____。
✅ C。合金是混合物（A错），熔点一般低于组分金属（B错），硬度一般大于组分金属（C对），合金中可含有非金属如碳（D错）。
