第7章 应用广泛的酸、碱、盐
🎯学习目标
  • 会用酸碱指示剂和pH试纸检验溶液的酸碱性
  • 掌握常见酸（盐酸、硫酸）和常见碱（氢氧化钠、氢氧化钙）的性质和用途
  • 理解中和反应的实质及其应用
  • 了解常见盐的性质、俗称和复分解反应的条件
  • 掌握离子共存问题的判断方法

7.1 溶液的酸碱性

一、酸碱指示剂

指示剂酸性溶液中性溶液碱性溶液
紫色石蕊溶液红色紫色蓝色
无色酚酞溶液无色无色红色

记忆口诀：石蕊——"酸红""碱蓝"；酚酞——"遇碱变红"（酸性和中性均为无色）。

二、pH —— 溶液酸碱度的表示

pH范围：0 ~ 14

pH < 7：酸性溶液，pH越小酸性越强

pH = 7：中性溶液

pH > 7：碱性溶液，pH越大碱性越强

注意：酸溶液一定呈酸性（pH < 7），但酸性溶液不一定是酸溶液（如NaHSO₄溶液呈酸性，但是盐溶液）。同理，碱溶液一定呈碱性，但碱性溶液不一定是碱溶液（如Na₂CO₃溶液呈碱性，但是盐溶液）。

三、pH试纸使用方法详解

正确操作步骤：

  1. 取一小片pH试纸放在洁净干燥的玻璃片或表面皿上
  2. 用洁净干燥的玻璃棒蘸取少量待测液
  3. 将蘸有待测液的玻璃棒点在pH试纸中部
  4. 待试纸变色后，在半分钟内与标准比色卡对比，读出pH（整数）
⚠️ pH试纸使用的关键注意事项（高频考点）：
  • 不能将pH试纸直接伸入待测液中——会污染试剂
  • 不能先用水润湿pH试纸——润湿后相当于稀释待测液：测酸性溶液时pH偏大（酸性减弱），测碱性溶液时pH偏小（碱性减弱），测中性溶液时pH不变
  • pH试纸测出的pH只能是整数（0~14），不能精确到小数。精确测定需用pH计（酸度计）

四、溶液酸碱性与生活

物质/环境pH范围酸碱性说明
胃液（胃酸）0.9 ~ 1.5强酸性主要成分HCl，帮助消化和杀菌
正常雨水≈ 5.6弱酸性因空气中CO₂溶于水形成H₂CO₃
酸雨< 5.6较强酸性SO₂和NO₂等污染物溶于雨水所致
纯水7中性—
血液7.35 ~ 7.45弱碱性维持在此范围内对生命至关重要
肥皂水≈ 10碱性含碱性物质，可去油污
土壤各地不同多数弱酸性酸性土壤可施加熟石灰改良

📖 化学史话：酸碱指示剂的发现

17世纪，英国化学家波义耳（Robert Boyle）在一次实验中，将盐酸溅到了紫罗兰花上。他发现紫罗兰的花瓣变成了红色。受此启发，波义耳采集各种植物花叶制成浸出液，用来检验溶液的酸碱性，从而发现了酸碱指示剂。他发现紫罗兰浸出液遇酸变红，遇碱变蓝（或绿）。此后，他又发现了石蕊指示剂——至今仍是实验室最常用的酸碱指示剂之一。波义耳的这一发现，为酸碱化学的发展奠定了基础。

7.2 常见的酸和碱

一、酸的分类

按是否含氧元素：

含氧酸：H₂SO₄（硫酸）、HNO₃（硝酸）、H₂CO₃（碳酸）、H₃PO₄（磷酸）

无氧酸：HCl（盐酸/氢氯酸）、H₂S（氢硫酸）、HF（氢氟酸）

按电离出H⁺的个数：

一元酸：HCl、HNO₃（每个分子电离出1个H⁺）

二元酸：H₂SO₄、H₂CO₃（每个分子电离出2个H⁺）

三元酸：H₃PO₄（每个分子电离出3个H⁺）

二、常见的酸

盐酸（HCl的水溶液）：

  • 物理性质：无色有刺激性气味的液体，有挥发性（打开瓶盖有白雾——HCl气体与空气中的水蒸气结合形成盐酸小液滴）。
  • 化学性质：使石蕊变红；与活泼金属反应产生H₂；与金属氧化物反应；与碱反应；与AgNO₃反应产生白色沉淀（AgCl）。

Zn + 2HCl → ZnCl₂ + H₂↑

HCl + AgNO₃ → AgCl↓ + HNO₃（检验Cl⁻的方法）

硫酸（H₂SO₄）：

  • 物理性质：无色粘稠油状液体，不易挥发，密度大于水。
  • 吸水性：浓硫酸可吸收空气中的水分，可作干燥剂（但不能干燥NH₃——会反应生成(NH₄)₂SO₄）。

三、浓硫酸的特性 —— 脱水性与"黑面包"实验

脱水性：浓硫酸能将有机物（如蔗糖、纸张、木材、棉布等）中的氢、氧元素按2:1比例脱去（相当于脱去H₂O），使有机物碳化变黑。这是化学变化。

🔬 "黑面包"实验：在蔗糖（C₁₂H₂₂O₁₁）中加入浓硫酸，蔗糖迅速变黑并膨胀成"黑面包"状。
C₁₂H₂₂O₁₁ →[浓H₂SO₄] 12C + 11H₂O
反应放热，生成的水蒸发，碳被水蒸气推出形成疏松多孔结构。同时浓硫酸还可将部分碳氧化：
C + 2H₂SO₄(浓) →[加热] CO₂↑ + 2SO₂↑ + 2H₂O
⚠️ 浓硫酸的安全使用：
稀释浓硫酸时，必须将浓硫酸沿器壁缓慢注入水中，并用玻璃棒不断搅拌——"酸入水，沿器壁，不断搅拌"。绝不能将水倒入浓硫酸中！浓硫酸密度大于水，水倒入酸中会浮在表面，局部温度剧升导致水沸腾，使酸液飞溅，造成严重灼伤。
若浓硫酸不慎溅到皮肤上：先用干布擦拭，再用大量水冲洗，最后涂上3%~5%的NaHCO₃溶液。

四、常见的碱 —— 分类与特性

碱的分类：

按溶解性分：可溶性碱（NaOH、KOH、Ba(OH)₂、Ca(OH)₂微溶）和不溶性碱（Cu(OH)₂蓝色沉淀、Fe(OH)₃红褐色沉淀、Mg(OH)₂白色沉淀等）。

口诀：钾钠钡铵皆可溶（KOH、NaOH、Ba(OH)₂、NH₃·H₂O），钙是微溶（Ca(OH)₂），其余一般不溶。

氢氧化钠（NaOH，俗称烧碱、火碱、苛性钠）：

  • 白色固体，极易溶于水（溶于水放出大量热），溶液有滑腻感。
  • 有强腐蚀性（能腐蚀皮肤和织物）。
  • 易潮解（吸收空气中的水分而表面变潮湿），可作干燥剂（但不能干燥CO₂、SO₂等酸性气体——会反应）。
⚠️ NaOH的保存方法：
①必须密封保存——防止潮解（吸收水分）和与空气中的CO₂反应变质：2NaOH + CO₂ → Na₂CO₃ + H₂O
②NaOH溶液不能用玻璃塞——NaOH会与玻璃中的SiO₂反应生成Na₂SiO₃（具有粘性），将瓶塞与瓶口粘住。应使用橡胶塞。
③盛放NaOH溶液的试剂瓶若瓶口出现白色固体，说明NaOH已与CO₂反应生成了Na₂CO₃。

氢氧化钙（Ca(OH)₂，俗称熟石灰、消石灰）：

  • 白色粉末，微溶于水（溶解度随温度升高而减小）。
  • 其水溶液为石灰水（Ca(OH)₂的饱和溶液，常用于检验CO₂）。
  • CaO（生石灰）+ H₂O → Ca(OH)₂（大量放热），此反应可用于制取Ca(OH)₂。

Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O（石灰水变浑浊——检验CO₂的经典方法）

五、酸与碱的通性总结

类别酸（H⁺的性质）碱（OH⁻的性质）
与指示剂使石蕊变红，酚酞不变色使石蕊变蓝，使酚酞变红
与活泼金属反应生成盐和H₂一般不反应（除Al、Zn等两性金属）
与金属氧化物反应生成盐和水一般不反应
与非金属氧化物一般不反应（除特殊）反应生成盐和水
与酸/碱与碱发生中和反应与酸发生中和反应
与某些盐反应生成新酸和新盐反应生成新碱和新盐

六、中和反应

定义：酸与碱作用生成盐和水的反应。

实质：H⁺ + OH⁻ → H₂O

NaOH + HCl → NaCl + H₂O

Ca(OH)₂ + 2HCl → CaCl₂ + 2H₂O

2NaOH + H₂SO₄ → Na₂SO₄ + 2H₂O

💡 中和反应的应用（更多实际例子）：
应用场景原理具体物质
改良酸性土壤中和土壤中的酸用熟石灰Ca(OH)₂（比NaOH便宜且腐蚀性较小）
处理工业酸性废水中和废酸加入熟石灰或石灰石
治疗胃酸过多中和胃酸（HCl）Al(OH)₃（氢氧化铝）、NaHCO₃（小苏打）、Mg(OH)₂
被蚊虫叮咬止痒中和蚁酸涂抹肥皂水或氨水（弱碱性）
被碱液灼伤中和残留碱用硼酸溶液冲洗
洗发护发调节pH洗发水弱碱性去油、护发素弱酸性（中和残留碱）

7.3 几种重要的盐

一、盐的分类

按组成不同：

正盐：酸和碱完全中和的产物，只含金属离子（或NH₄⁺）和酸根离子。如：NaCl、Na₂CO₃、CuSO₄。

酸式盐：电离出的阳离子除金属离子（或NH₄⁺）外还有H⁺。如：NaHCO₃（碳酸氢钠）、NaHSO₄（硫酸氢钠）。

碱式盐：电离出的阴离子除酸根离子外还有OH⁻。如：Cu₂(OH)₂CO₃（碱式碳酸铜/铜绿）。

二、常见盐的俗称和用途

化学式俗称颜色/状态主要用途备注
NaCl食盐白色晶体调味品、防腐剂、生理盐水（0.9%）、化工原料（制Na₂CO₃、NaOH、Cl₂等）溶解度随温度变化不大
Na₂CO₃纯碱、苏打白色粉末玻璃、造纸、纺织、洗涤剂 纯碱不是碱，是盐！水溶液呈碱性
NaHCO₃小苏打白色细小晶体发酵粉（焙制糕点）、治疗胃酸过多、干粉灭火器 加热分解：2NaHCO₃ →[加热] Na₂CO₃ + H₂O + CO₂↑
CaCO₃石灰石、大理石白色固体建筑材料、补钙剂、制生石灰和CO₂ CaCO₃ →[高温] CaO + CO₂↑
CuSO₄蓝矾、胆矾（CuSO₄·5H₂O）蓝色晶体配制波尔多液（农药）、电镀 CuSO₄·5H₂O加热失水变白色（无水CuSO₄），可用于检验水
K₂SO₄—白色晶体钾肥—

三、复分解反应 —— 条件详解

定义：两种化合物相互交换成分，生成另外两种化合物的反应。

通式：AB + CD → AD + CB

发生条件（满足其中至少一个）：

  1. 生成物中有沉淀（↓）
  2. 生成物中有气体（↑）
  3. 生成物中有水（H₂O）

注意：反应物中的碱和盐必须可溶（溶于水），酸则不一定。若反应物中碱或盐不可溶，反应不能发生。

📋 盐的溶解性口诀（完整版）：
钾钠铵硝皆可溶（K⁺、Na⁺、NH₄⁺、NO₃⁻的盐全溶）
盐酸不溶银亚汞（AgCl↓、Hg₂Cl₂↓）
硫酸不溶钡和铅（BaSO₄↓、PbSO₄↓；CaSO₄、Ag₂SO₄微溶）
碳酸只溶钾钠铵（其他碳酸盐均不溶）
碱溶钾钠钡和铵（KOH、NaOH、Ba(OH)₂、NH₃·H₂O可溶；Ca(OH)₂微溶）

四、离子共存问题

在同一溶液中，离子间若能发生反应（生成沉淀、气体或水），则不能大量共存。

不能共存的离子对原因产物
H⁺ 与 OH⁻中和反应H₂O
H⁺ 与 CO₃²⁻生成气体CO₂↑ + H₂O
H⁺ 与 HCO₃⁻生成气体CO₂↑ + H₂O
OH⁻ 与 NH₄⁺生成气体NH₃↑ + H₂O
OH⁻ 与 Cu²⁺生成沉淀Cu(OH)₂↓（蓝色）
OH⁻ 与 Fe³⁺生成沉淀Fe(OH)₃↓（红褐色）
OH⁻ 与 Mg²⁺生成沉淀Mg(OH)₂↓（白色）
Ag⁺ 与 Cl⁻生成沉淀AgCl↓（白色，不溶于酸）
Ba²⁺ 与 SO₄²⁻生成沉淀BaSO₄↓（白色，不溶于酸）
Ba²⁺ 与 CO₃²⁻生成沉淀BaCO₃↓（白色，溶于酸）
💡 离子共存判断题的解题思路：
① 先看题目是否有隐含条件（如"无色溶液"——排除Cu²⁺蓝色、Fe³⁺黄色、Fe²⁺浅绿色、MnO₄⁻紫红色；"pH=1"——含大量H⁺；"pH=13"——含大量OH⁻）
② 再看离子间是否会发生反应（生成沉淀/气体/水）
③ 注意："大量共存"和"能共存"含义不同——"大量共存"要求更严格

五、粗盐提纯详细步骤

粗盐中含有不溶性杂质（泥沙）和可溶性杂质（CaCl₂、MgCl₂、Na₂SO₄等）。

步骤操作目的注意事项
① 溶解 加水，玻璃棒搅拌 使NaCl溶解 水量不宜过多（否则蒸发耗时长）
② 过滤 滤纸过滤 除去不溶性杂质（泥沙） "一贴二低三靠"
③ 除杂 依次加入过量BaCl₂（除SO₄²⁻）→过量NaOH（除Mg²⁺）→过量Na₂CO₃（除Ca²⁺和Ba²⁺）→过滤→加HCl调至中性 除去可溶性杂质 Na₂CO₃必须在BaCl₂之后加入（才能除去过量的Ba²⁺）；最后加HCl除去过量的OH⁻和CO₃²⁻
④ 蒸发 加热蒸发，玻璃棒不断搅拌 获得NaCl晶体 出现较多固体时停止加热，用余热蒸干
⑤ 计算产率 称量纯净NaCl质量 产率 = 纯净NaCl/粗盐质量 × 100% 产率偏低原因：溶解不充分、过滤/蒸发时有损失

六、化学肥料

氮肥：促进茎叶生长。代表物：尿素CO(NH₂)₂（含氮量最高）、NH₄NO₃、NH₄HCO₃。

⚠️ 铵态氮肥不能与碱性物质混用！铵盐与碱反应会放出NH₃，使肥效降低：
NH₄Cl + NaOH → NaCl + NH₃↑ + H₂O

磷肥：促进根系发达、籽粒饱满。代表物：Ca(H₂PO₄)₂（过磷酸钙）。

钾肥：增强抗倒伏、抗病虫害能力。代表物：KCl、K₂SO₄。

复合肥：含两种或以上营养元素。代表物：KNO₃（含K、N）、NH₄H₂PO₄（含N、P）。

🔬 基础实验

基础实验7：溶液酸碱性的检验

用紫色石蕊溶液和无色酚酞溶液分别检验稀盐酸、NaOH溶液、NaCl溶液的酸碱性。再用pH试纸分别测定三种溶液的pH值，并与标准比色卡对比读数。

实验结论：稀盐酸（酸性）使石蕊变红，酚酞不变色，pH < 7；NaOH溶液（碱性）使石蕊变蓝，酚酞变红，pH > 7；NaCl溶液（中性）石蕊仍为紫色，酚酞不变色，pH = 7。

基础实验8：酸与碱的化学性质

酸的性质实验：

  • 与石蕊反应：稀盐酸使紫色石蕊变红
  • 与活泼金属(Zn)反应：Zn + 2HCl → ZnCl₂ + H₂↑（产生气泡，Zn逐渐溶解）
  • 与铁锈(Fe₂O₃)反应：Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O（铁锈消失，溶液变黄色）

碱的性质实验：

  • 与石蕊和酚酞反应：NaOH溶液使石蕊变蓝，使酚酞变红
  • 与CO₂反应：Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O（石灰水变浑浊）
  • 与CuSO₄反应：2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄（产生蓝色沉淀）

📝 章节练习题

1能使紫色石蕊溶液变红的是____。
✅ B。酸使石蕊变红，碱使石蕊变蓝。稀盐酸是酸，石灰水（Ca(OH)₂）和纯碱溶液（Na₂CO₃）都是碱性的，食盐溶液是中性的。
2中和反应的实质是____。
✅ B。中和反应的实质是H⁺与OH⁻结合生成H₂O分子。A是中和反应的定义（现象层面），不是实质。C过于笼统，D不一定成立（中和后溶液pH不一定恰好为7）。
3下列物质属于盐的是____。
✅ C。Na₂CO₃由金属离子（Na⁺）和酸根离子（CO₃²⁻）构成，属于盐。NaOH是碱，H₂SO₄是酸，CaO是氧化物。注意：纯碱（Na₂CO₃）不是碱，是盐！
4稀释浓硫酸的正确操作是____。
✅ B。浓硫酸稀释口诀："酸入水，沿器壁，不断搅拌"。水入酸会引起液滴飞溅，造成严重灼伤！因为浓硫酸密度大于水且稀释时放出大量热。
5下列物质中，不能与稀盐酸发生复分解反应的是____。
✅ C。复分解反应需生成沉淀/气体/水。A（NaOH + HCl → NaCl + H₂O，生成水✓）；B（Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑，生成气体✓）；D（AgNO₃ + HCl → AgCl↓ + HNO₃，生成沉淀✓）；C交换后仍为NaCl和HCl，不满足条件。
6用pH试纸测某溶液pH时，若先将试纸用水润湿，再测酸性溶液，则测得pH值____。
✅ A。试纸润湿相当于稀释待测液。测酸性溶液时，酸性减弱，pH值偏大（向7靠近）。测碱性溶液则pH偏小。测中性溶液pH不变。
7下列各组离子中，能在水溶液中大量共存的是____。
✅ C。A中H⁺与OH⁻结合生成H₂O，不能共存；B中Ba²⁺与SO₄²⁻生成BaSO₄↓，不能共存；D中Ag⁺与Cl⁻生成AgCl↓，不能共存。C中所有离子互不反应，可以大量共存。
8关于NaOH的保存，下列说法正确的是____。
✅ C。NaOH必须密封保存（防止潮解和与CO₂反应），且使用橡胶塞（不能用玻璃塞——NaOH会与玻璃中的SiO₂反应生成Na₂SiO₃，将瓶塞粘住）。敞口放置会与CO₂反应变质：2NaOH + CO₂ → Na₂CO₃ + H₂O。
