2024年上海中考化学（真题整理）

理化综合135分 · 化学部分约55分

⚠️ 基于2024真题还原，标注章节考点

五、选择题（共20分，17题）

✅ 2024真题 第21题
物质分类第1章
1下列物质中，属于纯净物的是____。
✅ B。NaCl是纯净物。空气（混合物）、溶液都是混合物。（第1章·物质分类）
✅ 2024真题 第22题
化学变化第1章
2下列属于化学变化的是____。
✅ C。燃烧→有新物质生成→化学变化。ABD均为物理变化。（第1章·物理/化学变化）
✅ 2024真题 第23题
空气氧气第2章
3空气中含量最多的气体是____。
✅ B。空气组成：N₂ 78%、O₂ 21%。（第2章·空气组成）
✅ 2024真题 第24题
元素第3章
4地壳中含量最多的金属元素是____。
✅ C。地壳含量：O>Si>Al>Fe>Ca。Al是含量最多的金属。（第3章·元素）
✅ 2024真题 第25题
质量守恒第4章
5化学反应前后一定不变的是____。
✅ B。质量守恒：总质量不变。原子种类/数目/质量不变。分子种类一定变。（第4章）
✅ 2024真题 第26题
金属第5章
6下列金属中，不能与稀盐酸发生置换反应的是____。
✅ D。Cu排在H后（金属活动性顺序），不能置换酸中的氢。（第5章·金属活动性）
✅ 2024真题 第27题
溶液第6章
7下列物质溶于水，溶液温度降低的是____。
✅ D。NH₄NO₃溶解吸热。NaOH和浓H₂SO₄溶解放热。（第6章·溶解热效应）
✅ 2024真题 第28题
酸碱盐第7章
8能使无色酚酞溶液变红的是____。
✅ B。酚酞遇碱变红。石灰水Ca(OH)₂是碱。盐酸是酸，食盐水中性。（第7章·指示剂）
✅ 2024真题 第29题
复分解反应第7章
9下列各组物质能发生复分解反应的是____。
✅ B。BaCl₂+Na₂SO₄→BaSO₄↓+2NaCl（生成沉淀）。其余无沉淀/气体/水。（第7章·复分解条件）
✅ 2024真题 第30题
有机物第8章
10下列物质中属于有机高分子化合物的是____。
✅ C。淀粉(C₆H₁₀O₅)n是天然有机高分子。CH₄、C₂H₅OH、C₆H₁₂O₆是小分子。（第8章）

六、填空与简答题（约20分）

✅ 2024真题 填空
化学用语化合价
11（1）2个氮原子____；（2）3个水分子____；（3）Fe₂O₃中铁元素化合价____；（4）Na⁺中数字"1"的含义____。
✅ (1)2N；(2)3H₂O；(3)+3；(4)每个钠离子带1个单位正电荷。考点：化学符号的意义（第3章）
✅ 2024真题 方程式
化学方程式第4章
12写出下列反应的化学方程式并注明反应类型：①实验室制取CO₂；②铁与硫酸铜溶液反应。
✅ ①CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑（复分解反应）；②Fe+CuSO₄→FeSO₄+Cu（置换反应）。（第2/4/5章）
✅ 2024真题 推断
推断题第7章
13【推断】白色固体A，加水全部溶解→无色溶液。加入稀盐酸→产生使石灰水变浑浊的气体B。加入AgNO₃溶液→白色沉淀C。A是____，B是____，C是____。
✅ Na₂CO₃、CO₂、AgCl。Na₂CO₃+2HCl→2NaCl+H₂O+CO₂↑，NaCl+AgNO₃→AgCl↓+NaNO₃。（第7章·物质推断）
✅ 2024真题 计算
计算题第4章
14将6.5g锌粒放入足量稀盐酸中。求：（1）产生H₂的质量；（2）需要消耗HCl的质量。
✅ Zn+2HCl→ZnCl₂+H₂↑。n(Zn)=6.5/65=0.1mol。(1)m(H₂)=0.1×2=0.2g；(2)m(HCl)=0.2×36.5=7.3g。考点：根据化学方程式计算（第4章）
✅ 2024真题 实验
实验题第2章
15【实验】用大理石和稀盐酸制取CO₂。（1）发生装置应选____（填"固固加热型"或"固液不加热型"）；（2）收集方法为____，验满方法为____；（3）不能用稀硫酸代替稀盐酸的原因____。
✅ (1)固液不加热型；(2)向上排空气法，用燃着木条放瓶口，木条熄灭则满；(3)生成微溶CaSO₄覆盖在大理石表面，阻止反应。（第2章·实验）

2023年上海中考化学精选

📋 2023真题
燃烧第4章
1纸火锅在使用过程中纸不会燃烧，原因是____。
✅ C。水蒸发吸热→温度<着火点→纸不燃烧。着火点是物质固有属性不可改变。（第4章·燃烧条件）
📋 2023真题
溶解度第6章
2如图是KNO₃和NaCl的溶解度曲线。将60℃时二者饱和溶液降温至20℃，下列说法正确的是____。
✅ B。KNO₃溶解度随温度变化大→降温明显析出。NaCl溶解度随温度变化小→可能不析出。（第6章·溶解度曲线）
📋 2023真题
中和反应第7章
3向一定量稀盐酸中逐滴加入NaOH溶液，反应过程中pH的变化情况是____。
✅ C。初始pH<7(酸性)，随NaOH加入pH增大，恰好中和pH=7，NaOH过量pH>7。（第7章·中和反应pH变化）
📋 2023真题
金属第5章
4要比较Fe、Cu、Ag三种金属的活动性，下列方案可行的是____。
✅ C。Cu放入FeSO₄→不反应（Fe>Cu）。Cu放入AgNO₃→置换出Ag（Cu>Ag）。得Fe>Cu>Ag。（第5章·金属活动性比较）
📋 2023真题
方程式计算
5用含CaCO₃ 90%的石灰石10g与足量稀盐酸反应，求生成CO₂的质量（标准状况下体积）。
✅ m(CaCO₃)=10×90%=9g。CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑。Mr(CaCO₃)=100,Mr(CO₂)=44。m(CO₂)=9×44/100=3.96g。V(CO₂)=3.96/1.977≈2.0L。考点：不纯物质计算（第4章）

2022年上海中考化学精选

📋 2022真题
氧气的制取第2章
1实验室用KMnO₄制取O₂的发生装置需用到的仪器是____。
✅ A。KMnO₄→固固加热型→试管+酒精灯+单孔塞+导管+水槽+集气瓶。（第2章·实验装置选择）
📋 2022真题
化肥第7章
2下列化肥中属于复合肥的是____。
✅ D。KNO₃含K和N两种营养元素→复合肥。A氮肥，B钾肥，C磷肥。（第7章·化肥分类）
📋 2022真题
除杂综合
3除去CO₂中混有的少量CO，可将混合气体通过____。
✅ C。CuO+CO→(△) Cu+CO₂，CO被氧化为CO₂。石灰水和NaOH会吸收CO₂。（第2/4章·除杂）
📋 2022真题
能源第9章
4为减少大气污染，下列措施不合理的是____。
✅ C。大量使用含硫煤→SO₂排放增加→酸雨。不合理。（第9章·环境保护）

2021年上海中考化学精选

📋 2021真题
质量守恒第4章
1在反应A+3B→2C+3D中，已知2.3gA与4.8gB恰好完全反应生成4.4gC，则生成D的质量为____g。
✅ A。m(D)=m(A)+m(B)-m(C)=2.3+4.8-4.4=2.7g。质量守恒定律的应用（第4章）
📋 2021真题
水的净化第2章
2下列水的净化操作中，净化程度最高的是____。
✅ D。蒸馏→得到纯水，净化程度最高。沉淀→过滤→吸附→蒸馏（净化程度递增）。（第2章·水的净化）
📋 2021真题
金属防护第5章
3下列措施不能防止铁制品生锈的是____。
✅ C。潮湿环境提供水和氧气→加速生锈。防锈需隔绝O₂和H₂O。（第5章·金属防锈）
📋 2021真题
溶质质量分数
4将20g NaCl完全溶于80g水中，所得溶液的溶质质量分数为____。
✅ A。ω=20/(20+80)×100%=20%。（第6章·溶质质量分数）

2020年上海中考化学精选

📋 2020真题
物理变化第1章
1下列变化中，属于物理变化的是____。
✅ C。矿石粉碎→形状改变，无新物质→物理变化。（第1章）
📋 2020真题
金属活动性第5章
2将铁钉放入硫酸铜溶液中，观察到的现象是____。
✅ B。Fe+CuSO₄→FeSO₄+Cu。红色Cu，浅绿色FeSO₄。（第5章·金属置换反应）
📋 2020真题
计算第4章
3加热分解15.8g KMnO₄，可以制得O₂多少克？（K:39, Mn:55, O:16）
✅ 2KMnO₄→(△) K₂MnO₄+MnO₂+O₂↑。Mr(KMnO₄)=158，n=15.8/158=0.1mol。n(O₂)=0.05mol，m(O₂)=1.6g。（第4章·方程式计算）

2019年上海中考化学精选

📋 2019真题
物质性质第1章
1下列用途中利用化学性质的是____。
✅ C。可燃性→化学性质。ABD均为物理性质。（第1章）
📋 2019真题
合金第5章
2关于合金的说法正确的是____。
✅ B。合金是混合物，可含非金属，硬度更大，熔点更低。（第5章·合金性质）
📋 2019真题
计算第7章
3用100g 7.3%稀盐酸与足量NaOH反应，求生成NaCl的质量。
✅ m(HCl)=7.3g。HCl+NaOH→NaCl+H₂O。m(NaCl)=7.3×58.5/36.5=11.7g。（第7章·中和反应计算）

2018年上海中考化学精选

📋 2018真题
电解水第2章
1电解水实验中，正极产生____，负极产生____，V(O₂):V(H₂)=____。
✅ B。正氧负氢，氢二氧一。（第2章·电解水）
📋 2018真题
粗盐提纯第7章
2粗盐提纯步骤：①____→过滤；②____。除去CaCl₂可加入____溶液。
✅ ①溶解→过滤；②蒸发结晶。加入Na₂CO₃溶液：CaCl₂+Na₂CO₃→CaCO₃↓+2NaCl。（第7章·粗盐提纯）
📋 2018真题
计算第4章
3用含CaCO₃ 85%的大理石25g与足量稀盐酸反应，求生成CO₂的质量。
✅ m(CaCO₃)=25×85%=21.25g。CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑。m(CO₂)=21.25×44/100=9.35g。（第4章·综合计算）
