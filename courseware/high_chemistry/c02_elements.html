第二章 常见元素及其化合物

金属元素

钠及其化合物：
Na：与水剧烈反应，2Na + 2H₂O = 2NaOH + H₂↑
Na₂O₂：淡黄色固体，供氧剂，2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂
Na₂CO₃：苏打、纯碱，稳定
NaHCO₃：小苏打，受热易分解，与酸反应更快
铝及其化合物：
Al：两性，与酸、强碱都能反应
Al₂O₃：两性氧化物
Al(OH)₃：两性氢氧化物，白色胶状沉淀，净水剂
铁及其化合物：
Fe²⁺：浅绿色，还原性，遇KSCN不变色，加氯水变血红色
Fe³⁺：黄色，氧化性，遇KSCN变血红色
相互转化：2Fe³⁺ + Fe = 3Fe²⁺；2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻

非金属元素

氯及其化合物：
Cl₂：黄绿色气体，强氧化性，有毒
漂白粉：Ca(ClO)₂ + CaCl₂，有效成分Ca(ClO)₂
HClO：弱酸，强氧化性，漂白性，不稳定
硫及其化合物：
SO₂：无色有刺激性气味，漂白品红（可逆），酸性氧化物，还原性
浓H₂SO₄：吸水性、脱水性、强氧化性
氮及其化合物：
NO：无色，与O₂反应生成NO₂
NO₂：红棕色，与水反应：3NO₂ + H₂O = 2HNO₃ + NO
NH₃：碱性气体，使湿润红色石蕊试纸变蓝，喷泉实验
HNO₃：强酸，浓HNO₃有强氧化性，见光分解
