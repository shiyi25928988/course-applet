第七章 水溶液中的离子平衡

7.1 弱电解质的电离

对于 CH₃COOH ⇌ CH₃COO⁻ + H⁺
电离常数 Kₐ = [CH₃COO⁻][H⁺]/[CH₃COOH]
电离平衡特点：弱电解质、可逆、部分电离
温度升高，电离程度增大（电离吸热）
浓度越稀，电离程度越大（越稀越电离）
水的离子积 Kw = [H⁺][OH⁻] = 1×10⁻¹⁴（25℃）
Kw只与温度有关，温度升高，Kw增大。适用于任何水溶液。

7.2 溶液的酸碱性和pH

pH = -lg[H⁺]
25℃时：
pH < 7，酸性，[H⁺] > [OH⁻]
pH = 7，中性，[H⁺] = [OH⁻]
pH > 7，碱性，[H⁺] < [OH⁻]

7.3 盐类的水解

水解规律：有弱才水解，无弱不水解，越弱越水解，谁强显谁性。
影响因素：
温度：升高温度，水解程度增大
浓度：越稀越水解
外加酸碱：促进或抑制

7.4 沉淀溶解平衡

对于 AgCl(s) ⇌ Ag⁺(aq) + Cl⁻(aq)
Ksp = [Ag⁺][Cl⁻]
溶度积Ksp：反映难溶电解质的溶解能力。
Q > Ksp，溶液过饱和，有沉淀析出
Q = Ksp，溶液饱和
Q < Ksp，溶液未饱和
