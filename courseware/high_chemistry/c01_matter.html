第一章 物质及其变化

1.1 物质的分类

树状分类法：
物质 → 纯净物 → 单质（金属、非金属）
物质 → 纯净物 → 化合物 → 氧化物、酸、碱、盐
物质 → 混合物
胶体：分散质粒子直径1~100nm，有丁达尔效应，能透过滤纸不能透过半透膜。
常见胶体：Fe(OH)₃胶体、淀粉溶液、蛋白质溶液、豆浆、雾

1.2 离子反应

电解质：在水溶液里或熔融状态下能够导电的化合物。
强电解质：强酸、强碱、大部分盐
弱电解质：弱酸、弱碱、水
离子方程式书写：写 → 拆 → 删 → 查
不可拆：沉淀、气体、弱电解质、氧化物、单质
离子共存判断：
生成沉淀（如Ba²⁺与SO₄²⁻）、气体（如H⁺与CO₃²⁻）、弱电解质（如H⁺与OH⁻）则不能共存

1.3 氧化还原反应

特征：反应前后有元素化合价变化
本质：电子转移（得失或偏移）
概念关系：
还原剂 → 失电子 → 化合价升高 → 被氧化 → 发生氧化反应 → 生成氧化产物
氧化剂 → 得电子 → 化合价降低 → 被还原 → 发生还原反应 → 生成还原产物
