7.1 电流 电压

电流的形成

电流是电荷的定向移动形成的。金属导体中，电流是自由电子定向移动形成的；在电解液中，电流由正、负离子的定向移动共同形成。

形成持续电流的条件：

（1）电路中有电源（提供电压）；

（2）电路是闭合的（通路）。

电流方向规定为正电荷定向移动的方向。在金属导体中，电流方向与自由电子实际移动方向相反。

单位：安培（A），常用毫安（mA）、微安（μA）。

1A = 1000mA，1mA = 1000μA。

电流表的使用

电流表用于测量电路中的电流，使用规则：

① 串联：电流表必须串联在被测电路中；

② 正进负出：电流从"+"接线柱流入，从"-"接线柱流出；

③ 不超量程：被测电流不能超过电流表的量程，可用试触法选择合适量程；

④ 禁止直接接电源：绝对不允许把电流表直接接到电源两极（会烧坏）。

读数：先确认量程和分度值，再读数。

电压的概念

电压是使电荷定向移动形成电流的原因。电压是电位差（电势差），电源是提供电压的装置。

单位：伏特（V），常用千伏（kV）、毫伏（mV）。

1kV = 1000V，1V = 1000mV。

电压是形成电流的原因，有电压不一定有电流（电路需闭合），有电流则一定有电压。

电压表的使用

电压表用于测量电路两端的电压，使用规则：

① 并联：电压表必须并联在被测电路两端；

② 正进负出：电流从"+"接线柱流入，从"-"接线柱流出；

③ 不超量程：被测电压不能超过量程；

④ 可直接接电源：电压表可以直接接在电源两极测电源电压。

⚡ 常见电压值速记：
• 一节干电池：1.5V  • 一节铅蓄电池：2V  • 手机锂电池：3.7V
• 人体安全电压：不高于36V  • 我国家庭电路：220V  • 工业动力用电：380V
• 发生闪电时云层间电压可达几万伏至几十万伏

7.2 欧姆定律 电阻

电阻的概念

电阻（R）表示导体对电流的阻碍作用的大小。导体的电阻越大，表示导体对电流的阻碍作用越大。

单位：欧姆（Ω），常用千欧（kΩ）、兆欧（MΩ）。

1kΩ = 1000Ω，1MΩ = 10⁶Ω。

电阻是导体本身的一种性质，它的大小取决于导体的材料、长度、横截面积、温度，与导体两端的电压和通过导体的电流无关。

影响电阻的因素

导体的电阻与以下因素有关（控制变量法）：

① 材料：不同材料的电阻率不同（银<铜<铝<铁）；

② 长度：同种材料、横截面积相同时，长度越长，电阻越大；

③ 横截面积：同种材料、长度相同时，横截面积越大，电阻越小；

④ 温度：大多数金属导体的电阻随温度升高而增大（少数半导体相反）。

超导现象：某些材料在温度降到足够低时，电阻突然变为零。

滑动变阻器

滑动变阻器原理：通过改变接入电路中电阻线的长度来改变电阻，从而改变电路中的电流。

接法：采用"一上一下"接线法（一个上接线柱 + 一个下接线柱）。

使用注意事项：

① 闭合开关前，滑片应置于阻值最大处（保护电路）；

② 通过变阻器的电流不能超过其额定电流；

③ 铭牌标注如"50Ω 1.5A"表示最大阻值50Ω，允许通过的最大电流1.5A。

欧姆定律

欧姆定律：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。这是电学最基本的定律。

I = U / R

变形公式：U = IR R = U / I

注意：R = U/I 是电阻的计算式，不是决定式。R由导体本身性质决定，与U和I无关。不能说"R与U成正比，与I成反比"。

欧姆定律适用于纯电阻电路（如电灯、电炉等），不适用于含电动机等非纯电阻元件。

📐 伏安法测电阻：
用电压表测出待测电阻两端的电压U，用电流表测出通过待测电阻的电流I，根据R = U/I计算出电阻值。
• 实验电路：电源、开关、待测电阻、滑动变阻器（串联）、电流表（串联）、电压表（并联）
• 滑动变阻器作用：①保护电路（闭合开关前调到最大阻值）；②改变待测电阻两端电压和通过的电流，以便多次测量求平均值减小误差
• 多次测量求平均值可以减小实验误差（注意：测小灯泡电阻时不能求平均值，因为灯丝电阻随温度变化）。

7.3 串联电路

串联电路

串联电路——各元件逐个顺次连接。电流只有一条路径，各用电器相互影响（一个断开，全部停止工作）。

物理量关系式说明
电流 II = I₁ = I₂串联电路中各处电流相等
电压 UU = U₁ + U₂总电压等于各部分电压之和
电阻 RR = R₁ + R₂总电阻等于各电阻之和（越串越大）
分压原理U₁/U₂ = R₁/R₂电压分配与电阻成正比
💡 串联电路分压原理应用：
• 电阻越大，分得的电压越多（U₁/U₂ = R₁/R₂）
• 串联电阻有分压作用：在电路中串联一个电阻可以分担一部分电压
• 推导：U₁ = IR₁ = [U/(R₁+R₂)] × R₁ = U × R₁/(R₁+R₂)
• 串联电路中，总电阻大于任何一个分电阻：R > R₁, R > R₂

7.4 并联电路

并联电路

并联电路——各元件并列连接在电路中。电流有多条路径，各支路用电器独立工作，互不影响。

物理量关系式说明
电压 UU = U₁ = U₂各支路电压相等，等于电源电压
电流 II = I₁ + I₂干路电流等于各支路电流之和
电阻 R1/R = 1/R₁ + 1/R₂总电阻的倒数等于各电阻倒数之和
分流原理I₁/I₂ = R₂/R₁电流分配与电阻成反比
💡 并联电路分流原理应用：
• 电阻越大，分得的电流越少（I₁/I₂ = R₂/R₁）
• 并联电阻有分流作用：在电路中并联一个电阻可以分担一部分电流
• 并联电路中，总电阻小于任何一个分电阻：R < R₁, R < R₂
• n个相同的电阻R₀并联：R总 = R₀/n
• 两个电阻并联：R = R₁R₂/(R₁+R₂)（积除以和）

⚡ 串联与并联电路对比

串联电路

R总=R₁+R₂, I处处相等, U=U₁+U₂

并联电路

1/R总=1/R₁+1/R₂, U处处相等, I=I₁+I₂

7.5 电路故障分析

断路（开路）

断路：电路中某处断开，电路中没有电流通过。

常见原因：开关断开、导线断裂、灯丝烧断、接线柱松动。

现象：

• 用电器不工作（灯不亮、电机不转）

• 电流表无示数

• 电压表接在断路处两端时有示数（等于电源电压），接在其他地方无示数

短路

短路：电流不经过用电器而直接构成回路。

电源短路：导线直接连接电源两极，电流极大，会烧坏电源（非常危险！）。

局部短路（用电器短路）：导线并联在某用电器两端，电流绕过该用电器，该用电器不工作。

现象：

• 被短路的用电器不工作

• 电流表示数变大（总电阻变小）

• 电压表测被短路元件两端示数为零

⚠ 电路故障判断方法：
• 电流表判断法：电流表有示数 → 电路是通路；无示数 → 可能是断路
• 电压表判断法：电压表有示数且接近电源电压 → 与电压表并联的部分断路；电压表无示数 → 与电压表并联的部分短路或外部断路
• 导线检测法：用一根导线依次并联在各元件两端，若其他用电器恢复工作，则该元件断路
• 严禁用导线直接连接电源两极来检测（会造成电源短路）！

7.6 串并联电路识别方法

① 定义法：元件逐个顺次连接为串联；并列连接为并联。

② 电流路径法：电流只有一条路径无分支为串联；有多条路径有分支为并联。

③ 拆除法：去掉任意一个用电器，其他用电器都不能工作为串联；其他仍能工作为并联。

④ 节点法：电路中各元件两端用字母标记，若元件两端字母相同则并联；若各元件首尾依次相连则串联。

📝 章节练习题

1欧姆定律的表达式是____。
✅ B。欧姆定律基本表达式为 I=U/R，表示导体中的电流与电压成正比、与电阻成反比。U=IR 和 R=U/I 是变形公式。
2在串联电路中，总电阻R与各电阻R₁、R₂的关系是____。
✅ B。串联电路总电阻等于各电阻之和 R=R₁+R₂。并联电路才是 1/R=1/R₁+1/R₂。
3我国家庭电路的电压是____。
✅ C。我国家庭电路电压为220V。安全电压不高于36V，工业动力用电为380V，一节干电池为1.5V。
4并联电路中，各支路两端的电压____。
✅ C。并联电路各支路电压相等 U=U₁=U₂，干路电流 I=I₁+I₂。这是并联电路最重要的特征之一。
5一节新干电池的电压约为____。
✅ A。一节新干电池电压为1.5V，铅蓄电池为2V，手机锂电池约为3.7V。
6导体的电阻与下列哪个因素无关？
✅ D。电阻是导体本身的性质，由材料、长度、横截面积和温度决定，与电压和电流无关。R=U/I 只是计算式，不是决定式。
7两个电阻R₁=3Ω、R₂=6Ω并联，总电阻为____。
✅ B。并联总电阻 R = R₁R₂/(R₁+R₂) = 3×6/(3+6) = 18/9 = 2Ω。注意：并联总电阻小于任一分电阻。
8在电路中，若某用电器两端电压为零，可能是____。
✅ A。用电器两端电压为零，可能是该用电器被短路（导线并联在两端），也可能是电路其他部分断路导致没有电流流过该用电器。
