7.1 电流 电压

电流（I）

电流：电荷的定向移动形成电流。方向规定为正电荷定向移动的方向。

单位：安培（A），常用 mA、μA。

1A = 1000 mA，1mA = 1000 μA。

测量工具：电流表（串联在电路中，正进负出）。

形成持续电流的条件：有电源、电路闭合。

电压（U）

电压是形成电流的原因。电源是提供电压的装置。

单位：伏特（V），常用 kV、mV。

1kV=1000V，1V=1000mV。

测量工具：电压表（并联在电路中，正进负出）。

常见电压：干电池1.5V、家庭电路220V、安全电压不高于36V。

电阻（R）

电阻：导体对电流的阻碍作用。

单位：欧姆（Ω），常用kΩ、MΩ。

影响因素：材料、长度、横截面积、温度。

（长度越长电阻越大，横截面积越大电阻越小）

滑动变阻器：通过改变接入电路中电阻线的长度来改变电阻。

7.2 欧姆定律 电阻

欧姆定律

导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。这是电学最基本的定律。

I = U / R

变形：U = IR，R = U/I（R是导体本身的性质，与U和I无关）。

7.3 串联电路

串联电路特点

串联电路——各元件逐个顺次连接。

  • 电流：I = I₁ = I₂（各处电流相等）
  • 电压：U = U₁ + U₂（总电压等于各部分电压之和）
  • 电阻：R = R₁ + R₂（总电阻等于各电阻之和）
  • 电压分配：U₁/U₂ = R₁/R₂（分压原理）

7.4 并联电路

并联电路特点

并联电路——各元件并列连接在电路中。

  • 电流：I = I₁ + I₂（干路电流等于各支路电流之和）
  • 电压：U = U₁ = U₂（各支路电压相等，等于电源电压）
  • 电阻：1/R = 1/R₁ + 1/R₂（总电阻的倒数等于各电阻倒数之和）
  • 电流分配：I₁/I₂ = R₂/R₁（分流原理）

⚡ 串联与并联电路对比

串联电路

R总=R₁+R₂, I处处相等, U=U₁+U₂

并联电路

1/R总=1/R₁+1/R₂, U处处相等, I=I₁+I₂

📝 章节练习题

1欧姆定律的表达式是____。
✅ B。欧姆定律基本表达式为I=U/R，变形得U=IR和R=U/I。
2在串联电路中，总电阻R与各电阻R₁、R₂的关系是____。
✅ B。串联总电阻R=R₁+R₂，并联则是1/R=1/R₁+1/R₂。
3我国家庭电路的电压是____。
✅ C。我国家庭电路电压为220V。安全电压不高于36V。
4并联电路中，各支路两端的电压____。
✅ C。并联电路各支路电压相等U=U₁=U₂，干路电流I=I₁+I₂。
5一节新干电池的电压约为____。
✅ A。一节新干电池电压为1.5V，铅蓄电池为2V，锂电池3.7V。
6导体的电阻与下列哪个因素无关？
✅ D。电阻是导体本身的性质，与材料、长度、横截面积和温度有关，与电压和电流无关。
