1压强计算

一个质量为 50kg 的人站立在水平地面上，双脚与地面的接触面积为 400cm²。求：

(1) 人受到的重力；

(2) 人对地面的压强。（g = 10N/kg）

答案：(1) 500N；(2) 1.25×10⁴Pa
解析：
(1) G = mg = 50kg × 10N/kg = 500N
(2) F = G = 500N，S = 400cm² = 0.04m²
p = F/S = 500N / 0.04m² = 1.25×10⁴Pa
2浮力计算

一个体积为 1000cm³ 的木块浸没在水中，求：

(1) 木块受到的浮力；

(2) 若木块漂浮时有 2/5 体积露出水面，求木块密度。（ρ水 = 1.0×10³kg/m³，g = 10N/kg）

答案：(1) 10N；(2) 0.6×10³kg/m³
解析：
(1) 浸没时 V排 = V木 = 1×10⁻³m³
F浮 = ρ水gV排 = 1×10³×10×1×10⁻³ = 10N
(2) 漂浮时 F浮 = G木，ρ水gV排 = ρ木gV木
V排 = (3/5)V木，∴ ρ木 = (3/5)ρ水 = 0.6×10³kg/m³
3液体压强

一个水池水深 3m，求：

(1) 池底受到水的压强；

(2) 若池底有一个面积为 10cm² 的排水口，求排水口受到水的压力。（ρ水 = 1.0×10³kg/m³，g = 10N/kg）

答案：(1) 3×10⁴Pa；(2) 30N
解析：
(1) p = ρgh = 1×10³ × 10 × 3 = 3×10⁴Pa
(2) S = 10cm² = 1×10⁻³m²，F = pS = 3×10⁴ × 1×10⁻³ = 30N
4阿基米德原理

一个物体在空气中称重为 15N，浸没在水中称重为 10N。求：

(1) 物体受到的浮力；(2) 物体的体积；(3) 物体的密度。（ρ水 = 1.0×10³kg/m³，g = 10N/kg）

答案：(1) 5N；(2) 5×10⁻⁴m³；(3) 3×10³kg/m³
解析：
(1) F浮 = G - F示 = 15 - 10 = 5N（称重法）
(2) F浮 = ρ水gV排，V = V排 = F浮/(ρ水g) = 5/(10³×10) = 5×10⁻⁴m³
(3) ρ = m/V = (G/g)/V = 1.5/(5×10⁻⁴) = 3×10³kg/m³
5功和功率

起重机将 2t 的重物匀速提升 10m，用时 20s。求：

(1) 起重机做的功；(2) 起重机的功率。（g = 10N/kg）

答案：(1) 2×10⁵J；(2) 1×10⁴W（10kW）
解析：
(1) G = mg = 2×10³×10 = 2×10⁴N
W = Gh = 2×10⁴ × 10 = 2×10⁵J
(2) P = W/t = 2×10⁵/20 = 1×10⁴W = 10kW
6杠杆平衡

一根均匀杠杆长 1m，支点离左端 30cm。在左端挂 60N 的重物。求：

(1) 在右端需加多大的力才能使杠杆平衡？

(2) 这是省力杠杆还是费力杠杆？

答案：(1) 约 25.7N；(2) 费力杠杆
解析：
(1) L₁ = 30cm，L₂ = 100-30 = 70cm
F₁L₁ = F₂L₂ → 60×30 = F₂×70，F₂ = 25.7N
(2) L₁ < L₂，动力臂 < 阻力臂→费力杠杆（但省距离）
7滑轮组

用一个动滑轮（不计滑轮重和摩擦）提升重 200N 的物体。求：

(1) 拉力 F 的大小；(2) 绳子自由端移动 4m 时，物体上升的高度；(3) 该滑轮组的机械效率。

答案：(1) 100N；(2) 2m；(3) 100%
解析：
(1) 动滑轮（单股）：F = G/2 = 200/2 = 100N
(2) s = 2h，h = s/2 = 4/2 = 2m
(3) 不计滑轮重和摩擦→η = 100%。若计滑轮重，η < 100%
8机械效率

用斜面将重 500N 的物体推上高 2m 的卡车。斜面长 5m，推力为 250N。求：

(1) 有用功；(2) 总功；(3) 斜面的机械效率；(4) 摩擦力做的额外功。

答案：(1) 1000J；(2) 1250J；(3) 80%；(4) 250J
解析：
(1) W有 = Gh = 500×2 = 1000J
(2) W总 = Fs = 250×5 = 1250J
(3) η = W有/W总 = 1000/1250 = 80%
(4) W额 = W总 - W有 = 1250 - 1000 = 250J（克服摩擦力做功）
9浮力与密度综合

一块金属在空气中重 27N，浸没在水中重 17N，浸没在某液体中重 19N。求：

(1) 金属的体积；(2) 金属的密度；(3) 该液体的密度。（ρ水 = 1.0×10³kg/m³，g = 10N/kg）

答案：(1) 1×10⁻³m³；(2) 2.7×10³kg/m³；(3) 0.8×10³kg/m³
解析：
(1) F浮水 = 27-17 = 10N，V = F浮/(ρ水g) = 10/(10³×10) = 1×10⁻³m³
(2) ρ金 = m/V = (27/10)/1×10⁻³ = 2.7×10³kg/m³（可能是铝）
(3) F浮液 = 27-19 = 8N，ρ液 = F浮/(Vg) = 8/(1×10⁻³×10) = 0.8×10³kg/m³
10压强综合

一个正方体铁块边长为 10cm，放在水平桌面上。（ρ铁 = 7.9×10³kg/m³，g = 10N/kg）

(1) 铁块对桌面的压强是多少？

(2) 若在铁块上施加一个竖直向下 50N 的力，压强变为多少？

答案：(1) 7.9×10³Pa；(2) 1.29×10⁴Pa
解析：
(1) V = 0.1³ = 1×10⁻³m³，m = ρV = 7.9×10³×1×10⁻³ = 7.9kg
G = 79N，S = 0.01m²，p = 79/0.01 = 7.9×10³Pa
(2) F总 = 79+50 = 129N，p = 129/0.01 = 1.29×10⁴Pa
11大气压强

托里拆利实验中，玻璃管内水银柱高度为 760mm。求：

(1) 大气压强是多少 Pa？

(2) 若将玻璃管倾斜，水银柱的竖直高度如何变化？

(3) 若换用更粗的玻璃管，水银柱高度如何变化？（ρ水银 = 13.6×10³kg/m³，g = 10N/kg）

答案：(1) 1.0336×10⁵Pa；(2) 不变；(3) 不变
解析：
(1) p = ρgh = 13.6×10³×10×0.76 = 1.0336×10⁵Pa
(2) 倾斜后水银柱长度增加，但竖直高度不变（始终等于大气压支撑的高度）
(3) 水银柱高度与管粗细无关，只与大气压有关
12运动和力

一辆质量为 1.5t 的汽车在水平公路上匀速行驶，受到的阻力为车重的 0.1 倍。求：

(1) 汽车的牵引力；(2) 车速为 72km/h 时，牵引力的功率。（g = 10N/kg）

答案：(1) 1500N；(2) 3×10⁴W（30kW）
解析：
(1) G = 1500×10 = 1.5×10⁴N，f = 0.1G = 1500N
匀速→F牵 = f = 1500N
(2) v = 72km/h = 20m/s，P = Fv = 1500×20 = 3×10⁴W
13惯性

关于惯性，下列说法正确的是：

A. 物体静止时没有惯性

B. 物体运动速度越大惯性越大

C. 惯性是物体的固有属性，与运动状态无关

D. 物体受力时惯性消失

答案：C
解析：
惯性是物体保持原来运动状态不变的性质，是物体的固有属性。一切物体在任何情况下都有惯性。惯性大小只与质量有关，与速度、是否受力、是否运动无关。
14二力平衡

一个重 20N 的物体静止在水平桌面上。求：

(1) 物体受到的支持力；

(2) 若用 5N 的水平力推物体，物体仍静止，求摩擦力；

(3) 若用 15N 的水平力推物体，物体匀速运动，求摩擦力。

答案：(1) 20N；(2) 5N；(3) 15N
解析：
(1) 静止→二力平衡：N = G = 20N
(2) 静止→推力=静摩擦力：f = 5N
(3) 匀速→推力=滑动摩擦力：f = 15N。滑动摩擦力由压力和粗糙程度决定，与推力大小无关
15力学综合压轴

如图，杠杆 AB 长 1.2m，支点 O 距 A 端 0.4m。A 端挂一个体积为 2×10⁻³m³ 的物体浸没在水中，B 端施加竖直向下的力 F 使杠杆水平平衡。已知物体密度为 4×10³kg/m³，g = 10N/kg。

(1) 物体受到的重力和浮力；

(2) A 端绳子对杠杆的拉力；

(3) B 端力 F 的大小。

答案：(1) G=80N，F浮=20N；(2) 60N；(3) 30N
解析：
(1) m = ρV = 4×10³×2×10⁻³ = 8kg，G = 80N
F浮 = ρ水gV = 10³×10×2×10⁻³ = 20N
(2) 绳子拉力 T = G - F浮 = 80-20 = 60N（物体浸没，拉力减小）
(3) OA = 0.4m，OB = 1.2-0.4 = 0.8m
T×OA = F×OB → 60×0.4 = F×0.8，F = 30N
