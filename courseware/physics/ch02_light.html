2.1 光的反射

光的直线传播

光在同种均匀介质中沿直线传播。光在真空中传播速度最快：c = 3×10⁸ m/s。

实例：小孔成像、日食月食、影子的形成。

光速 c = 3×10⁸ m/s（真空中）

光的反射定律

光射到物体表面时，一部分光被物体表面反射回去的现象叫做光的反射。

反射定律：

  • 反射光线、入射光线和法线在同一平面内（三线共面）
  • 反射光线和入射光线分居法线两侧（两线分居）
  • 反射角等于入射角：∠r = ∠i（两角相等）

镜面反射与漫反射：都遵循反射定律。漫反射使我们能从不同方向看到物体。

2.2 光的折射

光的折射规律

光从一种介质斜射入另一种介质时，传播方向发生改变的现象叫做光的折射。

  • 折射光线、入射光线和法线在同一平面内
  • 光从空气斜射入水（或玻璃）中时：折射角 < 入射角（向法线偏折）
  • 光从水（或玻璃）斜射入空气中时：折射角 > 入射角（远离法线偏折）
  • 垂直入射时：传播方向不变

实例：筷子在水中"弯折"、池水看起来变浅、海市蜃楼。

2.3 透镜成像

凸透镜

中间厚、边缘薄的透镜。对光线有会聚作用（会聚透镜）。

三条特殊光线：

  • 平行于主光轴 → 过焦点
  • 过光心 → 方向不变
  • 过焦点 → 平行射出
1/f = 1/u + 1/v

f=焦距 u=物距 v=像距

凸透镜成像规律

物距 u像的性质应用
u > 2f倒立、缩小、实像照相机
u = 2f倒立、等大、实像—
f < u < 2f倒立、放大、实像投影仪
u = f不成像平行光源
u < f正立、放大、虚像放大镜

凹透镜

中间薄、边缘厚的透镜。对光线有发散作用（发散透镜）。

应用：近视眼镜片。

2.4 光的色散

白光（太阳光）通过三棱镜后分解为红、橙、黄、绿、蓝、靛、紫七种色光的现象叫做光的色散。

这说明白光是由各种色光混合而成的复色光。

光的三原色：红、绿、蓝（RGB）。三种颜色的光按不同比例混合可以得到各种颜色的光。

应用：彩色电视机屏幕、电脑显示器。

📝 章节练习题

1光在真空中的传播速度是____。
✅ C。光在真空中的速度为3×10⁸ m/s。
2光的反射定律中，反射角____入射角。
✅ C。反射角等于入射角，这是反射定律的核心内容。
3当物体放在凸透镜的焦点以内（u<f）时，所成的像是____。
✅ C。u<f时成正立、放大、虚像，应用为放大镜。
4下列现象中，属于光的折射的是____。
✅ B。筷子在水中"弯折"是光的折射现象。水中倒影和平面镜成像是反射，小孔成像是直线传播。
5光的三原色是____。
✅ B。光的三原色是红(R)、绿(G)、蓝(B)。注意颜料三原色是青、品红、黄，两者不同。
