第八章 电能与磁
🎯学习目标
  • 掌握电功和电功率的概念及计算
  • 理解焦耳定律（电流的热效应）
  • 了解电流的磁效应（奥斯特实验）
  • 了解电磁感应现象和发电机原理
  • 了解电能的输送和无线电通信

8.1 电功率

电功

电流做的功叫做电功。电流做功的过程是将电能转化为其他形式能的过程。

W = UIt

单位：焦耳（J）。常用单位：千瓦时（kW·h，度）。

1 kW·h = 3.6×10⁶ J。

电功率

电功率：电流在单位时间内所做的功，表示电流做功的快慢。

P = W/t = UI

单位：瓦特（W），常用 kW。

推导公式（纯电阻电路）：P = I²R = U²/R

额定功率：用电器在额定电压下工作时的功率。

实际功率：用电器在实际电压下工作时的功率。

焦耳定律

电流通过导体产生的热量跟电流的平方成正比，跟导体的电阻成正比，跟通电时间成正比。

Q = I²Rt

应用：电热水器、电饭锅、电烙铁等。

8.2 电流的磁场

电流的磁效应

奥斯特实验：通电导线周围存在磁场，磁场方向与电流方向有关。这证明了电可以生磁。

通电螺线管的磁场与条形磁铁的磁场相似。右手螺旋定则（安培定则）：右手握住螺线管，四指弯曲方向为电流方向，拇指所指即为螺线管N极。

电磁铁：内部带有铁芯的通电螺线管。优点：磁性有无可控（通断电）、磁性强弱可调（改变电流）、极性可换（改变电流方向）。

电磁继电器：用低电压、弱电流控制高电压、强电流的自动开关。

8.3 电能的获得和输送

电磁感应

电磁感应：闭合电路的一部分导体在磁场中做切割磁感线运动时，导体中产生电流的现象。产生的电流叫感应电流。这是磁生电的现象。

感应电流方向与导体运动方向和磁场方向有关。

发电机原理：电磁感应（机械能→电能）。

电动机原理：通电导体在磁场中受力（电能→机械能）。

交流电：方向和大小做周期性变化的电流。我国交流电频率为50 Hz，周期0.02秒。

电能的输送

采用高压输电可以减少电能在输电线上的损失。

原理：P = UI，输送功率一定时，电压越高，电流越小。输电线上损失功率 P损 = I²R，电流越小，损失越小。

变压器：用于改变交流电的电压。升压变压器用于输电端，降压变压器用于用户端。

8.4 无线电波和无线电通信

电磁波

电磁波可以在真空中传播，传播速度等于光速 c = 3×10⁸ m/s。

波长λ、频率f、波速c的关系：c = λf

电磁波谱（按频率从低到高）：无线电波、微波、红外线、可见光、紫外线、X射线、γ射线。

应用：广播、电视、手机通信、Wi-Fi、雷达、微波炉。

1电功率的计算公式是____。
✅ B。P=W/t=UI，电功率等于电压乘以电流。
2焦耳定律的表达式是____。
✅ B。焦耳定律Q=I²Rt，电流通过导体产生的热量与电流平方成正比。
3发电机的工作原理是____。
✅ B。发电机原理是电磁感应（磁生电）。电动机原理是通电导体在磁场中受力（电生力）。
41千瓦时（度）等于____焦耳。
✅ B。1kW·h=1000W×3600s=3.6×10⁶J。
