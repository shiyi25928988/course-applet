11.1 原子结构：卢瑟福散射实验

卢瑟福 α 粒子散射实验 (1911)

实验装置：放射源发射 α 粒子（He²⁺ 核），轰击极薄的金箔（~10⁻⁷ m），周围用荧光屏检测散射 α 粒子。

主要发现：

  • 绝大多数 α 粒子直接穿过金箔——原子内大部分是空的。
  • 少数 α 粒子发生大角度偏转（> 90°）——原子中心有一个质量集中、带正电的核。
  • 极少数 α 粒子被反弹回来——原子核的质量远大于 α 粒子。

结论：原子由带正电的原子核和核外电子组成，核的体积远小于原子体积（~10⁻¹⁵ m vs ~10⁻¹⁰ m）。

原子核的组成

原子核由质子 (p) 和中子 (n) 组成，统称核子。

质子带 +e 电荷，中子不带电。两者质量相近：m_p ≈ m_n ≈ 1.67 × 10⁻²⁷ kg。

原子序数 Z = 质子数，质量数 A = 质子数 + 中子数。

核素符号：A_Z X（如 ²³⁸₉₂U 表示铀-238，有 92 个质子和 146 个中子）。

11.2 放射性衰变 (α, β, γ)

🟡 α 衰变 (Alpha Decay)

α 粒子 = 氦核 ⁴₂He，带 +2e 电荷。

母核 A 减少 4，Z 减少 2：

ᴬ_Z X → ᴬ⁻⁴_{Z-2} Y + ⁴₂He

穿透力弱（一张纸或几厘米空气可挡住），电离能力强。

示例：²³⁸₉₂U → ²³⁴₉₀Th + ⁴₂He

🔵 β⁻ 衰变 (Beta-minus Decay)

中子转变为质子，发射电子和反中微子：

n → p + e⁻ + ν̄_e

A 不变，Z 增加 1：

ᴬ_Z X → ᴬ_{Z+1} Y + e⁻ + ν̄_e

穿透力中等（几毫米铝板），电离能力中等。

示例：¹⁴₆C → ¹⁴₇N + e⁻ + ν̄_e

🟣 γ 衰变 (Gamma Decay)

γ 射线 = 高能电磁波（光子），无质量无电荷。

核从激发态跃迁到基态时发射：

ᴬ_Z X* → ᴬ_Z X + γ

A 和 Z 均不变。穿透力极强（需几厘米铅板），电离能力弱。

常伴随 α 或 β 衰变发生。

11.3 半衰期与衰变常数

半衰期 (Half-Life, T₁/₂)

半衰期：放射性原子核衰变到原来数量的一半所需的时间。

放射性衰变是随机过程，对单个原子核无法预测何时衰变，但对大量核的统计规律是确定的。

N = N₀ × (1/2)^(t/T₁/₂)

其中 N₀ 为初始核数，N 为 t 时刻剩余的核数，T₁/₂ 为半衰期。

衰变常数 (λ) 与活度 (A)

衰变常数 λ 表示每个原子核在单位时间内衰变的概率：

λ = ln 2 / T₁/₂

活度 A 定义为单位时间内发生衰变的核数：

A = λN = −dN/dt

活度单位：贝克勒尔 (Bq)，1 Bq = 1 次衰变/秒。活度也按指数衰减：A = A₀ × e^(−λt)。

☢️ 放射性衰变曲线模拟

调整参数观察衰变曲线。N = N₀(1/2)^(t/T₁/₂)。红色虚线标记半衰期位置。

11.4 核反应与质能方程

质能方程 (Mass-Energy Equivalence)

爱因斯坦质能方程：

E = mc²

质量与能量可以相互转化。c = 3.00 × 10⁸ m/s。

质量亏损 Δm = 反应前总质量 − 反应后总质量。

释放的能量：ΔE = Δm × c²。

常用单位：1 u = 931.5 MeV/c²（1 原子质量单位的能量当量）。

结合能 (Binding Energy)

结合能是将原子核分解为单独质子和中子所需的能量。

结合能 = 质量亏损 × c² = [Z×m_p + (A−Z)×m_n − m_nucleus] × c²。

每个核子的结合能 = 结合能 / A。

铁 (Fe-56) 具有最高的每个核子结合能，是最稳定的核素。

轻核聚变和重核裂变都能释放能量，因为产物核的每个核子结合能更大。

⚡ 核裂变 (Fission)

重核（如 ²³⁵U）吸收中子后分裂成两个中等质量核，释放能量和中子（链式反应）。

应用：核电站、原子弹。

²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 3n + 能量

🌟 核聚变 (Fusion)

轻核在高温下合并成更重的核，释放巨大能量。

应用：氢弹、恒星能量来源（太阳核心约 1500 万 K）。

²H + ³H → ⁴He + n + 17.6 MeV

11.5 粒子物理标准模型

基本粒子分类

标准模型将所有基本粒子分为两大类：

🔴 费米子 (Fermions) — 物质粒子

夸克 (Quarks)：u, d, c, s, t, b（6 种味）。

夸克带分数电荷：u 型 +2/3e，d 型 −1/3e。

夸克禁闭：夸克不能单独存在，只能组成强子。

轻子 (Leptons)：电子 e⁻, μ子 μ⁻, τ子 τ⁻ 及对应的中微子 ν_e, ν_μ, ν_τ。

轻子带整数电荷（−e 或 0），可单独存在。

🟢 玻色子 (Bosons) — 传递相互作用的粒子

光子 (γ)：传递电磁力。

胶子 (g)：传递强力（将夸克束缚在强子内）。

W⁺/W⁻/Z⁰：传递弱力（负责 β 衰变等）。

引力子 (graviton)：引力（未发现，不在标准模型中）。

希格斯玻色子 (H⁰)：赋予粒子质量（2012 年发现）。

强子 (Hadrons) — 由夸克组成的复合粒子

重子 (Baryons)：由 3 个夸克组成。如质子 (uud)、中子 (udd)。重子数 B = 1。

介子 (Mesons)：由 1 个夸克 + 1 个反夸克组成。如 π⁺ (ud̄)、π⁻ (ūd)、π⁰。

在反应中，重子数守恒、轻子数守恒、电荷守恒。

📝 章节练习题

1在卢瑟福 α 粒子散射实验中，少数 α 粒子发生大角度偏转说明了 ____。
✅ 正确答案：B。大角度偏转说明 α 粒子遇到了质量集中、带正电荷的原子核。A 选项说明大多数 α 粒子直接穿过，不是大角度偏转的原因。
2²³⁸₉₂U 经过一次 α 衰变后，产物的原子序数和质量数分别为 ____。
✅ 正确答案：A。α 衰变：A 减少 4 → 234，Z 减少 2 → 90。产物为 ²³⁴₉₀Th。
3某放射性核素的半衰期为 8 天。经过 24 天后，剩余的核数占原来的 ____。
✅ 正确答案：C。24 天 = 3 个半衰期，N = N₀(1/2)³ = N₀/8。
4在核反应中，质量亏损为 0.1 u，释放的能量约为 ____。(1 u = 931.5 MeV)
✅ 正确答案：B。ΔE = Δm × 931.5 = 0.1 × 931.5 = 93.15 MeV。
5质子的夸克组成为 ____。
✅ 正确答案：A。质子由两个上夸克和一个下夸克组成 (uud)，电荷：+2/3 + 2/3 − 1/3 = +1。中子为 udd，电荷：+2/3 − 1/3 − 1/3 = 0。
