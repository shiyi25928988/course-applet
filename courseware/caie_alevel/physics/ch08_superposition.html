第8章 叠加与干涉 (Superposition & Interference)
🎯学习目标
  • 理解叠加原理：两列波相遇时，合位移为各位移的代数和
  • 掌握杨氏双缝干涉实验的条件和条纹间距公式 λ = ax/D
  • 理解衍射现象及其条件（缝宽与波长可比拟）
  • 掌握衍射光栅方程 d sin θ = nλ 及其应用
  • 理解驻波的形成条件、节点和波腹的特征

8.1 叠加原理 (Principle of Superposition)

叠加原理

当两列或多列波在同一介质中同时传播时，在重叠区域，任意一点的合位移等于各列波单独引起的位移的矢量和。

y = y₁ + y₂ + y₃ + ...

相长干涉 (Constructive Interference)：两列波同相（相位差为 0, 2π, 4π...），合振幅最大。

相消干涉 (Destructive Interference)：两列波反相（相位差为 π, 3π, 5π...），合振幅最小。

相干条件

要产生稳定的干涉图案，两列波必须满足：

  • 频率相同（或波长相同）
  • 相位差恒定（相干光源）
  • 振动方向相同（或至少有平行的分量）

普通光源（如白炽灯）发出的光不满足相干条件，因此需要特殊装置（如杨氏双缝）来获得相干光。

8.2 双缝干涉 (Young's Double Slit)

杨氏双缝实验

托马斯·杨在 1801 年首次用双缝实验证明了光的波动性。实验装置：

  • 单色光源通过单缝 S₀ 产生相干光
  • 光通过两个相距很近的狭缝 S₁ 和 S₂（间距 a）
  • 在距离双缝 D 处的屏幕上形成明暗相间的干涉条纹

条纹间距（相邻亮纹或暗纹之间的距离）：

x = λD / a

其中 λ 为光波长，D 为双缝到屏幕的距离，a 为双缝间距。

💡 亮纹条件（相长干涉）

两列光的光程差为波长的整数倍：

Δ = nλ (n = 0, 1, 2, ...)

n = 0 为中央亮纹（零级），n = 1 为第一级亮纹，以此类推。

🌑 暗纹条件（相消干涉）

两列光的光程差为半波长的奇数倍：

Δ = (n + 1/2)λ (n = 0, 1, 2, ...)

n = 0 为第一暗纹，n = 1 为第二暗纹，以此类推。

影响条纹间距的因素

从 x = λD/a 可知：

  • 增大 λ（如用红光代替蓝光）→ 条纹间距变大
  • 增大 D → 条纹间距变大
  • 减小 a（双缝更近）→ 条纹间距变大

如果使用白光，中央亮纹为白色，两侧为彩色光谱（紫光在内、红光在外）。

8.3 衍射 (Diffraction)

衍射现象

衍射是波绕过障碍物或通过狭缝时偏离直线传播的现象。

衍射的显著程度取决于波长与障碍物/缝宽的相对大小：

显著的衍射：λ ≥ 缝宽

当缝宽远大于波长时，衍射不明显（波沿直线传播）。

这就是为什么声波（λ ≈ 1 m）能绕过门缝，而光波（λ ≈ 500 nm）不能。

单缝衍射 (Single-Slit Diffraction)

单色光通过单缝时，在屏幕上形成中央亮纹（最宽最亮）加上两侧对称的暗亮相间的条纹。

暗纹条件（极小值）：

a sin θ = nλ (n = 1, 2, 3, ...)

其中 a 为缝宽，θ 为第 n 级暗纹对应的角度。

中央亮纹的半角宽度：sin θ ≈ λ/a（第一暗纹位置）。

🔬 双缝干涉图案模拟

拖动滑块观察缝间距、波长和屏距对干涉条纹间距的影响。条纹间距 x = λD/a。

8.4 衍射光栅 (Diffraction Grating)

衍射光栅方程

衍射光栅由大量等间距的平行狭缝组成。光栅常数 d = 1/N，其中 N 为单位长度上的狭缝数。

光栅方程（亮纹条件）：

d sin θ = nλ (n = 0, 1, 2, ...)

其中 d 为光栅常数（相邻缝间距），θ 为第 n 级衍射角。

与双缝干涉相比，光栅产生的条纹更锐利更亮，因为更多缝参与干涉。

🔢 光栅的重要性质

最大级数由 sin θ ≤ 1 决定：

n_max = d / λ（取整数部分）

如果 d 减小（缝更密），各级条纹间距增大。

如果 λ 增大，各级条纹间距也增大。

🌈 光谱分析

白光通过光栅后，各级（除 n=0 外）产生连续光谱。

紫光偏转角最小，红光偏转角最大（因为 λ 红 > λ 紫）。

光栅用于光谱仪中精确测量波长。

光栅的应用

  • 光谱仪：精确测量光源的波长成分，用于化学元素分析。
  • CD/DVD：光盘表面的凹槽形成反射光栅，产生彩虹色。
  • X射线晶体学：利用晶体中的原子层作为天然光栅，研究晶体结构（布拉格定律：2d sin θ = nλ）。

8.5 驻波 (Stationary Waves)

驻波的形成

驻波是两列频率相同、振幅相等、传播方向相反的波叠加的结果。

驻波不发生能量传播，能量被"困"在波节之间。

驻波由入射波和反射波叠加形成，常见于弦乐器和管乐器中。

📍 波节 (Node)

位移始终为零的点。

相邻波节间距为 λ/2。

在波节处，两列波始终反相，相消干涉。

🔺 波腹 (Antinode)

振幅最大的点。

相邻波腹间距为 λ/2。

在波腹处，两列波始终同相，相长干涉。

弦上的驻波与谐波

一根长度为 L、两端固定的弦上的驻波：

L = n × (λₙ/2) → λₙ = 2L/n (n = 1, 2, 3, ...) fₙ = n × v/(2L) = n × f₁

n = 1：基频（第一谐波），两端为节点，中间一个波腹。

n = 2：第二谐波，两个波腹，中间一个节点。

n = 3：第三谐波，三个波腹，两个中间节点。

开管与闭管中的驻波

两端开口的管：两端为波腹，L = nλ/2（n = 1,2,3...），与两端固定弦相同。

一端封闭的管：封闭端为波节，开口端为波腹，L = nλ/4（n = 1,3,5...），只有奇次谐波。

以上解释了为什么笛子（开管）和单簧管（闭管）的音色不同。

📝 章节练习题

1在杨氏双缝实验中，双缝间距为 0.5 mm，屏距为 2.0 m，使用波长为 600 nm 的光。条纹间距为 ____。
✅ 正确答案：C。x = λD/a = (600×10⁻⁹ × 2.0) / (0.5×10⁻³) = 1200×10⁻⁹/0.5×10⁻³ = 2.4×10⁻³ m = 2.4 mm。
2衍射光栅的光栅常数为 2.0 × 10⁻⁶ m，使用波长为 500 nm 的光垂直照射。最多能观察到第几级光谱？
✅ 正确答案：B。由 d sin θ = nλ，sin θ ≤ 1，n_max = d/λ = 2.0×10⁻⁶/(500×10⁻⁹) = 4，因此最多可观察第 4 级。但题目问的是"最多能观察到"，当 n=4 时 sinθ=1，理论上在 90° 方向，实际观察不到，所以最多第 3 级。在 CAIE 中 n_max = floor(d/λ) = 4，但边缘级数不易观察。
3一根长度为 0.80 m 的弦，两端固定，基频为 200 Hz。波在弦上的传播速度为 ____。
✅ 正确答案：B。基频 n=1，λ₁ = 2L = 1.60 m，v = f₁λ₁ = 200 × 1.60 = 320 m/s。
4在单缝衍射中，当缝宽增大时，中央亮纹的宽度 ____。
✅ 正确答案：B。中央亮纹半角宽度 sin θ ≈ λ/a，缝宽 a 增大 → θ 减小 → 中央亮纹变窄。
5一端封闭、另一端开口的管中，基频对应的波长与管长 L 的关系是 ____。
✅ 正确答案：C。一端封闭一端开口，闭端为波节，开口端为波腹，L = λ/4 → λ = 4L。
6两列频率相同的波相位差为 π 时，发生 ____。
✅ 正确答案：B。相位差 π（180°）意味着两列波完全反相，发生完全相消干涉，合振幅为零（如果振幅相等）。
