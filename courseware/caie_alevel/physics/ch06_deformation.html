第6章 固体形变

🎯 学习目标
  • 理解胡克定律 F = kx 及其适用条件
  • 掌握应力 (σ = F/A) 和应变 (ε = ΔL/L) 的概念
  • 理解杨氏模量 E = σ/ε 的物理意义和计算方法
  • 区分弹性形变和塑性形变
  • 能够解读应力-应变曲线中的关键特征点

6.1 胡克定律

胡克定律 (Hooke's Law)

在弹性限度内，弹簧的伸长量（或压缩量）与所施加的力成正比。

F = kx

其中：F 是施加的力 (N)，k 是弹簧的劲度系数 (spring constant, N m⁻¹)，x 是弹簧的形变量（伸长量或压缩量，m）。

弹簧的劲度系数 k：

  • k 越大，弹簧越"硬"，越难拉伸或压缩
  • k 取决于弹簧的材料、线径、圈数和直径
  • k 的单位：N m⁻¹

弹簧的串联与并联

串联 (Series)：相同力作用下，总伸长量相加。

1/ktotal = 1/k₁ + 1/k₂

并联 (Parallel)：相同伸长量下，总力相加。

ktotal = k₁ + k₂

F-x 图：胡克定律区域内的 F-x 图是一条过原点的直线，斜率 = k。直线下方的面积 = 储存在弹簧中的弹性势能 = ½Fx = ½kx²。

弹性限度 (Limit of Proportionality)

胡克定律只在弹性限度内成立。超过弹性限度后，F 与 x 不再成正比，但弹簧仍可能恢复原状（弹性形变）。

超过弹性极限 (Elastic Limit) 后，弹簧产生永久形变，不能完全恢复原状（塑性形变）。

6.2 应力与应变

应力 (Stress)

应力是单位横截面积上所受的力，衡量材料内部受力程度。

σ = F / A

单位：帕斯卡 (Pa)，即 N m⁻²。常用 MPa (10⁶ Pa) 或 GPa (10⁹ Pa)。

应力的类型：

  • 拉应力 (Tensile Stress)：使材料拉伸的应力
  • 压应力 (Compressive Stress)：使材料压缩的应力
  • 剪应力 (Shear Stress)：使材料层间滑动的应力

应变 (Strain)

应变是材料形变程度相对于原始尺寸的比例，是一个无量纲量（无单位）。

ε = ΔL / L

其中：ΔL 是长度变化量 (m)，L 是原始长度 (m)。

应变类型：

  • 拉应变 (Tensile Strain)：ΔL > 0，材料被拉长
  • 压应变 (Compressive Strain)：ΔL < 0，材料被压缩

注意：应变通常用百分比表示，如 ε = 0.002 = 0.2%。

6.3 杨氏模量

杨氏模量 (Young Modulus)

杨氏模量是材料在弹性范围内的刚度度量，定义为应力与应变的比值。

E = σ / ε = (F/A) / (ΔL/L) = FL / (A ΔL)

单位：Pa 或 N m⁻²。

杨氏模量的物理意义：

  • E 越大，材料越难被拉伸或压缩（越"刚"）
  • E 是材料的固有属性，与试样的尺寸和形状无关
  • 应力-应变图中直线部分的斜率 = 杨氏模量

常见材料的杨氏模量

材料 杨氏模量 (GPa)
金刚石~1200
钢 (Steel)~200
铜 (Copper)~120
铝 (Aluminium)~70
玻璃~70
橡胶~0.01-0.1

测定杨氏模量的实验

使用长金属丝、游标尺和螺旋测微计：

  1. 测量金属丝原长 L 和直径 d（计算截面积 A = πd²/4）
  2. 逐步增加砝码，测量伸长量 ΔL（使用游标尺或 travelling microscope）
  3. 绘制 F-ΔL 图，求斜率 = F/ΔL
  4. E = (F/ΔL) × (L/A) = 斜率 × L/A

为了减小误差，应使用长金属丝（使 ΔL 易于测量）和细金属丝（使应力较大，产生可测的应变）。

6.4 弹性形变与塑性形变

弹性形变 (Elastic Deformation)

定义：当外力移除后，材料能够完全恢复原来的形状和尺寸。

特征：

  • 形变是可逆的
  • 原子间键被拉伸但未断裂
  • 卸载时沿原路径返回（应力-应变图上为同一条线）
  • 储存在材料中的弹性势能完全释放

塑性形变 (Plastic Deformation)

定义：当外力移除后，材料不能完全恢复原来的形状，产生永久形变。

特征：

  • 形变是不可逆的
  • 原子层之间发生永久滑移
  • 卸载路径与加载路径不同，产生永久应变
  • 部分能量以热能形式散失（迟滞回线）

脆性材料 vs 韧性材料：

  • 脆性材料（如玻璃、陶瓷）：几乎没有塑性形变，直接断裂
  • 韧性材料（如铜、钢）：有明显的塑性形变区域，断裂前有颈缩(necking)

6.5 应力-应变曲线

典型韧性材料的应力-应变曲线

应力-应变曲线是材料力学性能的"身份证"，包含以下关键特征点：

  1. 比例极限 (Limit of Proportionality, P)：应力与应变成正比的终点。在此之前，材料服从胡克定律。直线斜率 = 杨氏模量。
  2. 弹性极限 (Elastic Limit, E)：材料能够完全恢复原状的最大应力。超过此点产生塑性形变。
  3. 屈服点 (Yield Point, Y)：材料在应力几乎不增加的情况下发生明显应变。有些材料（如低碳钢）有明显的上屈服点和下屈服点。
  4. 极限抗拉强度 (Ultimate Tensile Strength, UTS)：应力-应变曲线上的最高点，材料能承受的最大应力。在此之后出现颈缩 (necking)。
  5. 断裂点 (Fracture Point, F)：材料最终断裂的位置。

应力-应变曲线下的面积

应力-应变曲线与应变轴之间的面积代表单位体积材料在断裂前吸收的能量（应变能密度，单位：J m⁻³）。

韧性材料（如铜）的曲线下面积大，表示材料在断裂前能吸收大量能量（韧性好）。

脆性材料（如玻璃）的曲线下面积小，表示材料在断裂前吸收能量少（脆性）。

1一个弹簧的劲度系数为 200 N m⁻¹。当施加 50 N 的力时，弹簧的伸长量为：

答案：B
F = kx → x = F/k = 50/200 = 0.25 m

2一根长度为 2.0 m、横截面积为 1.0 × 10⁻⁶ m² 的金属丝，在 200 N 的拉力下伸长了 2.0 mm。其杨氏模量为：

答案：B
E = FL/(AΔL) = 200 × 2.0 / (1.0×10⁻⁶ × 2.0×10⁻³) = 400 / (2.0×10⁻⁹) = 2.0 × 10¹¹ Pa。这是钢的典型杨氏模量。

3在应力-应变曲线中，比例极限表示：

答案：B
比例极限是胡克定律成立的最大应力点，在此点之前应力与应变成正比。选项D描述的是弹性极限。

4两个劲度系数分别为 k₁ = 100 N m⁻¹ 和 k₂ = 200 N m⁻¹ 的弹簧并联。系统的等效劲度系数为：

答案：C
并联：ktotal = k₁ + k₂ = 100 + 200 = 300 N m⁻¹。串联时为 1/k = 1/100 + 1/200 → k = 66.7 N m⁻¹。

5以下哪种材料在应力-应变曲线中表现出最明显的塑性形变区域？

答案：C
铜是典型的韧性材料，在应力-应变曲线上有显著的塑性形变区域。玻璃和陶瓷是脆性材料，几乎没有塑性形变。铸铁也是脆性材料。
