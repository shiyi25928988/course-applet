10.1 根的定位 (Locating Roots)

符号变化法 (Sign Change Method)

若连续函数 f(x) 在区间 [a, b] 两端点的函数值符号相反，即 f(a) 和 f(b) 异号，则区间内至少存在一个根。

f(a) · f(b) < 0 → 存在 c ∈ (a, b)，使得 f(c) = 0

注意： 若 f(a) · f(b) > 0，则不能保证无根（可能有偶数个根）。

根的定位步骤

1. 计算整数点上的函数值：f(0), f(1), f(2), ...

2. 寻找函数值符号变化的相邻整数点

3. 确定根所在的具体区间

例： f(x) = x³ - 3x - 5，f(2) = 8 - 6 - 5 = -3，f(3) = 27 - 9 - 5 = 13，符号由负变正，故根在 (2, 3) 内。

10.2 迭代法 (Iteration)

不动点迭代法的基本思想

将方程 f(x) = 0 改写为 x = g(x) 的形式，然后从初始值 x₀ 出发，反复迭代：

xₙ₊₁ = g(xₙ)

若序列收敛，则极限值即为方程的一个根。

收敛条件

迭代 xₙ₊₁ = g(xₙ) 在根附近收敛的充分条件是：

|g'(x)| < 1 在根附近的区间上

若 |g'(x)| > 1，则迭代发散，不会逼近根。

例： f(x) = x² - 5x + 4 = 0，可改写为 x = (x² + 4)/5，则 g(x) = (x² + 4)/5。

迭代法的几何意义 — 蛛网图 (Cobweb Diagram)

在坐标平面上画出 y = x 和 y = g(x) 两条曲线，从 (x₀, 0) 出发，垂直向上到 y = g(x)，再水平到 y = x，再垂直到 y = g(x)... 形成一条"蛛网"状的折线，趋向交点。

📊 迭代法 — 蛛网图 (Cobweb Diagram)

g(x) = (x² + 4)/5，从 x₀ = 0 开始迭代，收敛到 x = 1

10.3 牛顿-拉夫森法 (Newton-Raphson Method)

牛顿-拉夫森公式

牛顿-拉夫森法是求解 f(x) = 0 的一种快速迭代方法，利用切线逼近根：

xₙ₊₁ = xₙ - f(xₙ) / f'(xₙ)

从初始估计 x₀ 出发，每次迭代用切线代替曲线，计算切线与 x 轴的交点作为新的近似值。

牛顿法的几何意义

在点 (xₙ, f(xₙ)) 处作曲线的切线，切线方程为：

y - f(xₙ) = f'(xₙ)(x - xₙ)

令 y = 0，得切线与 x 轴的交点 xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ)，这正是下一次迭代的值。

牛顿法的收敛性

牛顿法通常具有二次收敛速度，比普通迭代法快得多。但需要注意：

1. 初始值 x₀ 必须足够接近真实根

2. f'(xₙ) 不能为零（否则切线平行于 x 轴）

3. 在拐点附近可能不收敛或收敛到错误的根

牛顿法应用示例

求 √2 的近似值：令 f(x) = x² - 2 = 0，f'(x) = 2x。

xₙ₊₁ = xₙ - (xₙ² - 2)/(2xₙ) = (xₙ + 2/xₙ)/2

取 x₀ = 1：x₁ = (1 + 2)/2 = 1.5，x₂ = (1.5 + 2/1.5)/2 ≈ 1.4167，x₃ ≈ 1.4142... 快速收敛。

10.4 数值积分：梯形法则 (Trapezium Rule)

梯形法则公式

将区间 [a, b] 等分为 n 个宽度为 h = (b - a)/n 的子区间，用梯形面积之和近似曲线下的面积：

∫ₐᵇ f(x) dx ≈ (h/2) [y₀ + 2(y₁ + y₂ + ... + yₙ₋₁) + yₙ]

其中 yᵢ = f(xᵢ)，xᵢ = a + ih。

梯形法则的精度

梯形法则的误差约为 O(h²)。增加条数 n（减小 h）可以提高精度。

对于凹函数，梯形法则高估面积；对于凸函数，梯形法则低估面积。

例： 用 n = 4 条梯形近似 ∫₀² x² dx：

h = 0.5，x: 0, 0.5, 1, 1.5, 2

y: 0, 0.25, 1, 2.25, 4

∫₀² x² dx ≈ (0.5/2)[0 + 2(0.25 + 1 + 2.25) + 4] = 0.25 × 11 = 2.75

精确值 = 8/3 ≈ 2.667，梯形法则略有高估（因为 x² 是凸函数，等等，x² 是 convex 凸函数，梯形法则应该高估。2.75 > 2.667，确实高估了）。

📝 章节练习题

1f(x) = x³ - x - 2，f(1) = -2，f(2) = 4，则方程 f(x) = 0 的根在哪个区间内？
✅ 正确答案：B。f(1) = -2 < 0，f(2) = 4 > 0，符号变化，故根在 (1, 2) 内。
2迭代公式 xₙ₊₁ = (xₙ² + 2)/3，g(x) = (x² + 2)/3，则 g'(1) = ____。
✅ 正确答案：B。g'(x) = 2x/3，g'(1) = 2/3。|g'(1)| < 1，故在 x = 1 附近收敛。
3用牛顿法求 f(x) = x² - 5 = 0 的根，x₀ = 2，则 x₁ = ____。
✅ 正确答案：A。f'(x) = 2x，x₁ = 2 - (4-5)/4 = 2 + 1/4 = 2.25。
4用 n = 2 条梯形近似 ∫₀¹ eˣ dx，结果约为____。
✅ 正确答案：C。h = 0.5，x: 0, 0.5, 1；y: 1, e⁰·⁵≈1.6487, e¹≈2.7183。∫ ≈ (0.5/2)[1 + 2×1.6487 + 2.7183] = 0.25 × 7.0157 ≈ 1.754。
5关于牛顿-拉夫森法，以下说法正确的是____。
✅ 正确答案：C。牛顿法具有二次收敛速度，但需要初始值接近真实根，且需要 f'(x) ≠ 0。它需要一阶导数，不是二阶。牛顿法不是总是收敛的。
6迭代公式 xₙ₊₁ = 2xₙ - 1，g'(x) = 2，从 x₀ = 0 开始迭代，序列会____。
✅ 正确答案：C。|g'(x)| = 2 > 1，迭代发散。x₀ = 0 → x₁ = -1 → x₂ = -3 → x₃ = -7 → ... 发散到 -∞。
