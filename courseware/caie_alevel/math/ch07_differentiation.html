第7章 微分 (Differentiation)
🎯学习目标
  • 理解导数作为函数变化率（切线斜率）的几何意义
  • 掌握基本函数的导数公式
  • 熟练运用乘积法则、商的法则、链式法则
  • 能够求隐函数和参数方程定义的函数的导数
  • 利用导数求切线/法线方程，判断驻点性质

7.1 导数的定义

导数作为极限

函数 f(x) 在 x = a 处的导数定义为：

f'(a) = lim(h→0) [f(a+h) - f(a)] / h

几何意义：导数 f'(a) 是曲线 y = f(x) 在点 (a, f(a)) 处的切线斜率。

莱布尼茨记号

dy/dx = d/dx[f(x)]

dy/dx 表示 y 对 x 的变化率。二阶导数记作 d²y/dx²。

7.2 基本求导法则

常用函数的导数

d/dx(xⁿ) = n xⁿ⁻¹ d/dx(sin x) = cos x d/dx(cos x) = -sin x d/dx(tan x) = sec² x d/dx(eˣ) = eˣ d/dx(ln x) = 1/x

乘积法则

d/dx(uv) = u'v + uv'

例：d/dx(x² sin x) = 2x sin x + x² cos x

商的法则

d/dx(u/v) = (u'v - uv') / v²

例：d/dx(x/(x+1)) = (1·(x+1) - x·1)/(x+1)² = 1/(x+1)²

7.3 切线、法线与驻点

切线与法线方程

曲线 y = f(x) 在点 (a, f(a)) 处的：

切线方程：y - f(a) = f'(a)(x - a)

法线方程：y - f(a) = -1/f'(a)(x - a)（当 f'(a) ≠ 0）

驻点与增减性

驻点 (Stationary Point)：f'(x) = 0 的点。

利用一阶导数判断增减性：

f'(x) > 0 → 函数递增；f'(x) < 0 → 函数递减。

二阶导数判断驻点性质

在驻点 x = a 处，f'(a) = 0：

f''(a) > 0 → 极小值点

f''(a) < 0 → 极大值点

f''(a) = 0 → 需进一步判断（可能是拐点）

📈 切线随曲线变化

蓝色曲线 f(x) = x³ - 3x，橙色切线随 x 位置变化

7.4 链式法则与隐函数求导 (P2/P3)

链式法则 (Chain Rule)

dy/dx = dy/du × du/dx

例：y = (3x² + 1)⁵，令 u = 3x² + 1，则 dy/dx = 5u⁴ × 6x = 30x(3x² + 1)⁴。

隐函数求导 (Implicit Differentiation)

当 y 不能显式表达为 x 的函数时，对等式两边同时求导，y 视作 x 的函数：

d/dx(y²) = 2y × dy/dx

例：x² + y² = 25 → 2x + 2y·dy/dx = 0 → dy/dx = -x/y。

7.5 参数方程求导 (P3)

参数方程形式

若 x = f(t)，y = g(t)，则：

dy/dx = (dy/dt) / (dx/dt)

二阶导数：

d²y/dx² = d/dx(dy/dx) = d/dt(dy/dx) / (dx/dt)

📝 章节练习题

1d/dx (x⁴) = ____。
✅ 正确答案：A。d/dx(xⁿ) = n xⁿ⁻¹，所以 d/dx(x⁴) = 4x³。
2d/dx (sin 2x) = ____。
✅ 正确答案：B。链式法则：d/dx(sin 2x) = cos 2x × 2 = 2 cos 2x。
3曲线 y = x² - 2x + 3 在 x = 2 处的切线斜率为____。
✅ 正确答案：C。dy/dx = 2x - 2，在 x = 2 处，斜率为 2×2 - 2 = 2。
4f(x) = x³ - 3x，f'(x) = 0 时的 x 值为____。
✅ 正确答案：B。f'(x) = 3x² - 3 = 3(x² - 1) = 0 → x = ±1。
5d/dx (x² eˣ) = ____。
✅ 正确答案：D。乘积法则：d/dx(x²)·eˣ + x²·d/dx(eˣ) = 2x eˣ + x² eˣ = (x² + 2x) eˣ。
6若 y = (2x - 1)³，则 dy/dx = ____。
✅ 正确答案：B。链式法则：dy/dx = 3(2x-1)² × 2 = 6(2x-1)²。
