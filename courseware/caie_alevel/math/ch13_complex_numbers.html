第13章 复数 (Complex Numbers)
🎯学习目标
  • 理解虚数单位 i 的定义（i² = -1）及复数的代数形式 z = a + bi
  • 掌握复数的四则运算：加法、减法、乘法和除法
  • 能够在复平面 (Argand Diagram) 上表示复数并理解其几何意义
  • 掌握复数的极坐标形式 r(cos θ + i sin θ)，理解模与辐角
  • 熟练运用棣莫弗定理 (De Moivre's Theorem) 求幂和根

13.1 复数的引入：i² = -1，实部和虚部

虚数单位 i

在实数范围内，方程 x² + 1 = 0 无解。引入虚数单位 i，定义为：

i² = -1 或 i = √(-1)

由此所有负数都可以开平方：√(-a) = i√a（a > 0）。

复数的定义

复数 z 可以表示为 z = a + bi，其中 a, b 为实数：

  • a 称为实部 (Real Part)，记作 Re(z) = a
  • b 称为虚部 (Imaginary Part)，记作 Im(z) = b
  • 当 b = 0 时，z 为实数；当 a = 0 且 b ≠ 0 时，z 为纯虚数
z = a + bi，其中 a, b ∈ ℝ

i 的幂次循环

i¹ = i，i² = -1，i³ = -i，i⁴ = 1

i 的幂次以 4 为周期循环：i^(4k) = 1，i^(4k+1) = i，i^(4k+2) = -1，i^(4k+3) = -i。

13.2 复数的代数运算：加减乘除

加法与减法

设 z₁ = a + bi，z₂ = c + di，则：

z₁ + z₂ = (a + c) + (b + d)i z₁ - z₂ = (a - c) + (b - d)i

复数的加减法满足交换律和结合律。

乘法

z₁ × z₂ = (a + bi)(c + di) = (ac - bd) + (ad + bc)i

注意：i² = -1，所以 bd·i² = -bd。

共轭复数与除法

z = a + bi 的共轭复数 (Conjugate) 为 z* = a - bi。

z · z* = (a + bi)(a - bi) = a² + b²（实数）

除法：分子分母同乘以分母的共轭：

z₁ / z₂ = (a+bi)/(c+di) = ((a+bi)(c-di)) / (c² + d²)

13.3 复数的几何表示：复平面 (Argand Diagram)

复平面 (Argand Diagram)

复数 z = a + bi 可以用复平面上的点 (a, b) 表示：

  • 横轴（x 轴）为实轴 (Real Axis)
  • 纵轴（y 轴）为虚轴 (Imaginary Axis)
  • 复数 z 对应从原点指向 (a, b) 的向量

模 (Modulus)

复数 z = a + bi 的模 |z| 表示该点到原点的距离：

|z| = √(a² + b²)

性质：|z₁ · z₂| = |z₁| · |z₂|，|z₁ / z₂| = |z₁| / |z₂|。

辐角 (Argument)

辐角 arg(z) 是向量与正实轴之间的夹角，通常取主值范围 (-π, π] 或 [0, 2π)。

arg(z) = θ，其中 tan θ = b/a

注意：需要根据 a 和 b 的符号确定 θ 所在的象限。

13.4 极坐标形式：r(cos θ + i sin θ)，模与辐角

极坐标形式 (Polar Form / Modulus-Argument Form)

复数 z = a + bi 可以表示为极坐标形式：

z = r(cos θ + i sin θ)

其中 r = |z| = √(a² + b²)，θ = arg(z)，且 a = r cos θ，b = r sin θ。

极坐标形式的乘法

设 z₁ = r₁(cos θ₁ + i sin θ₁)，z₂ = r₂(cos θ₂ + i sin θ₂)，则：

z₁ · z₂ = r₁r₂ [cos(θ₁ + θ₂) + i sin(θ₁ + θ₂)]

乘法：模相乘，辐角相加。

极坐标形式的除法

z₁ / z₂ = (r₁/r₂) [cos(θ₁ - θ₂) + i sin(θ₁ - θ₂)]

除法：模相除，辐角相减。

13.5 棣莫弗定理 (De Moivre's Theorem) 及应用

棣莫弗定理

对于任意整数 n：

[r(cos θ + i sin θ)]ⁿ = rⁿ (cos nθ + i sin nθ)

这是极坐标形式乘法的自然推广：模取 n 次方，辐角乘以 n。

🔢 求复数的幂

将复数化为极坐标形式后，直接应用棣莫弗定理。

例如：(1 + i)⁸：先求 r = √2, θ = π/4，则 (1 + i)⁸ = (√2)⁸(cos 2π + i sin 2π) = 16。

(1 + i)⁸ = 16

🌱 求复数的 n 次方根

z^(1/n) = r^(1/n) [cos((θ+2kπ)/n) + i sin((θ+2kπ)/n)]，k = 0, 1, ..., n-1。

n 个根均匀分布在复平面以原点为圆心、r^(1/n) 为半径的圆上。

n 个根呈正 n 边形分布

三角恒等式的推导

棣莫弗定理可用于推导三角函数的倍角公式：

由 (cos θ + i sin θ)ⁿ = cos nθ + i sin nθ，展开左边并比较实部和虚部。

例如 n = 3：cos 3θ = 4cos³θ - 3cos θ，sin 3θ = 3sin θ - 4sin³θ。

cos 3θ = 4cos³θ - 3cos θ sin 3θ = 3sin θ - 4sin³θ

🎨 复平面上复数乘法可视化

拖动滑块观察两个复数及其乘积在复平面上的位置变化（蓝色=z₁，红色=z₂，绿色=z₁·z₂）

📝 章节练习题

1若 z = 3 + 4i，则 |z| = ____。
✅ 正确答案：B。|z| = √(3² + 4²) = √(9 + 16) = √25 = 5。
2计算 (2 + i)(3 - 2i) = ____。
✅ 正确答案：C。(2+i)(3-2i) = 6 - 4i + 3i - 2i² = 6 - i + 2 = 8 - i。
3复数 z = 1 + i 的辐角主值 arg(z) = ____。
✅ 正确答案：B。tan θ = 1/1 = 1，且 a=1>0, b=1>0，在第一象限，θ = π/4。
4利用棣莫弗定理，计算 (1 + i√3)⁶ = ____。
✅ 正确答案：A。r = √(1+3) = 2，θ = π/3，(1+i√3)⁶ = 2⁶(cos 2π + i sin 2π) = 64(1 + 0) = 64。
5方程 z³ = 8 在复数范围内有几个根？
✅ 正确答案：C。根据棣莫弗定理，z = 8^(1/3)[cos(2kπ/3) + i sin(2kπ/3)]，k=0,1,2，共3个根：2, -1+i√3, -1-i√3。
6若 z₁ = 2(cos 30° + i sin 30°)，z₂ = 3(cos 45° + i sin 45°)，则 z₁ · z₂ 的模为____。
✅ 正确答案：B。|z₁·z₂| = |z₁|·|z₂| = 2 × 3 = 6。辐角为 30° + 45° = 75°。
