🔧 力学部分 (Mechanics - M1)

M1.1 运动学：匀加速运动公式 (suvat equations)

五个基本量

匀加速直线运动中涉及五个基本物理量：

  • s — 位移 (displacement)，单位：m
  • u — 初速度 (initial velocity)，单位：m/s
  • v — 末速度 (final velocity)，单位：m/s
  • a — 加速度 (acceleration)，单位：m/s²
  • t — 时间 (time)，单位：s

五个 suvat 公式

v = u + at s = ut + (1/2)at² s = vt - (1/2)at² v² = u² + 2as s = (u + v)t / 2

每个公式包含四个量，已知其中三个可求第四个。解题时列出已知量并选择合适的公式。

重力加速度

在竖直方向运动中，加速度为重力加速度 g = 9.8 m/s²（或取 10 m/s²）。

取向上为正方向时，a = -g。

v = u - gt，s = ut - (1/2)gt²

M1.2 牛顿定律与力的平衡

🔵 牛顿第一定律

物体在不受外力（或合力为零）时保持静止或匀速直线运动状态。

ΣF = 0 → v 恒定

🔴 牛顿第二定律

物体的加速度与合力成正比，与质量成反比。

F = ma

F 为合力 (N)，m 为质量 (kg)，a 为加速度 (m/s²)。

🟢 牛顿第三定律

作用力与反作用力大小相等、方向相反，作用在不同物体上。

F₁₂ = -F₂₁

力的平衡条件

物体处于平衡状态时：

ΣF_x = 0（水平方向合力为零） ΣF_y = 0（竖直方向合力为零）

常见力：重力 (mg)、法向力 (R)、摩擦力 (F = μR)、张力 (T)。

斜面问题

物体在倾角为 θ 的斜面上：

沿斜面分量：mg sin θ 垂直斜面分量：mg cos θ

当物体沿斜面运动时，需分解重力并考虑摩擦力。

M1.3 动量与冲量

动量 (Momentum)

动量定义为质量与速度的乘积：

p = mv

动量是矢量，方向与速度方向相同。单位：kg·m/s。

动量守恒定律

在无外力作用的系统中，总动量守恒：

m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂

适用于碰撞和爆炸问题。

冲量 (Impulse)

冲量是力在时间上的累积效果：

I = F × Δt = Δp = mv - mu

冲量等于动量的变化量。单位：N·s。

📐 抛体运动轨迹可视化

拖动滑块调整初速度 u 和发射角 θ，观察抛体运动轨迹及最大高度和射程的变化

📊 统计部分 (Statistics - S1)

S1.1 数据的表示：直方图、累积频率、箱线图

📊 直方图 (Histogram)

用矩形面积表示频率。关键在于：

频数密度 = 频数 / 组距

面积 = 频数密度 × 组距 = 频数。

📈 累积频率图

绘制累积频率 (Cumulative Frequency) 对应上组界的折线图。

用于估计中位数、四分位数和百分位数。

中位数 ≈ Q₂，下四分位数 ≈ Q₁，上四分位数 ≈ Q₃

📦 箱线图 (Box Plot)

显示数据五数概括：最小值、Q₁、中位数、Q₃、最大值。

四分位距 IQR = Q₃ - Q₁，用于识别异常值。

异常值：< Q₁ - 1.5×IQR 或 > Q₃ + 1.5×IQR

S1.2 概率：条件概率、独立事件

条件概率

在事件 B 已发生的条件下，事件 A 发生的概率：

P(A|B) = P(A ∩ B) / P(B)，其中 P(B) > 0

乘法公式：P(A ∩ B) = P(A|B) × P(B) = P(B|A) × P(A)。

独立事件

若 A 和 B 独立，则：

P(A ∩ B) = P(A) × P(B) P(A|B) = P(A)，P(B|A) = P(B)

互斥事件 (Mutually Exclusive)：A ∩ B = ∅，P(A ∪ B) = P(A) + P(B)。

全概率公式与贝叶斯定理

P(A) = P(A|B₁)P(B₁) + P(A|B₂)P(B₂) + ... P(B|A) = P(A|B)P(B) / P(A)（贝叶斯定理）

S1.3 离散随机变量与正态分布

离散随机变量

随机变量 X 的取值及其概率分布：

期望 E(X) = Σ x · P(X = x) 方差 Var(X) = E(X²) - [E(X)]² E(aX + b) = aE(X) + b，Var(aX + b) = a²Var(X)

二项分布 (Binomial Distribution)

n 次独立试验，每次成功概率为 p：

X ~ B(n, p)，P(X = r) = C(n,r) × p^r × (1-p)^(n-r) E(X) = np，Var(X) = np(1-p)

正态分布 (Normal Distribution)

连续型分布，概率密度函数呈钟形曲线：

X ~ N(μ, σ²) Z = (X - μ) / σ ~ N(0, 1)（标准化）

约 68% 的数据在 μ ± σ 内，约 95% 在 μ ± 2σ 内，约 99.7% 在 μ ± 3σ 内。

📝 章节练习题

🔧 力学练习题 (Mechanics)

1一辆汽车从静止开始以 2 m/s² 匀加速行驶 8 秒，求行驶的距离。
✅ 正确答案：B。u=0, a=2, t=8 → s = ut + (1/2)at² = 0 + (1/2)×2×64 = 64 m。
2一个质量为 5 kg 的物体受到 20 N 的合力，则加速度为____。
✅ 正确答案：B。F = ma → 20 = 5a → a = 4 m/s²。
3质量为 2 kg 的小球以 6 m/s 的速度撞击墙壁后以 2 m/s 反向弹回，则冲量大小为____。
✅ 正确答案：B。I = mv - mu = 2×(-2) - 2×6 = -4 - 12 = -16，冲量大小为 16 N·s。

📊 统计练习题 (Statistics)

4一组数据：3, 7, 8, 5, 12, 14, 21, 18, 25。中位数是____。
✅ 正确答案：B。排序：3,5,7,8,12,14,18,21,25，共 9 个数，第 5 个为中位数 = 12。
5袋中有 4 个红球和 6 个蓝球，不放回地依次取出 2 个球。第一次取到红球且第二次取到蓝球的概率是____。
✅ 正确答案：A。P(红蓝) = (4/10) × (6/9) = 24/90 = 4/15。
6随机变量 X ~ B(10, 0.3)，则 E(X) = ____。
✅ 正确答案：A。对于二项分布 X ~ B(n, p)，E(X) = np = 10 × 0.3 = 3。
