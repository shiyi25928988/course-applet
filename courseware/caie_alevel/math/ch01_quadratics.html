第1章 二次函数 (Quadratics)
🎯学习目标
  • 掌握二次函数的三种表达形式：标准形式、顶点形式、因式分解形式
  • 理解判别式 b² - 4ac 与二次方程根的关系
  • 熟练运用配方法 (completing the square) 求顶点坐标
  • 掌握韦达定理 (Vieta's formulas) 解决根与系数的关系问题
  • 能够解二次不等式并正确表示解集

1.1 二次函数的标准形式

二次函数的三种形式

二次函数的一般形式为 f(x) = ax² + bx + c，其中 a ≠ 0。

标准形式：y = ax² + bx + c，可直接读出 y 轴截距 (0, c)。

顶点形式：y = a(x - h)² + k，顶点坐标为 (h, k)。

因式分解形式：y = a(x - p)(x - q)，其中 p 和 q 是函数的零点（x 轴截距）。

y = ax² + bx + c = a(x - h)² + k = a(x - p)(x - q)

配方法 (Completing the Square)

将标准形式转化为顶点形式的步骤：

  1. 提取 a：y = a(x² + (b/a)x) + c
  2. 括号内配方：y = a[(x + b/(2a))² - (b/(2a))²] + c
  3. 展开化简：y = a(x + b/(2a))² + [c - b²/(4a)]

顶点坐标为 (-b/(2a), c - b²/(4a))，对称轴为 x = -b/(2a)。

顶点 x = -b/(2a)，顶点 y = (4ac - b²)/(4a)

📈 抛物线随参数变化

拖动滑块观察 a（开口方向和宽度）、b（对称轴位置）、c（y轴截距）对抛物线的影响

1.2 判别式与根的性质

判别式 (Discriminant)

对于二次方程 ax² + bx + c = 0，判别式 Δ = b² - 4ac 决定了根的性质：

Δ = b² - 4ac

🔺 Δ > 0

方程有两个不相等的实根。

抛物线在 x 轴上有两个交点。

x = (-b ± √Δ) / (2a)

🔸 Δ = 0

方程有两个相等的实根（一个重根）。

抛物线与 x 轴相切（顶点在 x 轴上）。

x = -b / (2a)（重根）

🔻 Δ < 0

方程没有实根（有两个共轭复根）。

抛物线完全在 x 轴上方或下方。

无实根，抛物线不与 x 轴相交

判别式与直线交点

当判断一条直线 y = mx + c 与抛物线 y = ax² + bx + c 的交点数量时：

令 ax² + bx + c = mx + c → ax² + (b-m)x + (c-c) = 0，计算判别式。

Δ > 0：两个交点；Δ = 0：相切（一个交点）；Δ < 0：无交点。

1.3 韦达定理 (Vieta's Formulas)

根与系数的关系

设二次方程 ax² + bx + c = 0 的两个根为 α 和 β，则：

α + β = -b/a（根之和） α × β = c/a（根之积）

韦达定理可用于：

  • 已知一个根求另一个根
  • 构造以给定两个数为根的二次方程
  • 求关于根的对称表达式（如 α² + β²、α³ + β³、1/α + 1/β 等）

常用变形公式

α² + β² = (α + β)² - 2αβ α³ + β³ = (α + β)³ - 3αβ(α + β) 1/α + 1/β = (α + β) / (αβ) |α - β| = √((α + β)² - 4αβ) = √Δ / |a|

1.4 二次不等式

解二次不等式的方法

解 ax² + bx + c > 0（或 < 0, ≥ 0, ≤ 0）的步骤：

  1. 求出对应方程 ax² + bx + c = 0 的根（可用因式分解或求根公式）
  2. 画出抛物线草图，标出 x 轴截距
  3. 根据 a 的正负确定开口方向
  4. 根据不等号确定解集

📈 a > 0（开口向上）

ax² + bx + c > 0 的解集：x < 小根 或 x > 大根（两根之外）

ax² + bx + c < 0 的解集：小根 < x < 大根（两根之间）

📉 a < 0（开口向下）

ax² + bx + c > 0 的解集：小根 < x < 大根（两根之间）

ax² + bx + c < 0 的解集：x < 小根 或 x > 大根（两根之外）

恒正/恒负条件

对于所有实数 x，ax² + bx + c > 0 恒成立的条件：a > 0 且 Δ < 0。

对于所有实数 x，ax² + bx + c < 0 恒成立的条件：a < 0 且 Δ < 0。

📝 章节练习题

1将 y = 2x² - 8x + 5 写成顶点形式 y = a(x - h)² + k，则 h + k = ____。
✅ 正确答案：A。y = 2(x² - 4x) + 5 = 2[(x - 2)² - 4] + 5 = 2(x - 2)² - 3，顶点 (2, -3)，h + k = 2 + (-3) = -1。
2方程 3x² - 4x + k = 0 有两个相等的实根，则 k = ____。
✅ 正确答案：A。Δ = 0 → (-4)² - 4×3×k = 0 → 16 - 12k = 0 → k = 4/3。
3若 α 和 β 是方程 x² - 5x + 3 = 0 的两个根，则 α² + β² = ____。
✅ 正确答案：B。α + β = 5，αβ = 3，α² + β² = (α+β)² - 2αβ = 25 - 6 = 19。
4不等式 x² - 3x - 10 < 0 的解集是____。
✅ 正确答案：C。(x-5)(x+2) < 0，a=1>0开口向上，解集在两根之间：-2 < x < 5。
5直线 y = 2x + k 与抛物线 y = x² - 3x + 1 相切，则 k = ____。
✅ 正确答案：A。x² - 3x + 1 = 2x + k → x² - 5x + (1-k) = 0，Δ = 25 - 4(1-k) = 0 → 25 - 4 + 4k = 0 → k = -21/4。
6对于所有实数 x，不等式 2x² + 3x + m > 0 恒成立，则 m 的取值范围是____。
✅ 正确答案：A。a=2>0已满足，需 Δ < 0：9 - 8m < 0 → m > 9/8。
