第3章 坐标几何 (Coordinate Geometry)
🎯学习目标
  • 掌握直线方程的各种形式：点斜式、斜截式、一般式
  • 理解平行线和垂直线的斜率关系
  • 熟练计算两点间的距离和中点坐标
  • 掌握圆的标准方程和一般方程
  • 能够求解直线与圆的交点

3.1 直线方程

斜率的计算

过两点 A(x₁, y₁) 和 B(x₂, y₂) 的直线斜率为：

m = (y₂ - y₁) / (x₂ - x₁)

斜率表示直线的倾斜程度。m > 0 表示上升；m < 0 表示下降；m = 0 表示水平；m 无定义表示垂直。

点斜式

过点 (x₁, y₁)，斜率为 m：

y - y₁ = m(x - x₁)

斜截式

斜率为 m，y 轴截距为 c：

y = mx + c

一般式

最通用的直线方程形式：

ax + by + c = 0

平行线与垂直线

若两条直线的斜率分别为 m₁ 和 m₂：

平行（parallel）：m₁ = m₂（斜率相等）

垂直（perpendicular）：m₁ × m₂ = -1（斜率互为负倒数）

若 m₁ = 2，则垂线斜率 m₂ = -1/2

3.2 距离与中点

两点间距离公式

两点 A(x₁, y₁) 和 B(x₂, y₂) 之间的距离：

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

这是勾股定理在坐标几何中的应用。

中点坐标公式

线段 AB 的中点 M 的坐标为：

M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

3.3 圆的方程

圆的标准方程

圆心为 C(a, b)，半径为 r 的圆：

(x - a)² + (y - b)² = r²

圆的一般方程

展开标准式可得：

x² + y² + 2gx + 2fy + c = 0

圆心：(-g, -f)，半径：√(g² + f² - c)

注意：g² + f² - c > 0 才表示一个实圆；等于 0 表示一个点；小于 0 无意义。

3.4 直线与圆的交点

求解方法

将直线方程 y = mx + c 代入圆的方程 (x - a)² + (y - b)² = r²，得到一个关于 x 的二次方程。

使用判别式判断交点情况：

  • Δ > 0：直线与圆有两个交点（相交）
  • Δ = 0：直线与圆相切（一个交点）
  • Δ < 0：直线与圆无交点（相离）

切线方程

求圆上一点 (x₁, y₁) 处的切线方程：

切线垂直于该点处的半径，斜率 m_tangent = -1/m_radius。

半径斜率 = (y₁ - b)/(x₁ - a)，切线斜率 = -(x₁ - a)/(y₁ - b)。

(x₁ - a)(x - a) + (y₁ - b)(y - b) = r²

📝 章节练习题

1过点 A(1, 2) 和 B(4, 8) 的直线斜率为____。
✅ 正确答案：B。m = (8-2)/(4-1) = 6/3 = 2。
2直线 y = 3x - 1 的垂线斜率为____。
✅ 正确答案：D。m₁ × m₂ = -1，3 × m₂ = -1，m₂ = -1/3。
3点 A(-1, 3) 和点 B(5, -5) 之间的距离为____。
✅ 正确答案：B。d = √[(5-(-1))² + (-5-3)²] = √[36 + 64] = √100 = 10。
4圆 x² + y² - 6x + 4y - 12 = 0 的圆心坐标是____。
✅ 正确答案：B。2g = -6 → g = -3，2f = 4 → f = 2，圆心 (-g, -f) = (3, -2)。
5直线 y = 2x + 1 与圆 x² + y² = 5 的交点个数为____。
✅ 正确答案：C。x² + (2x+1)² = 5 → x² + 4x² + 4x + 1 = 5 → 5x² + 4x - 4 = 0，Δ = 16 + 80 = 96 > 0，两个交点。
6过点 (2, 3) 且与直线 y = -2x + 1 平行的直线方程为____。
✅ 正确答案：B。平行线斜率相同 m = -2，过 (2,3)：y - 3 = -2(x - 2) → y = -2x + 7。
