第9章 对数与指数函数 (Logarithms & Exponentials)
🎯学习目标
  • 理解指数函数 eˣ 的定义、图像和性质
  • 掌握对数函数 ln x 的定义、换底公式和性质
  • 能够求解指数方程和对数方程
  • 理解指数增长与衰减模型及其应用
  • 能够将指数关系转化为线性关系 (y = kxⁿ 和 y = abˣ)

9.1 指数函数 (Exponential Functions)

指数函数的定义

形如 f(x) = aˣ（a > 0, a ≠ 1）的函数称为指数函数。在 CAIE 9709 中，最常用的是以 e 为底的指数函数：

f(x) = eˣ，其中 e ≈ 2.71828...

e 是一个无理数，称为自然对数的底数（欧拉数）。

指数函数的性质

定义域： x ∈ R（所有实数）

值域： y > 0（始终为正）

关键点： 过 (0, 1)，因为 e⁰ = 1

单调性： eˣ 在整个定义域上严格递增

渐近线： y = 0（x → -∞ 时）

eˣ⁺ʸ = eˣ · eʸ (eˣ)ⁿ = eⁿˣ d/dx (eˣ) = eˣ

📊 指数函数 y = eˣ 的图像

y = eˣ 过 (0, 1)，y = 0 为水平渐近线

9.2 对数函数 (Logarithmic Functions)

对数的定义

若 aˣ = b，则 x = logₐ b。对数是指数的逆运算。

自然对数 ln x = logₑ x，是 eˣ 的反函数。

ln(eˣ) = x eˡⁿˣ = x（x > 0）

对数运算法则

logₐ(xy) = logₐ x + logₐ y logₐ(x/y) = logₐ x - logₐ y logₐ(xⁿ) = n logₐ x logₐ 1 = 0 logₐ a = 1

换底公式 (Change of Base)

logₐ b = (log꜀ b) / (log꜀ a)

特别地，常用对数 lg x = log₁₀ x，与自然对数 ln x 之间的转换：

lg x = ln x / ln 10 ln x = lg x / lg e

y = ln x 的图像和性质

定义域： x > 0

值域： y ∈ R

关键点： 过 (1, 0)，因为 ln 1 = 0

单调性： 严格递增

渐近线： x = 0（垂直渐近线）

d/dx (ln x) = 1/x

9.3 指数方程与对数方程

指数方程的解法

对于形如 aˣ = b 的方程，两边取对数：

x = ln b / ln a

技巧： 对于 e²ˣ + 3eˣ - 4 = 0，令 t = eˣ，转化为二次方程 t² + 3t - 4 = 0，解得 t = 1 或 t = -4（舍去），故 eˣ = 1，x = 0。

对数方程的解法

对于对数方程，利用对数运算法则合并，然后去掉对数：

例：ln(2x + 1) = ln(x - 1) + ln 3

→ ln(2x + 1) = ln[3(x - 1)]

→ 2x + 1 = 3x - 3 → x = 4

注意： 解必须满足真数 > 0 的条件。

9.4 指数增长与衰减模型

指数增长模型

N = N₀ eᵏᵗ（k > 0）

其中 N₀ 为初始值，k 为增长率，t 为时间。

应用： 人口增长、细菌繁殖、连续复利。

指数衰减模型

N = N₀ e⁻ᵏᵗ（k > 0）

应用： 放射性衰变、药物代谢、温度冷却（牛顿冷却定律）。

半衰期： N = N₀/2 时，t = ln 2 / k。

线性化：将指数关系转化为线性关系 (P2/P3)

y = abˣ → 两边取 ln：ln y = ln a + x ln b，作 ln y 对 x 的图，斜率为 ln b，截距为 ln a。

y = kxⁿ → 两边取 ln：ln y = ln k + n ln x，作 ln y 对 ln x 的图，斜率为 n，截距为 ln k。

ln y = ln a + x ln b ln y = ln k + n ln x

📝 章节练习题

1若 e²ˣ = 5，则 x = ____。
✅ 正确答案：B。e²ˣ = 5 → 2x = ln 5 → x = (1/2) ln 5。
2ln e³ = ____。
✅ 正确答案：C。ln e³ = 3 ln e = 3 × 1 = 3。
3求解方程 ln(x + 2) + ln(x - 2) = 0，x = ____。
✅ 正确答案：C。ln[(x+2)(x-2)] = 0 → x² - 4 = 1 → x² = 5 → x = √5（x > 2，舍去负根）。
4某细菌数量 N = 100 e⁰·⁵ᵗ，t 小时后数量为 400，则 t = ____。
✅ 正确答案：C。100 e⁰·⁵ᵗ = 400 → e⁰·⁵ᵗ = 4 → 0.5t = ln 4 = 2 ln 2 → t = 4 ln 2。
5log₂ 8 + log₃ 27 = ____。
✅ 正确答案：B。log₂ 8 = 3（因为 2³ = 8），log₃ 27 = 3（因为 3³ = 27），3 + 3 = 6。
6d/dx (e³ˣ) = ____。
✅ 正确答案：B。利用链式法则：d/dx (e³ˣ) = e³ˣ · 3 = 3e³ˣ。
