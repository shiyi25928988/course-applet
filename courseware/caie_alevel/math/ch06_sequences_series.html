第6章 数列与级数 (Sequences & Series)

6.1 等差数列 (Arithmetic Progression)

定义与通项公式

等差数列中，相邻两项的差为常数 d（公差）。

aₙ = a + (n - 1)d

其中 a 为首项，d 为公差，n 为项数。

前 n 项和公式

Sₙ = n/2 × (a + l) = n/2 × [2a + (n - 1)d]

其中 l = aₙ 是末项。第一个公式利用首末项平均，第二个利用首项和公差。

6.2 等比数列 (Geometric Progression)

定义与通项公式

等比数列中，相邻两项的比为常数 r（公比）。

aₙ = a × rⁿ⁻¹

其中 a 为首项，r 为公比。

前 n 项和公式

Sₙ = a(1 - rⁿ) / (1 - r)，当 r ≠ 1 Sₙ = na，当 r = 1

6.3 二项式展开 (Binomial Expansion)

二项式定理 (P1)

(a + b)ⁿ = Σ(k=0→n) C(n,k) × aⁿ⁻ᵏ × bᵏ

其中 C(n,k) = n! / [k!(n-k)!] 是二项式系数。

广义二项式展开 (P2/P3)

当 n 为有理数（非正整数）时，展开为无穷级数：

(1 + x)ⁿ = 1 + nx + n(n-1)/2! x² + n(n-1)(n-2)/3! x³ + ...

收敛条件：|x| < 1

求特定项的技巧

Caie 考试中常要求求展开式中 xᵏ 的系数。通项公式：

T_{k+1} = C(n,k) × aⁿ⁻ᵏ × bᵏ

6.4 无穷等比级数

收敛条件与求和

当 |r| < 1 时，无穷等比级数收敛：

S∞ = a / (1 - r)，当 |r| < 1

当 |r| ≥ 1 时，级数发散（除 r = 1 且 a = 0 外）。

📝 章节练习题

1等差数列首项 3，公差 4，第 10 项为____。
✅ 正确答案：B。a₁₀ = 3 + (10-1)×4 = 3 + 36 = 39。
2等差数列 2, 5, 8, ..., 前 20 项和为____。
✅ 正确答案：C。a=2, d=3, n=20。S₂₀ = 20/2 × [2×2 + 19×3] = 10 × (4+57) = 610。
3等比数列首项 2，公比 3，第 5 项为____。
✅ 正确答案：C。a₅ = 2 × 3⁴ = 2 × 81 = 162。
4(1 + 2x)⁵ 展开式中 x³ 的系数为____。
✅ 正确答案：C。T₄ = C(5,3) × 1² × (2x)³ = 10 × 8x³ = 80x³，系数为 80。
5无穷等比级数 1 + 1/2 + 1/4 + 1/8 + ... 的和为____。
✅ 正确答案：C。a=1, r=1/2, |r|<1。S∞ = 1/(1-1/2) = 2。
6等比数列前 3 项和为 14，首项为 2，公比为____。
✅ 正确答案：C。S₃ = 2(1-r³)/(1-r) = 14 → 2(1+r+r²) = 14 → r²+r+1 = 7 → r²+r-6 = 0 → (r+3)(r-2) = 0 → r = 2 或 r = -3。
