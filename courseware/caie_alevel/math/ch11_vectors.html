第11章 三维向量 (Vectors in 3D)
🎯学习目标
  • 掌握三维向量的表示方法：位置向量和单位向量
  • 掌握向量运算：加减、数乘、模长计算
  • 理解点积 (Dot Product) 的定义和几何意义
  • 掌握向量夹角公式及其应用
  • 能够写出三维空间中直线的向量方程和参数方程 (P3)
  • 理解平面方程及其与直线的关系 (P3)

11.1 三维向量的表示

向量的坐标表示

在三维空间中，一个向量可以用三个分量表示：

a = (a₁, a₂, a₃) = a₁ i + a₂ j + a₃ k

其中 i = (1, 0, 0)，j = (0, 1, 0)，k = (0, 0, 1) 是沿三个坐标轴方向的单位向量。

位置向量 (Position Vector)

点 P(x, y, z) 相对于原点 O(0, 0, 0) 的位置向量为：

OP = (x, y, z) = x i + y j + z k

两点 A(x₁, y₁, z₁) 和 B(x₂, y₂, z₂) 之间的向量：

AB = OB - OA = (x₂-x₁, y₂-y₁, z₂-z₁)

单位向量 (Unit Vector)

与向量 a 方向相同的单位向量为：

â = a / |a|

单位向量的模长为 1，即 |â| = 1。

11.2 向量运算

向量加法与减法

a + b = (a₁+b₁, a₂+b₂, a₃+b₃) a - b = (a₁-b₁, a₂-b₂, a₃-b₃)

向量加法满足平行四边形法则和三角形法则。

数乘 (Scalar Multiplication)

λa = (λa₁, λa₂, λa₃)

若 λ > 0，方向不变；若 λ < 0，方向相反。|λa| = |λ|·|a|。

向量的模长 (Magnitude)

|a| = √(a₁² + a₂² + a₃²)

两点 A 和 B 之间的距离：

|AB| = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]

11.3 点积 (Dot Product)

点积的定义

代数定义：

a · b = a₁b₁ + a₂b₂ + a₃b₃

几何定义：

a · b = |a| |b| cos θ

其中 θ 是两个向量之间的夹角（0 ≤ θ ≤ π）。

夹角公式

cos θ = (a · b) / (|a| |b|)

由此可推导：

垂直（正交）： a · b = 0（θ = 90°）

平行： a · b = ±|a||b|（θ = 0° 或 180°）

锐角： a · b > 0

钝角： a · b < 0

点积的性质

a · a = |a|² a · b = b · a（交换律） a · (b + c) = a · b + a · c（分配律） i · i = j · j = k · k = 1 i · j = j · k = k · i = 0

11.4 直线方程 (P3)

直线的向量方程

过点 A（位置向量为 a）且方向向量为 d 的直线：

r = a + λd，λ ∈ R

其中 r 是直线上任意点的位置向量，λ 是参数。

直线的参数方程

若 a = (x₀, y₀, z₀)，d = (l, m, n)，则：

x = x₀ + λl y = y₀ + λm z = z₀ + λn

直线的对称式方程 (Cartesian Form)

消去参数 λ，得到：

(x - x₀)/l = (y - y₀)/m = (z - z₀)/n (= λ)

若某个方向分量为 0（如 l = 0），则相应的坐标恒为常数（x = x₀）。

两直线的夹角

两条直线的夹角等于它们方向向量之间的夹角：

cos θ = |d₁ · d₂| / (|d₁| |d₂|)

取绝对值是因为直线夹角通常取锐角（0 ≤ θ ≤ 90°）。

11.5 平面方程 (P3)

平面的法向量

平面的方向由法向量 n 确定，法向量垂直于平面内的所有向量。

若平面过点 A（位置向量为 a），法向量为 n，则对平面上任意点 r：

(r - a) · n = 0

平面方程的几种形式

向量形式： r · n = a · n = d（常数）

标量形式： 若 n = (A, B, C)，则：

Ax + By + Cz = D

其中 D = n · a。

直线与平面的交点

将直线的参数方程 r = a + λd 代入平面方程 r · n = d：

(a + λd) · n = D → λ = (D - a · n) / (d · n)

若 d · n = 0，则直线平行于平面（无交点或直线在平面内）。

点到平面的距离

点 P（位置向量为 p）到平面 r · n = D 的距离：

d = |p · n - D| / |n|

📝 章节练习题

1向量 a = (3, 4, 0)，则 |a| = ____。
✅ 正确答案：C。|a| = √(3² + 4² + 0²) = √(9 + 16) = 5。
2a = (1, 2, 2)，b = (4, 0, -3)，则 a · b = ____。
✅ 正确答案：B。a · b = 1×4 + 2×0 + 2×(-3) = 4 + 0 - 6 = -2。
3a = (1, 0, -1)，b = (2, 1, 2)，则 a 与 b 的夹角为____。
✅ 正确答案：D。a · b = 1×2 + 0×1 + (-1)×2 = 2 + 0 - 2 = 0，故 a ⟂ b，夹角 90°。
4过点 A(1, 2, 3) 且方向向量为 d = (2, -1, 4) 的直线参数方程为____。
✅ 正确答案：B。r = a + λd = (1, 2, 3) + λ(2, -1, 4) → x = 1+2λ, y = 2-λ, z = 3+4λ。
5平面 2x - y + 3z = 6 的法向量为____。
✅ 正确答案：C。平面 Ax + By + Cz = D 的法向量 n = (A, B, C) = (2, -1, 3)。
6点 P(1, 2, 2) 到平面 x + 2y + 2z = 6 的距离为____。
✅ 正确答案：A。n = (1, 2, 2)，|n| = √(1+4+4) = 3。p·n = 1×1 + 2×2 + 2×2 = 9。d = |9 - 6| / 3 = 1。
