5.1 基本三角恒等式

核心恒等式

tan θ = sin θ / cos θ sin² θ + cos² θ = 1

由 sin²θ + cos²θ = 1 可推导出：

1 + tan² θ = sec² θ 1 + cot² θ = csc² θ

ASTC 象限规则

记忆口诀：All Students Take Calculus（逆时针四个象限）

象限I (0°-90°)II (90°-180°)III (180°-270°)IV (270°-360°)
正值的函数All 全部Sin 正弦Tan 正切Cos 余弦

5.2 三角方程

解三角方程的一般步骤

  1. 将方程化为单一三角函数的形式（如 sin θ = k）
  2. 求基本解（使用 sin⁻¹、cos⁻¹、tan⁻¹）
  3. 根据 ASTC 规则找出所有象限解
  4. 利用周期性给出通解（sin 和 cos 周期为 2π，tan 周期为 π）

通解公式

sin θ = k → θ = nπ + (-1)ⁿ α（n ∈ Z） cos θ = k → θ = 2nπ ± α（n ∈ Z） tan θ = k → θ = nπ + α（n ∈ Z）

其中 α 是基本解（主值）。

5.3 三角函数的图像

sin x

定义域：R；值域：[-1, 1]

周期：2π；奇函数

过原点 (0, 0)

极大值 1 在 x = π/2 + 2kπ

cos x

定义域：R；值域：[-1, 1]

周期：2π；偶函数

在 x = 0 处取值 1

极大值 1 在 x = 2kπ

tan x

定义域：x ≠ π/2 + kπ

值域：R；周期：π

奇函数，有垂直渐近线

在 x = π/2 + kπ 处无定义

图像变换

y = a sin(bx + c) + d：

|a| = 振幅 (amplitude)，周期 = 2π/|b|，c 影响相位位移，d 影响垂直位移。

5.4 和差公式与倍角公式

和差公式 (P2/P3)

sin(A ± B) = sin A cos B ± cos A sin B cos(A ± B) = cos A cos B ∓ sin A sin B tan(A ± B) = (tan A ± tan B) / (1 ∓ tan A tan B)

倍角公式

sin 2θ = 2 sin θ cos θ cos 2θ = cos² θ - sin² θ = 2cos² θ - 1 = 1 - 2sin² θ tan 2θ = 2 tan θ / (1 - tan² θ)

和差化积与积化和差 (P3)

sin A + sin B = 2 sin((A+B)/2) cos((A-B)/2) sin A - sin B = 2 cos((A+B)/2) sin((A-B)/2) cos A + cos B = 2 cos((A+B)/2) cos((A-B)/2) cos A - cos B = -2 sin((A+B)/2) sin((A-B)/2)

📝 章节练习题

1sin² 30° + cos² 30° = ____。
✅ 正确答案：B。根据基本恒等式 sin²θ + cos²θ = 1，对任意 θ 都成立。
2在 0 ≤ x ≤ 2π 范围内，sin x = 1/2 的解为____。
✅ 正确答案：A。sin 在第一、二象限为正，sin(π/6)=1/2，sin(5π/6)=sin(π-π/6)=1/2。
3函数 y = 3 sin(2x) 的周期是____。
✅ 正确答案：B。周期 = 2π/|b| = 2π/2 = π。
4cos 2θ 用 sin θ 表示为____。
✅ 正确答案：A。cos 2θ = 1 - 2sin² θ（由 cos 2θ = cos²θ - sin²θ = (1-sin²θ) - sin²θ = 1 - 2sin²θ）。
5sin(A + B) = ____。
✅ 正确答案：A。sin(A+B) = sin A cos B + cos A sin B。
6在 0 ≤ x ≤ 2π 内，方程 cos 2x = cos x 的解的个数为____。
✅ 正确答案：C。cos 2x - cos x = 0 → 2cos²x - 1 - cos x = 0 → 2cos²x - cos x - 1 = 0 → (2cos x + 1)(cos x - 1) = 0 → cos x = -1/2 或 cos x = 1。cos x = -1/2 → x = 2π/3, 4π/3；cos x = 1 → x = 0, 2π。共 4 个解。
