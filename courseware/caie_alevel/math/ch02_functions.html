第2章 函数与图像变换 (Functions)
🎯学习目标
  • 理解函数的定义、定义域 (domain) 和值域 (range)
  • 掌握一对一函数 (one-to-one) 和多对一函数 (many-to-one) 的区别
  • 熟练计算复合函数 fg(x) = f(g(x))
  • 理解反函数的概念并能够求反函数
  • 掌握函数图像的四种基本变换：平移、伸缩、对称

2.1 函数的基本概念

函数的定义

函数是一种映射关系，将定义域中的每一个输入值 x 映射到唯一的输出值 y = f(x)。

定义域 (Domain)：所有可接受的输入值 x 的集合。

值域 (Range)：所有可能的输出值 f(x) 的集合。

f: x → f(x)，x ∈ Domain

1-1 一对一函数

定义域中不同的 x 映射到不同的 f(x)。

水平线检验：任何水平线与函数图像最多有一个交点。

示例：f(x) = 2x + 1, f(x) = eˣ

只有一对一函数才有反函数。

多对一函数

定义域中不同的 x 可能映射到相同的 f(x)。

水平线检验：存在水平线与图像有多个交点。

示例：f(x) = x², f(x) = sin x

需要限制定义域才能定义反函数。

常见定义域限制

  • 分母 ≠ 0（如 f(x) = 1/x，定义域为 x ≠ 0）
  • 平方根内 ≥ 0（如 f(x) = √x，定义域为 x ≥ 0）
  • 对数真数 > 0（如 f(x) = ln x，定义域为 x > 0）

2.2 复合函数 (Composite Functions)

复合函数的定义

复合函数 fg(x) 表示先应用 g，再将结果输入 f：

fg(x) = f(g(x))

注意：fg(x) ≠ gf(x)（一般情况），顺序很重要！

fg(x) 的定义域：g(x) 的值域中属于 f 定义域的那部分 x 值。

例子：复合函数计算

设 f(x) = 2x + 1，g(x) = x² - 3

fg(x) = f(g(x)) = f(x² - 3) = 2(x² - 3) + 1 = 2x² - 5

gf(x) = g(f(x)) = g(2x + 1) = (2x + 1)² - 3 = 4x² + 4x - 2

可见 fg(x) ≠ gf(x)，顺序不同结果不同。

2.3 反函数 (Inverse Functions)

反函数的概念

反函数 f⁻¹(x) 将原函数的输出"反向映射"回输入。

关系：f(f⁻¹(x)) = x 且 f⁻¹(f(x)) = x。

f⁻¹(x) 的图像是 f(x) 关于直线 y = x 的对称。

f(a) = b ⇔ f⁻¹(b) = a

求反函数的步骤

  1. 将 y = f(x) 写成 y = ...
  2. 交换 x 和 y：x = f(y)
  3. 解出 y（用 x 表示）
  4. 写成 f⁻¹(x) = ...

f⁻¹(x) 的定义域 = f(x) 的值域；f⁻¹(x) 的值域 = f(x) 的定义域。

例子：求反函数

求 f(x) = (2x - 3)/(x + 1) 的反函数，x ≠ -1

y = (2x - 3)/(x + 1) → y(x + 1) = 2x - 3 → yx + y = 2x - 3

yx - 2x = -3 - y → x(y - 2) = -(y + 3) → x = (y + 3)/(2 - y)

f⁻¹(x) = (x + 3)/(2 - x)，x ≠ 2

2.4 图像变换 (Transformations)

四种基本变换

设原函数为 y = f(x)，则：

↕️ 平移 (Translation)

y = f(x) + a：向上平移 a 单位

y = f(x) - a：向下平移 a 单位

y = f(x - a)：向右平移 a 单位

y = f(x + a)：向左平移 a 单位

↔️ 伸缩 (Stretch)

y = a f(x)：垂直方向拉伸 a 倍（a>1）或压缩（0

y = f(ax)：水平方向压缩 a 倍（a>1）或拉伸（0

🪞 对称 (Reflection)

y = -f(x)：关于 x 轴反射

y = f(-x)：关于 y 轴反射

📝 章节练习题

1设 f(x) = 3x - 2，g(x) = x² + 1，则 fg(2) = ____。
✅ 正确答案：A。g(2) = 2² + 1 = 5，fg(2) = f(5) = 3×5 - 2 = 13。
2函数 f(x) = √(x - 3) 的定义域是____。
✅ 正确答案：B。平方根内必须 ≥ 0，x - 3 ≥ 0 → x ≥ 3。
3函数 f(x) = (x + 1)/(x - 2) 的反函数为____。
✅ 正确答案：A。y = (x+1)/(x-2) → y(x-2) = x+1 → yx - 2y = x + 1 → yx - x = 2y + 1 → x(y-1) = 2y+1 → x = (2y+1)/(y-1)，f⁻¹(x) = (2x+1)/(x-1)。
4将 y = f(x) 的图像向右平移 3 个单位，再向上平移 2 个单位，得到的函数是____。
✅ 正确答案：A。向右平移是 x → x - 3，向上平移是 + 2。
5f(x) = x² - 2x + 3，x ≤ 1。f⁻¹(x) 的值域是____。
✅ 正确答案：B。f⁻¹(x) 的值域 = f(x) 的定义域 = x ≤ 1。
6设 f(x) = 2x + 1，g(x) = 3 - x，则 gf(x) = ____。
✅ 正确答案：A。gf(x) = g(2x+1) = 3 - (2x+1) = 2 - 2x。
