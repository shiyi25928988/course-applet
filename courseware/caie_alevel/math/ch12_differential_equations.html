第12章 微分方程 (Differential Equations)
🎯学习目标
  • 理解微分方程的基本概念：阶、通解、特解、初始条件
  • 掌握分离变量法 (Separation of Variables) 解一阶微分方程
  • 掌握一阶线性微分方程的积分因子法
  • 能够将实际问题（增长率、衰减、混合问题）建模为微分方程并求解
  • 理解通解中任意常数的确定方法

12.1 微分方程的概念：阶、通解、特解

什么是微分方程

微分方程是包含未知函数及其导数的方程。例如，dy/dx = ky 表示 y 的变化率与 y 本身成正比。

dy/dx = f(x, y)

微分方程的阶 (order) 是方程中出现的最高阶导数的阶数。

📘 一阶微分方程

只包含一阶导数 dy/dx。

通解包含一个任意常数。

dy/dx = 2x → y = x² + C

📙 二阶微分方程

包含二阶导数 d²y/dx²。

通解包含两个任意常数。

d²y/dx² = 6x → y = x³ + C₁x + C₂

通解与特解

通解 (General Solution)：包含任意常数的解，表示一族曲线。

特解 (Particular Solution)：利用初始条件确定常数后得到的唯一解。

例如，dy/dx = 2x 的通解为 y = x² + C。若已知 y(0) = 3，则特解为 y = x² + 3。

初始条件：x = x₀ 时 y = y₀ → 确定 C

12.2 分离变量法 (Separation of Variables)

分离变量法的步骤

当微分方程可以写成 dy/dx = g(x) · h(y) 的形式时，可使用分离变量法：

  1. 将包含 y 的项移到一边，包含 x 的项移到另一边
  2. 两边同时积分：∫ (1/h(y)) dy = ∫ g(x) dx
  3. 求出积分，加上积分常数 C
  4. 利用初始条件确定 C，得到特解
(1/h(y)) dy = g(x) dx → ∫ (1/h(y)) dy = ∫ g(x) dx

典型例题

例1：解 dy/dx = 2xy，满足 y(0) = 5。

分离变量：dy/y = 2x dx → ∫ dy/y = ∫ 2x dx → ln|y| = x² + C

y = e^(x² + C) = Ae^(x²)，其中 A = e^C。

代入 y(0) = 5：A = 5，故特解为 y = 5e^(x²)。

y = 5e^(x²)

常见可分离类型

dy/dx = ky → y = Ae^(kx)（指数增长/衰减） dy/dx = k(y - a) → y = a + Ae^(kx) dy/dx = f(x) / g(y) → 分离为 g(y)dy = f(x)dx

12.3 一阶线性微分方程

标准形式与积分因子

一阶线性微分方程的标准形式为：

dy/dx + P(x)y = Q(x)

积分因子 (Integrating Factor) 为：

μ(x) = e^(∫ P(x) dx)

将方程两边乘以 μ(x)，左边变为 d/dx[μ(x)y]，然后积分即可求解。

d/dx[μ(x)y] = μ(x)Q(x) → μ(x)y = ∫ μ(x)Q(x) dx

求解步骤

  1. 将方程化为标准形式 dy/dx + P(x)y = Q(x)
  2. 计算积分因子 μ(x) = e^(∫ P(x) dx)
  3. 方程两边乘以 μ(x)，左边识别为 d/dx[μ(x)y]
  4. 两边积分：μ(x)y = ∫ μ(x)Q(x) dx + C
  5. 解出 y，代入初始条件求特解

典型例题

例2：解 dy/dx + 2y = e^x，满足 y(0) = 1。

P(x) = 2，μ(x) = e^(∫ 2dx) = e^(2x)。

乘以积分因子：e^(2x) dy/dx + 2e^(2x) y = e^(3x)

即 d/dx[e^(2x) · y] = e^(3x)

积分：e^(2x) y = (1/3)e^(3x) + C → y = (1/3)e^x + Ce^(-2x)

代入 y(0) = 1：1 = 1/3 + C → C = 2/3

y = (1/3)e^x + (2/3)e^(-2x)

12.4 微分方程的应用：增长率、衰减、混合问题

📈 指数增长模型

人口、细菌繁殖等自然增长过程。

dP/dt = kP → P = P₀e^(kt)

k > 0 表示增长，k < 0 表示衰减。

🌡️ 牛顿冷却定律

物体温度变化率与温差成正比。

dT/dt = -k(T - T₀)

T₀ 为环境温度，T 为物体温度。

🧪 混合问题

盐溶液流入流出容器的问题。

dQ/dt = 流入速率 - 流出速率

Q(t) 为容器中溶质的量。

应用建模步骤

  1. 识别变化率与当前量之间的关系，建立微分方程
  2. 确定初始条件
  3. 求解微分方程（通常用分离变量法或积分因子法）
  4. 代入初始条件得到特解
  5. 解释结果并回答实际问题

放射性衰变与半衰期

放射性物质的衰变遵循 dN/dt = -λN，其中 λ 为衰变常数。

N = N₀e^(-λt)

半衰期 T₁/₂：N = N₀/2 时，t = (ln 2)/λ。

T₁/₂ = ln 2 / λ

📊 微分方程解曲线可视化

拖动滑块观察 k（增长率）和初始值 y₀ 对解曲线 y = y₀e^(kx) 的影响

📝 章节练习题

1微分方程 dy/dx = 3y 的通解是____。
✅ 正确答案：B。分离变量：dy/y = 3dx → ln|y| = 3x + C → y = e^(3x+C) = Ce^(3x)。
2解微分方程 dy/dx = y cos x，满足 y(0) = 2，则 y(π/2) = ____。
✅ 正确答案：A。dy/y = cos x dx → ln|y| = sin x + C → y = Ae^(sin x)，y(0)=2→A=2，y(π/2) = 2e^(1) = 2e。
3微分方程 dy/dx + y = e^(-x) 的通解是____。
✅ 正确答案：B。积分因子 μ = e^(∫1dx) = e^x，乘入：e^x·dy/dx + e^x·y = 1 → d/dx(e^x·y)=1 → e^x·y = x + C → y = (x + C)e^(-x)。
4某细菌种群的增长速率与当前数量成正比。若初始数量为 100，2 小时后数量为 400，则 4 小时后的数量为____。
✅ 正确答案：C。P = 100e^(kt)，400 = 100e^(2k) → e^(2k) = 4 → k = (ln 4)/2。t=4时，P = 100e^(4k) = 100(e^(2k))² = 100×16 = 1600。
5放射性物质的衰变满足 dN/dt = -λN。若半衰期为 5 天，则 λ = ____。
✅ 正确答案：B。N = N₀e^(-λt)，半衰期时 N₀/2 = N₀e^(-λ×5) → e^(-5λ) = 1/2 → 5λ = ln 2 → λ = (ln 2)/5。
6一杯热咖啡放在 20°C 的房间中，冷却速率满足 dT/dt = -k(T - 20)。若初始温度为 80°C，10 分钟后为 60°C，则 k = ____。
✅ 正确答案：A。T - 20 = Ce^(-kt)，T(0)=80→C=60，T(10)=60→40=60e^(-10k)→e^(-10k)=2/3→k=(1/10)ln(3/2)。
