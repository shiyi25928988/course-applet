第8章 积分 (Integration)
🎯学习目标
  • 理解不定积分是微分的逆运算
  • 掌握基本积分公式
  • 能够计算定积分并求曲线下的面积
  • 掌握换元积分法和分部积分法
  • 能够计算旋转体体积

8.1 不定积分 (Indefinite Integration)

积分是微分的逆运算

若 F'(x) = f(x)，则 ∫ f(x) dx = F(x) + C，其中 C 为积分常数。

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C，n ≠ -1 ∫ 1/x dx = ln|x| + C ∫ eˣ dx = eˣ + C ∫ sin x dx = -cos x + C ∫ cos x dx = sin x + C ∫ sec² x dx = tan x + C

8.2 定积分 (Definite Integration)

牛顿-莱布尼茨公式

∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) - F(a)

定积分表示曲线 y = f(x) 与 x 轴在区间 [a, b] 上的净面积（x 轴上方面积为正，下方为负）。

8.3 面积与旋转体体积

曲线下的面积

曲线 y = f(x) 与 x 轴在 x = a 到 x = b 之间的面积：

A = ∫ₐᵇ |f(x)| dx

两条曲线之间的面积：

A = ∫ₐᵇ |f(x) - g(x)| dx

旋转体体积 (P2/P3)

曲线绕 x 轴旋转一周形成的旋转体体积：

V = π ∫ₐᵇ [f(x)]² dx

曲线绕 y 轴旋转：

V = π ∫ x² dy

📊 定积分 — 曲线下的面积

绿色区域 = ∫₀² (x²) dx = 8/3 ≈ 2.67

8.4 换元积分与分部积分 (P2/P3)

换元积分法 (Substitution)

∫ f(g(x)) g'(x) dx = ∫ f(u) du，其中 u = g(x)

例：∫ 2x cos(x²) dx，令 u = x²，du = 2x dx → ∫ cos u du = sin u + C = sin(x²) + C。

分部积分法 (Integration by Parts)

∫ u dv = uv - ∫ v du

例：∫ x eˣ dx，令 u = x, dv = eˣ dx → du = dx, v = eˣ → x eˣ - ∫ eˣ dx = x eˣ - eˣ + C = eˣ(x - 1) + C。

📝 章节练习题

1∫ x³ dx = ____。
✅ 正确答案：A。∫ xⁿ dx = xⁿ⁺¹/(n+1) + C，∫ x³ dx = x⁴/4 + C。
2∫₀¹ (2x + 1) dx = ____。
✅ 正确答案：B。∫₀¹ (2x+1)dx = [x² + x]₀¹ = (1+1) - 0 = 2。
3曲线 y = x² 与 x 轴在 x = 0 到 x = 2 之间的面积为____。
✅ 正确答案：C。∫₀² x² dx = [x³/3]₀² = 8/3。
4∫ cos x dx = ____。
✅ 正确答案：A。∫ cos x dx = sin x + C。
5曲线 y = √x 绕 x 轴从 x = 0 到 x = 1 旋转，旋转体体积为____。
✅ 正确答案：B。V = π ∫₀¹ (√x)² dx = π ∫₀¹ x dx = π [x²/2]₀¹ = π/2。
6∫ x eˣ dx = ____。
✅ 正确答案：A。分部积分：u=x, dv=eˣdx → du=dx, v=eˣ → x eˣ - ∫ eˣ dx = x eˣ - eˣ + C = eˣ(x-1) + C。
