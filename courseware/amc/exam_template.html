1 代数运算 （6分）

若 x + 1/x = 5，求 x³ + 1/x³ 的值。

答案：110
解析：
已知 x + 1/x = 5
两边平方：x² + 2 + 1/x² = 25 ⇒ x² + 1/x² = 23
x³ + 1/x³ = (x + 1/x)(x² - 1 + 1/x²) = 5×(23 - 1) = 110
2 几何计算 （6分）

直角三角形的两条直角边分别为 6 和 8，求斜边上的高。

答案：4.8
解析：
斜边 c = √(6² + 8²) = √100 = 10
面积：½×6×8 = ½×10×h ⇒ 24 = 5h ⇒ h = 4.8
