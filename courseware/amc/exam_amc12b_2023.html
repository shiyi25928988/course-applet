📐 问题求解
1 代数运算 （6分）

若 x + 1/x = 5，求 x³ + 1/x³ 的值。

答案：110
解析：
已知 x + 1/x = 5
两边平方：x² + 2 + 1/x² = 25 ⇒ x² + 1/x² = 23
x³ + 1/x³ = (x + 1/x)(x² - 1 + 1/x²) = 5×(23 - 1) = 5×22 = 110
2 二次方程 （6分）

若方程 x² + bx + c = 0 的两根为 2 和 -3，求 b + c 的值。

答案：-7
解析：
由韦达定理：
两根和：2 + (-3) = -b ⇒ b = 1
两根积：2×(-3) = c ⇒ c = -6
b + c = 1 + (-6) = -7
3 三角形 （6分）

直角三角形的两条直角边分别为 6 和 8，求斜边上的高。

答案：4.8
解析：
斜边 c = √(6² + 8²) = √100 = 10
面积：½×6×8 = ½×10×h ⇒ 24 = 5h ⇒ h = 4.8
4 圆 （6分）

圆 O 的半径为 5，弦 AB 长为 8，求圆心 O 到弦 AB 的距离。

答案：3
解析：
作 OC⊥AB 于 C，则 AC = 4
在 Rt△OAC 中，OC² + AC² = OA²
OC² + 16 = 25 ⇒ OC² = 9 ⇒ OC = 3
5 排列组合 （6分）

从 5 个男生和 4 个女生中选出 3 人组成团队，要求至少有 1 名女生，有多少种选法？

答案：74
解析：
总选法 C(9,3) = 84
全是男生的选法 C(5,3) = 10
至少1名女生 = 84 - 10 = 74
6 概率 （6分）

掷两枚骰子，点数之和大于 8 的概率是多少？

答案：5/18
解析：
总情况数 = 6×6 = 36
点数和>8 的情况：(3,6),(4,5),(4,6),(5,4),(5,5),(5,6),(6,3),(6,4),(6,5),(6,6) 共10种
概率 = 10/36 = 5/18
7 因数分解 （6分）

求 360 的所有正因数的个数。

答案：24
解析：
360 = 2³ × 3² × 5¹
因数个数公式：(3+1)(2+1)(1+1) = 4×3×2 = 24
8 整除性 （6分）

若 1234a 能被 9 整除，求数字 a 的值。

答案：8
解析：
被9整除的数，各位数字和能被9整除：
1+2+3+4+a = 10+a 能被9整除 ⇒ 10+a = 18 ⇒ a = 8
