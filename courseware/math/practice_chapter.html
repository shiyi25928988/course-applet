1. -2024的相反数是（　　）
A. -2024
B. 2024
C. -1/2024
D. 1/2024
【答案】B
【解析】
相反数的定义：只有符号不同的两个数互为相反数。
所以 -2024 的相反数是 2024。

【考点】相反数的概念
【方法技巧】求一个数的相反数，只需改变这个数的符号：正数变负，负数变正，0的相反数是0。
2. 下列各数中，最小的数是（　　）
A. -3
B. -2
C. 0
D. 1
【答案】A
【解析】
有理数大小比较法则：
1. 正数 > 0 > 负数
2. 两个负数比较大小，绝对值大的反而小。

|-3| = 3，|-2| = 2，因为 3 > 2，所以 -3 < -2
因此 -3 < -2 < 0 < 1，最小的数是 -3。

【考点】有理数大小比较

13. 计算：(-2)³ + (-3) × [(-4)² - 6]
【答案】-38
【解析】
按照有理数混合运算顺序：先算乘方，再算乘除，最后算加减；有括号先算括号内的。

  (-2)³ + (-3) × [(-4)² - 6]
= -8 + (-3) × [16 - 6]     （先算乘方）
= -8 + (-3) × 10     （再算括号内）
= -8 + (-30)     （再算乘法）
= -38     （最后算加法）

【易错点】注意符号，特别是负数的乘方：(-2)³ = -8，(-4)² = 16。

➗ 第三章 一元一次方程 中档 共12题 | 选择4 填空4 解答4

1. 下列方程中，是一元一次方程的是（　　）
A. x² - 4x = 3
B. 3x - 1 = x/2
C. x + 2y = 1
D. xy - 3 = 5
【答案】B
【解析】
一元一次方程的定义：只含有一个未知数（元），未知数的次数都是1，等号两边都是整式的方程。

A. x² - 4x = 3：未知数最高次数是2，是一元二次方程 ❌
B. 3x - 1 = x/2：只含一个未知数x，次数为1，是一元一次方程 ✅
C. x + 2y = 1：含有两个未知数，是二元一次方程 ❌
D. xy - 3 = 5：含有两个未知数且次数为2，是二元二次方程 ❌

【考点】一元一次方程的定义
5. 若 x = 2 是方程 2x + a = 7 的解，则 a 的值为  
【答案】3
【解析】
方程的解：使方程左右两边相等的未知数的值。

把 x = 2 代入方程 2x + a = 7：
2×2 + a = 7
4 + a = 7
a = 7 - 4
a = 3

【考点】一元一次方程的解
10. 解方程：(2x - 1)/3 = (x + 2)/4 - 1
【答案】x = -2/5
【解析】
解一元一次方程的步骤：去分母 → 去括号 → 移项 → 合并同类项 → 系数化为1

解：
去分母（两边同乘12）：4(2x - 1) = 3(x + 2) - 12
去括号：8x - 4 = 3x + 6 - 12
移项：8x - 3x = 6 - 12 + 4
合并同类项：5x = -2
系数化为1：x = -2/5

【检验】代入原方程验证，左边 = 右边，所以 x = -2/5 是原方程的解。

【考点】解一元一次方程（含分母）
12. 某商店将某种服装按进价提高40%后标价，又以8折（即按标价的80%）优惠卖出，结果每件仍获利15元，这种服装每件的进价是多少元？
【答案】125元
【解析】
设这种服装每件的进价是 x 元。

分析等量关系：
进价 → 标价 → 售价 → 利润 = 售价 - 进价
标价 = 进价 × (1 + 40%) = 1.4x
售价 = 标价 × 80% = 1.4x × 0.8 = 1.12x
利润 = 售价 - 进价 = 15元

列方程：
1.12x - x = 15
0.12x = 15
x = 15 ÷ 0.12
x = 125

答：这种服装每件的进价是125元。

【考点】一元一次方程的应用（销售利润问题）
【常用公式】利润 = 售价 - 进价，利润率 = 利润/进价 × 100%

A. 35°
B. 55°
C. 70°

【解析】
1. 根据对顶角相等：∠AOC = ∠BOD = 70°

【考点】对顶角的性质、角平分线的定义
【关键知识点】对顶角相等是几何证明中最常用的性质之一。
8. 完成下面的证明：

已知：如图，AB∥CD，∠1 = ∠2。
求证：BE∥CF。

证明：∵ AB∥CD（已知）
        ∴ ∠ABC = ∠DCB（        ）
        又∵ ∠1 = ∠2（已知）
        ∴ ∠ABC - ∠1 = ∠DCB - ∠2
        即 ∠EBC = ∠FCB
        ∴ BE∥CF（        ）
【答案】两直线平行，内错角相等；内错角相等，两直线平行
【解析】
平行线的性质与判定的区别：

性质（已知平行→角相等/互补）：
1. 两直线平行，同位角相等
2. 两直线平行，内错角相等
3. 两直线平行，同旁内角互补

判定（角相等/互补→平行）：
1. 同位角相等，两直线平行
2. 内错角相等，两直线平行
3. 同旁内角互补，两直线平行

本题第一步由平行得到角相等，用性质；第二步由角相等得到平行，用判定。

1. 抛物线 y = (x - 2)² + 3 的顶点坐标是（　　）
A. (2, 3)
B. (-2, 3)
C. (2, -3)
D. (-2, -3)
【答案】A
【解析】
二次函数的顶点式：y = a(x - h)² + k，其顶点坐标为 (h, k)。

本题中 y = (x - 2)² + 3 是顶点式，其中：
h = 2，k = 3
所以顶点坐标为 (2, 3)。

【考点】二次函数的顶点式
【扩展】顶点式的对称轴为直线 x = h。
8. 已知二次函数 y = x² - 4x + 3。
(1) 求该函数的顶点坐标和对称轴；
(2) 求该函数与x轴的交点坐标；
(3) 当x取何值时，y随x的增大而增大？
【答案】(1) 顶点(2,-1)，对称轴x=2；(2) (1,0)和(3,0)；(3) x > 2时
【解析】
(1) 求顶点坐标和对称轴：
方法一（配方法）：
y = x² - 4x + 3
  = (x² - 4x + 4) - 4 + 3
  = (x - 2)² - 1

所以顶点坐标为 (2, -1)，对称轴为直线 x = 2。

方法二（公式法）：
对于 y = ax² + bx + c，顶点横坐标 x = -b/(2a)
这里 a = 1, b = -4, c = 3
x = -(-4)/(2×1) = 2，代入得 y = 2² - 4×2 + 3 = -1

(2) 求与x轴交点：
令 y = 0，则 x² - 4x + 3 = 0
(x - 1)(x - 3) = 0
x₁ = 1, x₂ = 3
所以与x轴交点为 (1, 0) 和 (3, 0)。

(3) 增减性：
因为 a = 1 > 0，抛物线开口向上，对称轴为 x = 2，
所以当 x > 2 时，y 随 x 的增大而增大。

【考点】二次函数的基本性质，是中考高频考点。
