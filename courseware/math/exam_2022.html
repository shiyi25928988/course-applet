二、填空题（共12题，每题4分，满分48分）
7 计算：2a · 3a² = ______。
6a³。2×3 × a×a² = 6a³。
8 已知f(x) = 3x - 1，那么f(2) = ______。
5。f(2) = 3×2 - 1 = 5。
9 方程组 { x + y = 5，2x - y = 1 的解是 ______。
x = 2, y = 3。两式相加：3x = 6 → x = 2，代入得y = 3。
10 已知关于x的方程x² - 2x + m = 0有两个不相等的实数根，那么m的取值范围是 ______。
m < 1。Δ = 4 - 4m > 0 → m < 1。
11 从1、2、3、4这四个数中任选一个数，选中的数恰好是偶数的概率是 ______。
1/2。偶数有2、4两个，概率 = 2/4 = 1/2。
12 已知正六边形的边长为2，那么它的外接圆半径是 ______。
2。正六边形的外接圆半径等于边长。
13 为了解某区2000名初中生的身高情况，从中抽取了200名学生测量身高，在这个问题中，样本容量是 ______。
200。样本容量即抽取的样本数。
14 已知点A(1, 2)、B(-3, 4)，那么线段AB的中点坐标是 ______。
(-1, 3)。中点坐标 = ((1-3)/2, (2+4)/2) = (-1, 3)。
15 如图，已知在平行四边形ABCD中，E是AB中点，设向量AB = a，AD = b，那么向量DE = ______（用a、b表示）。
b - a/2。DE = DA + AE = -b + a/2 = a/2 - b（或写成b - a/2的相反向量）。
16 小明从家到学校的路程是1200米，如图是小明上学路程s（米）与时间t（分钟）的函数图像。那么小明在途中停留了 ______ 分钟。
6。从第5分钟到第11分钟，路程不变，停留了6分钟。
17 在Rt△ABC中，∠C = 90°，AC = 3，BC = 4，以C为圆心，r为半径作圆C，使圆C与AB边只有一个交点，则r的取值范围是 ______。
r = 12/5 或 3 < r ≤ 4。相切时r = 12/5，或A在圆内B在圆外时。
18 定义：在平面内，一个图形绕着某一点旋转一定角度α后，如果与自身重合，就称这个图形为旋转对称图形。下列图形中：①正方形；②正五边形；③正六边形；④圆，是旋转对称图形且旋转角为60°的是 ______（填序号）。
③④。正六边形中心角60°，圆旋转任意角度都重合。

22 如图，在菱形ABCD中，E、F分别是边BC、CD上的点，且BE = DF，连接AE、AF。
（1）求证：AE = AF；
（2）如果∠B = 60°，E是BC中点，AB = 4，求△AEF的面积。
（1）证明：菱形ABCD中，AB = AD，∠B = ∠D，BE = DF
∴ △ABE ≅ △ADF（SAS），∴ AE = AF

23 某水果店购进苹果和梨共100千克，已知苹果的进价为每千克8元，售价为每千克12元；梨的进价为每千克6元，售价为每千克9元。设购进苹果x千克，销售完这批水果的总利润为y元。
（1）求y关于x的函数解析式；
（2）如果水果店决定苹果的进货量不超过梨的3倍，那么应如何进货才能使销售完这批水果的利润最大？最大利润是多少？
（1）每千克苹果利润4元，每千克梨利润3元
y = 4x + 3(100 - x) = x + 300
（2）x ≤ 3(100 - x) → x ≤ 75
y随x增大而增大，∴ x = 75时，y最大 = 375元。
24 已知抛物线 y = -x² + bx + c 经过点A(0, 3)和点B(1, 0)。
（1）求抛物线的解析式；
（2）求抛物线的顶点坐标和对称轴。
（1）代入A(0,3)得c = 3，代入B(1,0)得-1 + b + 3 = 0 → b = -2
∴ y = -x² - 2x + 3
（2）y = -(x+1)² + 4，顶点(-1, 4)，对称轴x = -1。
