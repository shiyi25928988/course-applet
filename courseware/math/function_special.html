1一次函数基础

已知一次函数 y = kx + b 的图象经过点 A(1, 3) 和 B(-1, -1)，求该函数的解析式，并求当 x = 3 时 y 的值。

答案：y = 2x + 1；当 x = 3 时，y = 7
解析：
将 A(1, 3) 和 B(-1, -1) 代入解析式：
k + b = 3 ...(1)
-k + b = -1 ...(2)
(1)+(2): 2b = 2 ⇒ b = 1，代入 (1): k = 2
解析式：y = 2x + 1
当 x = 3 时，y = 2×3 + 1 = 7
2反比例函数

反比例函数 y = k/x 的图象经过点 (2, 3)，求 k 的值，并求当 y = 6 时 x 的值。

答案：k = 6；x = 1
解析：
将点 (2, 3) 代入：3 = k/2 ⇒ k = 6
函数：y = 6/x
当 y = 6 时，6 = 6/x ⇒ x = 1
