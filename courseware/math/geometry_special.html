1三角形全等证明

已知：在△ABC 中，AB = AC，AD 是 BC 边上的中线。求证：(1) △ABD ≌ △ACD；(2) AD ⊥ BC。

答案：见解析
证明：
(1) 在△ABD 和 △ACD 中：
AB = AC（已知）
AD = AD（公共边）
BD = CD（AD 是中线）
∴ △ABD ≌ △ACD（SSS）

(2) 由△ABD ≌ △ACD 得 ∠ADB = ∠ADC
又 ∵ ∠ADB + ∠ADC = 180°
∴ ∠ADB = ∠ADC = 90°
∴ AD ⊥ BC
2圆的切线证明

已知 AB 是⊙O 的直径，点 C 在⊙O 上，过点 C 作 CD ⊥ AB 于 D，且 BC 平分∠ACD。求证：CD 是⊙O 的切线。

答案：见解析
证明：
连接 OC
∵ OB = OC，∴ ∠OCB = ∠OBC
∵ BC 平分∠ACD，∴ ∠ACB = ∠BCD
∵ AB 是直径，∴ ∠ACB = 90°
∴ ∠OCD = ∠OCB + ∠BCD = ∠OBC + ∠ACB = 90°
∴ OC ⊥ CD
又 ∵ OC 是半径，∴ CD 是⊙O 的切线
