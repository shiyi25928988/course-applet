一、选择题（本大题共 6 题，每题 4 分，共 24 分）
1函数综合（4分）

已知二次函数 y = ax² + bx + c 的图象开口向上，对称轴在 y 轴右侧，则下列结论正确的是：

A. a > 0, b > 0    B. a > 0, b < 0    C. a < 0, b > 0    D. a < 0, b < 0

答案：B
解析：开口向上 ⇒ a > 0；对称轴 x = -b/(2a) > 0 ⇒ b < 0
压轴题专区
25二次函数综合压轴题（14分）

已知抛物线 y = x² - 2x - 3 与 x 轴交于 A、B 两点（A 在 B 左侧），与 y 轴交于点 C。

(1) 求 A、B、C 三点坐标；（4分）

(2) 求△ABC 的面积；（4分）

(3) 在抛物线上是否存在点 P，使 S△ABP = S△ABC？若存在，求 P 点坐标。（6分）

答案：
(1) A(-1, 0), B(3, 0), C(0, -3)
(2) S△ABC = 6
(3) P(1+√7, 3) 或 P(1-√7, 3)
