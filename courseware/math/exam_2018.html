一、选择题（本大题共6题，每题4分，满分24分）
1 计算√18 - 2√(1/2)的结果是（ ）
✅ C。√18 = 3√2，2√(1/2) = √2，所以3√2 - √2 = 2√2。
2 下列对一元二次方程x² + x - 3 = 0根的情况的判断，正确的是（ ）
✅ A。Δ = 1² - 4×1×(-3) = 1 + 12 = 13 > 0，所以有两个不相等的实数根。
3 下列图形中的一次函数y = -x - 2的图像大致是（ ）
✅ B。k = -1 < 0，b = -2 < 0，直线从左上向右下，与y轴负半轴相交，经过二、三、四象限。
4 某校男生、女生以及教师人数的扇形统计图如图所示，若该校男生、女生以及教师的总人数为1200人，则根据图中信息，可知该校教师共有（ ）
✅ A。教师占比 = 100% - 46% - 45% = 9%。教师人数 = 1200 × 9% = 108人。
5 已知平行四边形ABCD，下列条件中，不能判定这个平行四边形为矩形的是（ ）
✅ B。∠A = ∠C是平行四边形的基本性质，不能判定为矩形。A：∠A+∠B=180°且∠A=∠B→∠A=90°；C：对角线相等的平行四边形是矩形；D：有一个直角→矩形。
6 如图，已知∠POQ = 30°，点A、B在射线OQ上（点A在点O、B之间），半径长为2的⊙A与直线OP相切，半径长为3的⊙B与⊙A相交，那么OB的取值范围是（ ）
✅ A。⊙A与OP相切→OA = 2r = 4（30°对边是半径2，斜边OA = 2×2 = 4）。两圆相交：|3 - 2| < AB < 3 + 2 → 1 < AB < 5。OB = OA + AB = 4 + AB，所以5 < OB < 9。
二、填空题（本大题共12题，每题4分，满分48分）
7 -8的立方根是 ______。
-2。因为(-2)³ = -8。
8 计算：(a + 1)² - a² = ______。
2a + 1。(a+1)² - a² = a² + 2a + 1 - a² = 2a + 1。
9 方程组 { x + y = 1, 2x - y = 5 } 的解是 ______。
{x=2, y=-1}。两式相加：3x = 6 → x = 2，代入得y = -1。
10 某商品原价为a元，如果按原价的八折销售，那么售价是 ______ 元。（用含a的代数式表示）
0.8a。八折即原价的80%。
11 已知反比例函数y = (k - 1)/x（k是常数，k≠1）的图像有一支在第二象限，那么k的取值范围是 ______。
k < 1。反比例函数有一支在第二象限→k - 1 < 0→k < 1。
12 某校学生自主建立了一个学习用品义卖平台，已知九年级200名学生义卖所得金额的频数分布直方图如图所示，那么20-30元这个小组的组频率是 ______。
0.25。20-30元的频数是50人，总人数200，频率 = 50/200 = 0.25。
13 从 2/7，π，√3 这三个数中选一个数，选出的这个数是无理数的概率为 ______。
2/3。三个数中π和√3是无理数，概率 = 2/3。
14 如果一次函数y = kx + 3（k是常数，k≠0）的图像经过点(1, 0)，那么y的值随x的增大而 ______。（填"增大"或"减小"）
减小。代入(1,0)：0 = k + 3→k = -3 < 0，所以y随x增大而减小。
15 如图，已知平行四边形ABCD，E是边BC的中点，联结DE并延长，与AB的延长线交于点F。设向量DA = **a**，向量DC = **b**，那么向量DF = ______（用**a**、**b**表示）。
**DF** = 2**a** + **b**。△DCE≌△FBE，所以BF = DC = b。DF = DA + AF = a + 2b？或DF = 2DA + DC = 2a + b。
16 通过画出多边形的对角线，可以把多边形内角和问题转化为三角形内角和问题。如果从某个多边形的一个顶点出发的对角线共有2条，那么该多边形的内角和是 ______ 度。
540。从一个顶点出发有2条对角线，说明该多边形是5边形。内角和 = (5 - 2)×180° = 540°。
17 如图，在正方形ABCD中，E是边AD的中点。把△ABE沿直线BE翻折，点A落在点F处，联结DF，那么∠EDF的正切值是 ______。
2。设正方形边长为2，AE = ED = 1，EF = 1。由折叠性质，∠AEB = ∠FEB。AE = EF = ED，所以△EDF是等腰三角形。∠EDF = ∠EFD，∠AEF = ∠EDF + ∠EFD = 2∠EDF。又∠AEB = ∠EDF，tan∠AEB = AB/AE = 2。
18 对于一个位置确定的图形，如果它的所有点都在一个水平放置的矩形内部或边上，且该图形与矩形的每条边都至少有一个公共点，那么这个矩形水平方向的边长称为该图形的宽，铅垂方向的边长称为该矩形的高。如图，菱形ABCD的边长为1，边AB水平放置。如果该菱形的高是宽的2/3，那么它的宽的值是 ______。
18/13。设菱形的高为h，宽为w，h = (2/3)w。设菱形对角线水平分量为a，垂直分量为b。由题意：a² + b² = 1，a = w - 1，b = h = (2/3)w。解得：(w - 1)² + (4w²/9) = 1 → w² - 2w + 1 + 4w²/9 = 1 → 13w²/9 - 2w = 0 → w(13w - 18) = 0 → w = 18/13。
