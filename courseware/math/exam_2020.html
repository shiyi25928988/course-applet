1 下列各式中，运算结果为a⁶的是（ ）
✅ D。D：(a²)³ = a⁶；A、B不是同类项不能合并；C：a²·a⁴ = a⁶，但D是更典型的幂的乘方运算。
2 用换元法解方程 2x/(x²-1) + (x²-1)/x = 3 时，设 y = x/(x²-1)，那么原方程可化为关于y的整式方程是（ ）
✅ B。原方程 = 2y + 1/y = 3 → 2y² - 3y + 1 = 0。
3 我们经常用调查方式收集数据，下列调查中适宜用普查方式的是（ ）
✅ B。普查适用于范围小、易操作的调查。A是破坏性测试，C、D范围太大，都适合抽样调查。
4 已知反比例函数 y = k/x 的图像经过点(1, -2)，则k的值为（ ）
✅ D。代入(1, -2)：-2 = k/1 → k = -2。
5 下列命题中，真命题是（ ）
✅ A。菱形对角线互相垂直，加上相等就是正方形。B是矩形，C是任意平行四边形，D是菱形。
6 如图，在矩形ABCD中，E是AB的中点，将△ADE沿DE翻折，点A恰好落在BC上的点F处，那么下列结论错误的是（ ）

二、填空题（共12题，每题4分，满分48分）
7 计算：2a · 3ab = ______。
6a²b。2×3 × a×a × b = 6a²b。
8 已知f(x) = x² - 1，那么f(-3) = ______。
8。f(-3) = (-3)² - 1 = 9 - 1 = 8。
9 不等式组 { x - 1 < 3，x + 1 > 3 的解集是 ______。
2 < x < 4。x < 4 且 x > 2，∴ 2 < x < 4。
10 如果关于x的方程x² - 4x + m = 0有两个相等的实数根，那么m的值是 ______。
4。Δ = 16 - 4m = 0 → m = 4。
11 从1、2、3、4、5这五个数中任选一个数，选中的数恰好是小于3的概率是 ______。
2/5。小于3的数有1、2两个，∴ 概率 = 2/5。
12 已知正方形的边长为4，那么这个正方形的外接圆半径是 ______。
2√2。正方形对角线 = 4√2，外接圆半径 = 对角线/2 = 2√2。
13 为了解某区六年级8400名学生中会游泳的学生人数，随机调查了其中400名学生，结果有150名学生会游泳，那么估计该区会游泳的六年级学生人数为 ______。
3150。8400 × 150/400 = 8400 × 3/8 = 3150。
14 如果从1、2、3、4、5、6、7、8、9、10这10个数中任意选取一个数，那么取到的数恰好是素数的概率是 ______。
2/5。素数有2、3、5、7共4个，4/10 = 2/5。
15 如图，已知在△ABC中，D、E分别是AB、AC的中点，设向量AB = a，AC = b，那么向量DE可以表示为 ______（用a、b表示）。
(b - a)/2。DE是中位线，BC = AC - AB = b - a，∴ DE = 1/2 BC = (b - a)/2。
16 小明从家步行到学校需走的路程为1800米，图中的折线OAB反映了小明从家步行到学校所走的路程s（米）与时间t（分钟）的函数关系。那么小明从家步行到学校的平均速度是 ______ 米/分。
90。平均速度 = 总路程/总时间 = 1800/20 = 90米/分。
17 在矩形ABCD中，AB = 6，BC = 8，点O在对角线AC上，圆O的半径为2，如果圆O与矩形的两边都相切，那么AO的长是 ______。
10/3。由相似三角形，AO/AC = 2/BC → AO = 10/3。
18 在△ABC中，AB = 4，∠C = 60°，∠A > ∠B，则BC的长的取值范围是 ______。
4 < BC ≤ 8√3/3。当∠A = 90°时BC最大，当∠A = ∠B时BC = AB = 4，结合∠A > ∠B可得范围。

20 解不等式组：{ 3x > x + 2，x + 4 < 2(2x - 1)
解第一个不等式：3x > x + 2 → 2x > 2 → x > 1
解第二个不等式：x + 4 < 4x - 2 → 6 < 3x → x > 2
∴ 不等式组的解集是 x > 2
21 如图，在直角坐标系中，O为原点，点A在第一象限，点B在x轴正半轴上，△AOB的面积为12，OA = AB，反比例函数 y = k/x 的图像经过点A。

（2）如果OB = 6，求直线AB的解析式。

∴ 1/2 × 2x × y = xy = 12，而k = xy = 12
（2）OB = 6，∴ x = 3，y = 4，A(3,4)，B(6,0)
直线AB：y = -4/3 x + 8

23 某公司生产某种产品，每件成本价是400元，销售价为510元，今年共销售了m件。为进一步扩大市场，该公司计划在降低销售价的同时降低生产成本。经市场调研，预测下季度这种产品每件销售价降低4%，销售量将提高10%，要使销售利润保持不变，该产品每件的成本价应降低多少元？
设成本降低x元，原利润 = m(510 - 400) = 110m
新售价 = 510×0.96 = 489.6，新成本 = 400 - x，新销量 = 1.1m
新利润 = 1.1m(489.6 - 400 + x) = 110m
解得：x = 10.4元
24 已知抛物线 y = ax² + bx + 5 经过点A(-5, 0)、B(1, 0)。
（1）求抛物线的解析式；
（2）点P是抛物线上一点，且△PAB的面积为10，求点P的坐标。
（1）抛物线过A(-5,0)、B(1,0)，可设y = a(x+5)(x-1)
代入(0,5)：5 = -5a → a = -1
∴ y = -(x+5)(x-1) = -x² - 4x + 5
（2）AB = 6，S = 1/2 × 6 × |y_P| = 10 → |y_P| = 10/3
解方程 -x² - 4x + 5 = ±10/3 得P点坐标。
