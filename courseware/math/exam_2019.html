一、选择题（共6题，每题4分，满分24分）
1 下列计算中，正确的是（ ）
✅ D。D：(-2a³)² = 4a⁶；A：a²与a³不是同类项；B：(-a²)³ = -a⁶；C：a⁶ ÷ a² = a⁴。
2 如果m > n，那么下列结论错误的是（ ）
✅ D。不等式两边乘负数，不等号方向改变。-2m < -2n。
3 下列函数中，函数值y随自变量x的值增大而增大的是（ ）
✅ A。A是正比例函数k=1/3>0，y随x增大而增大。B的k<0；C、D是反比例，需指定象限。
4 甲、乙两名同学本学期五次引体向上的测试成绩如图所示，下列判断正确的是（ ）
✅ A。甲的成绩波动小（7,8,8,8,9），乙波动大（6,7,8,9,10），所以甲更稳定。
5 下列命题中，假命题是（ ）
✅ C。矩形对角线相等且互相平分，但不一定垂直（只有正方形才垂直）。
6 已知⊙A与⊙B外切，⊙C与⊙A、⊙B都内切，且AB = 5，AC = 6，BC = 7，那么⊙C的半径长是（ ）
✅ C。设⊙A半径r₁，⊙B半径r₂，⊙C半径r₃。外切：r₁ + r₂ = 5；内切：r₃ - r₁ = 6，r₃ - r₂ = 7。三式相加：2r₃ = 18 → r₃ = 9。
二、填空题（共12题，每题4分，满分48分）
7 计算：(2a²)² = ______。
4a⁴。幂的乘方：(2a²)² = 2² × (a²)² = 4a⁴。
8 已知f(x) = 6/x，那么f(√3) = ______。
2√3。f(√3) = 6/√3 = 6√3/3 = 2√3。
9 如果关于x的方程x² - 2x + k = 0（k为常数）有两个不相等的实数根，那么k的取值范围是 ______。
k < 1。Δ = b² - 4ac = 4 - 4k > 0 → k < 1。
10 不透明的布袋中装有8个白球和若干个黑球，它们除颜色外完全相同，从中任意摸出一个球是白球的概率是1/3，则黑球有 ______ 个。
16。设黑球x个，8/(8+x) = 1/3 → 24 = 8 + x → x = 16。
11 一枚质地均匀的正方体骰子，六个面上的数字分别是1、2、3、4、5、6，掷两次骰子，第一次朝上一面的数字记为m，第二次朝上一面的数字记为n，则mn为偶数的概率是 ______。
3/4。mn为奇数的情况：m、n都为奇数，概率1/2 × 1/2 = 1/4。∴ 偶数概率1 - 1/4 = 3/4。
12 如图，已知直线l₁∥l₂，含30°角的三角板的直角顶点C在l₁上，30°角的顶点A在l₂上，边BC与l₂交于点D，如果∠1 = 135°，那么∠2 = ______度。
45°。∠1的补角是45°，由平行线和三角形外角性质，∠2 = 45°。
13 某品牌旗舰店平日将某商品按进价提高40%后标价，在某次电商购物节中，为促销该商品，按标价8折销售，售价2240元，则该商品的进价是 ______ 元。
2000。设进价x元：x × 1.4 × 0.8 = 2240 → 1.12x = 2240 → x = 2000。
14 已知∠A = 60°，则tanA = ______。
√3。tan60° = √3。
15 如图，在正多边形ABCDEF中，设向量AB = a，AF = b，那么向量AC可以表示为 ______（用a、b表示）。
2a + b。正六边形中，FC平行且等于2AB，∴ AC = AF + FC = AF + 2AB = 2a + b。
16 在△ABC和△A₁B₁C₁中，已知∠C = ∠C₁ = 90°，AC = A₁C₁ = 3，BC = 4，B₁C₁ = 2，点D、D₁分别在AB、A₁B₁上，且△ACD ≅ △C₁A₁D₁，则AD的长是 ______。
5/3。由相似和全等对应关系，AD = 5/3。
17 如图，在平面直角坐标系xOy中，已知点A(2,3)，将线段OA绕原点O顺时针旋转90°得到线段OA₁，则点A₁的坐标是 ______。
(3, -2)。顺时针旋转90°坐标变换：(x, y) → (y, -x)。
18 在△ABC中，AB = AC = 5，BC = 8，将△ABC绕着点C旋转得到△A₁B₁C，点A、B的对应点分别是A₁、B₁，如果点B₁恰好落在线段BA的延长线上，那么AA₁的长是 ______。
8√5/5。由余弦定理和旋转性质可求得。
三、解答题（共7题，满分78分）
19 计算：|√3 - 1| - 2sin60° + (1/3)⁻¹ + √(-2)²
解：|√3 - 1| = √3 - 1
2sin60° = 2 × √3/2 = √3
(1/3)⁻¹ = 3，√(-2)² = 2
原式 = (√3 - 1) - √3 + 3 + 2 = 4
20 解方程：3/(x-1) - 2/x = 0
解：去分母：3x - 2(x - 1) = 0 → 3x - 2x + 2 = 0 → x = -2
检验：x = -2时，x(x-1) ≠ 0，∴ x = -2是原方程的解。
21 在平面直角坐标系xOy中，已知一次函数图像经过点A(1,0)，B(2,3)。
（1）求这个一次函数的解析式；
（2）如果点P是该一次函数图像上一点，且△POA的面积为3，求点P的坐标。
（1）设y = kx + b，代入A(1,0)、B(2,3)：
k + b = 0，2k + b = 3 → k = 3，b = -3 → y = 3x - 3
（2）S△POA = 1/2 × OA × |y_P| = 1/2 × 1 × |y_P| = 3 → |y_P| = 6
y_P = 6时，x = 3；y_P = -6时，x = -1
∴ P(3, 6) 或 P(-1, -6)
22 如图，在梯形ABCD中，AD∥BC，AB = DC，AC、BD交于点O，AC⊥BC，M是AC中点。
（1）求证：四边形ABMD是平行四边形；
（2）如果AD = 2，BC = 6，求AC的长。
（1）证明：M是AC中点，在等腰梯形中，可证DM∥AB且DM = AB，∴ ABMD是平行四边形。
（2）AC = 4√3。由勾股定理可求得。
23 奉贤区开展"垃圾分类"活动，某中学在预备年级6个班中，每班随机抽取10名学生进行垃圾分类知识测试，将测试成绩整理后绘制成如下统计图：
（1）求这60名学生测试成绩的平均数、中位数和众数；
（2）如果该校预备年级共有300名学生，估计这次测试中成绩达到90分及以上的学生人数。
（1）平均数 = (70×4 + 80×8 + 85×14 + 90×18 + 95×16)/60 = 88
中位数 = 90，众数 = 90
（2）90分及以上人数：18 + 16 = 34人，300 × 34/60 = 170人。
24 已知抛物线y = ax² + bx经过点A(-1, 3)和点B(-4, 0)。
（1）求这条抛物线的解析式；
（2）将这条抛物线沿x轴平移，使其经过点O，求平移后抛物线的表达式。
（1）代入A(-1,3)、B(-4,0)：
a - b = 3，16a - 4b = 0 → 4a - b = 0
解得a = 1，b = 4 → y = x² + 4x
（2）y = x² + 4x = (x+2)² - 4，顶点(-2,-4)
向右平移2个单位：y = x² - 4
或向左平移6个单位：y = (x+8)² - 4
25 如图，在△ABC中，AB = AC，∠BAC = 90°，D是BC边上一点，将线段AD绕点A逆时针旋转90°得到AE，连接BE。
（1）求证：△ABD ≅ △ACE；
（2）若BD = 2，DC = 4，求DE的长。
（1）证明：AB = AC，AD = AE，∠BAD = ∠CAE（都等于90°-∠DAC）
∴ △ABD ≅ △ACE（SAS）
（2）DE = 2√5。由勾股定理可求得。
