5.1 定义

在Rt△ABC中，∠C=90°

sin A = a/c（对边/斜边） cos A = b/c（邻边/斜边） tan A = a/b（对边/邻边） cot A = b/a（邻边/对边）

0 < sin A < 1, 0 < cos A < 1, tan A > 0, cot A > 0

5.2 特殊角三角比

角αsinαcosαtanαcotα
30°1/2√3/2√3/3√3
45°√2/2√2/211
60°√3/21/2√3√3/3

重要关系

sin²α + cos²α = 1

tanα = sinα/cosα

sin(90°-α) = cosα（互余角关系）

5.3 解直角三角形

常见应用

仰角与俯角：视线与水平线的夹角。仰角在上，俯角在下。

坡度：i = h/l = tanα（垂直高度/水平宽度）

方位角：从正北方向顺时针旋转的角度。

解直角三角形：已知一边一角或两边→求其他边和角。
