一、选择题（每题3分，共12分）

1. 3分
下列说法正确的是（    ）

A. 两点之间，直线最短
B. 过一点有且只有一条直线与已知直线平行
C. 对顶角相等
D. 相等的角是对顶角
参考答案与解析
答案：C
【解析】
A. 两点之间，线段最短，不是直线最短，错误
B. 过直线外一点有且只有一条直线与已知直线平行，必须强调"直线外"，错误
C. 对顶角相等，这是对顶角的基本性质，正确
D. 相等的角不一定是对顶角，如等腰三角形的两个底角相等但不是对顶角，错误
评分标准：选择正确得3分，不选或选错得0分。
2. 3分
已知 $\angle A = 35^\circ$，则 $\angle A$ 的余角的度数为（    ）

A. $35^\circ$
B. $55^\circ$
C. $65^\circ$
D. $145^\circ$
参考答案与解析
答案：B
【解析】余角的定义：两个角的和为 $90^\circ$，则这两个角互余。
$\angle A$ 的余角 $= 90^\circ - 35^\circ = 55^\circ$
评分标准：选择正确得3分，不选或选错得0分。
3. 3分
如图，直线 $AB \parallel CD$，$\angle 1 = 60^\circ$，则 $\angle 2$ 的度数为（    ）

A. $30^\circ$
B. $60^\circ$
C. $120^\circ$
D. $150^\circ$
参考答案与解析
答案：C
【解析】因为 $AB \parallel CD$，所以 $\angle 1$ 与 $\angle 2$ 的对顶角是同旁内角，互补。
或者 $\angle 1$ 与 $\angle 2$ 的邻补角相等（同位角），所以 $\angle 2 = 180^\circ - 60^\circ = 120^\circ$
评分标准：选择正确得3分，不选或选错得0分。
4. 3分
下列条件中，不能判定两个三角形全等的是（    ）

A. SSS
B. SAS
C. ASA
D. SSA
参考答案与解析
答案：D
【解析】三角形全等的判定定理有：SSS、SAS、ASA、AAS、HL（直角三角形）。
SSA不能判定三角形全等，因为两边和其中一边的对角对应相等时，可能画出两个不同的三角形。
评分标准：选择正确得3分，不选或选错得0分。

二、填空题（每题3分，共12分）

5. 3分
已知线段 $AB = 8cm$，点C是AB的中点，则 $AC =$______$cm$。
参考答案与解析
答案：4
【解析】因为C是AB的中点，所以 $AC = BC = \frac{1}{2}AB = \frac{1}{2} \times 8 = 4cm$。
评分标准：答案正确得3分。
6. 3分
如果一个角的补角是 $120^\circ$，则这个角的度数是______。
参考答案与解析
答案：$60^\circ$
【解析】补角的定义：两个角的和为 $180^\circ$，则这两个角互补。
这个角 $= 180^\circ - 120^\circ = 60^\circ$
评分标准：答案正确得3分。
7. 3分
在 $\triangle ABC$ 中，$\angle A = 50^\circ$，$\angle B = 60^\circ$，则 $\angle C =$______。
参考答案与解析
答案：$70^\circ$
【解析】三角形内角和为 $180^\circ$。
$\angle C = 180^\circ - \angle A - \angle B = 180^\circ - 50^\circ - 60^\circ = 70^\circ$
评分标准：答案正确得3分。
8. 3分
如图，已知 $AB = AD$，要使 $\triangle ABC \cong \triangle ADC$，还需要添加一个条件是______（只填一个即可）。
参考答案与解析
答案：$BC = DC$ 或 $\angle BAC = \angle DAC$
【解析】已知 $AB = AD$，$AC = AC$（公共边），所以：
- 添加 $BC = DC$，可用SSS判定全等
- 添加 $\angle BAC = \angle DAC$，可用SAS判定全等
任选一个即可。
评分标准：写出一个正确条件得3分。

三、解答题（共26分）

9. 6分
如图，直线AB和CD相交于点O，OE平分 $\angle AOC$，$\angle BOD = 70^\circ$，求 $\angle AOE$ 的度数。
参考答案与解析
答案：$35^\circ$
【解析】
因为直线AB和CD相交于点O，所以 $\angle AOC = \angle BOD = 70^\circ$（对顶角相等）。
又因为OE平分 $\angle AOC$，所以 $\angle AOE = \frac{1}{2}\angle AOC = \frac{1}{2} \times 70^\circ = 35^\circ$。
答：$\angle AOE$ 的度数为 $35^\circ$。

评分标准：写出对顶角相等得2分，写出角平分线定义得2分，计算正确得1分，答句1分，共6分。
10. 8分
如图，已知 $AD \parallel BC$，$\angle B = \angle D$。
求证：$AB \parallel CD$。
参考答案与解析
证明：
$\because AD \parallel BC$（已知）
$\therefore \angle D = \angle DCE$（两直线平行，内错角相等）
又 $\because \angle B = \angle D$（已知）
$\therefore \angle B = \angle DCE$（等量代换）
$\therefore AB \parallel CD$（同位角相等，两直线平行）

（或用同旁内角证明：
$\because AD \parallel BC$
$\therefore \angle A + \angle B = 180^\circ$
又 $\because \angle B = \angle D$
$\therefore \angle A + \angle D = 180^\circ$
$\therefore AB \parallel CD$）

评分标准：每步2分，共8分。
11. 12分
如图，已知点B、E、C、F在同一直线上，$AB = DE$，$AC = DF$，$BE = CF$。

求证：(1) $\triangle ABC \cong \triangle DEF$；（6分）
(2) $AB \parallel DE$。（6分）
参考答案与解析
证明：

(1) $\because BE = CF$（已知）
$\therefore BE + EC = CF + EC$（等式的性质）
即 $BC = EF$

在 $\triangle ABC$ 和 $\triangle DEF$ 中：
$\begin{cases} AB = DE \text{（已知）} \\ AC = DF \text{（已知）} \\ BC = EF \text{（已证）} \end{cases}$
$\therefore \triangle ABC \cong \triangle DEF$（SSS）

(2) $\because \triangle ABC \cong \triangle DEF$（已证）
$\therefore \angle B = \angle DEF$（全等三角形的对应角相等）
$\therefore AB \parallel DE$（同位角相等，两直线平行）

评分标准：(1)6分：证明BC=EF得2分，正确写出全等条件得3分，结论1分；(2)6分：写出对应角相等得3分，写出平行判定得3分。
