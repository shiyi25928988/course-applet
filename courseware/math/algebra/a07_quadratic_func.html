7.1 定义与图像

二次函数图像——抛物线

一般式：y = ax² + bx + c (a ≠ 0)

开口方向：a > 0 → 开口向上；a < 0 → 开口向下。

对称轴：直线 x = -b/(2a)

顶点：(-b/(2a), (4ac-b²)/(4a))

📈 二次函数图像

7.2 三种表达形式

一般式

y = ax²+bx+c

任意三点可确定。

顶点式

y = a(x-h)² + k

顶点 (h, k)。已知顶点+另一点用。

交点式（两根式）

y = a(x-x₁)(x-x₂)

已知与x轴两交点用。

7.3 与方程、不等式的关系

ax²+bx+c=0 的根 = 抛物线与x轴交点的横坐标。

Δ>0→两个交点；Δ=0→一个交点（相切）；Δ<0→无交点。

ax²+bx+c>0→抛物线在x轴上方的部分。

平移规律：y=a(x-h)²+k，|a|越大抛物线越窄。

📝 练习题

1抛物线 y=x²-2x+3 的顶点坐标是____。
✅ B。x=-b/2a=1，y=1-2+3=2→(1,2)。
2抛物线 y=-x²+4x 的对称轴是____。
✅ B。x=-4/(2×(-1))=-4/(-2)=2。
3将抛物线 y=x² 向右平移2个单位，再向上平移3个单位，得到____。
✅ B。y=(x-2)²+3（顶点从原点移到(2,3)）。
4已知二次函数过点(0,-6)，且与x轴交于(-1,0)和(3,0)，解析式为____。
✅ A。设y=a(x+1)(x-3)，代入(0,-6):a·1·(-3)=-6→a=2→y=2(x+1)(x-3)=2x²-4x-6。
