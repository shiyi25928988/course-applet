5.1 解法

①直接开平方法

(x+m)²=n (n≥0) → x=-m±√n

例：(x-3)²=4 → x=3±2 → x₁=5, x₂=1

②配方法

ax²+bx+c=0 → 配方为(x+p)²=q

例：x²+6x+5=0 → (x+3)²=4 → x=-3±2

③公式法（万能）

x = (-b ± √(b²-4ac)) / (2a)

（Δ=b²-4ac，详见判别式）

④因式分解法

化为(x+m)(x+n)=0 → x₁=-m, x₂=-n

例：x²-5x+6=0 → (x-2)(x-3)=0→x=2或3

5.2 根的判别式

Δ = b² - 4ac

Δ的值根的情况
Δ > 0两个不相等的实数根
Δ = 0两个相等的实数根（x₁=x₂=-b/2a）
Δ < 0没有实数根

例：x²-4x+k=0有实根⇔Δ=16-4k≥0⇔k≤4

5.3 韦达定理（根与系数的关系）

若 x₁, x₂ 是 ax²+bx+c=0 的两根，则：

x₁ + x₂ = -b/a
x₁ · x₂ = c/a

应用：①已知一根求另一根 ②求对称式（x₁²+x₂²=(x₁+x₂)²-2x₁x₂）③构造方程

📝 练习题

1用公式法解 x²-3x-4=0，解为____。
✅ A。Δ=9+16=25。x=(3±5)/2→x₁=4,x₂=-1。也可因式分解：(x-4)(x+1)=0。
2方程 2x²-x+k=0 有两个不相等实根，则k____。
✅ A。Δ=1-8k>0→k<1/8。
3若x₁+x₂=3，x₁x₂=-4，则以x₁,x₂为根的方程为____。
✅ A。x²-(x₁+x₂)x+x₁x₂=0→x²-3x-4=0。
4【中考】x₁,x₂是x²-5x+3=0的两根，求x₁²+x₂²=____。
✅ B。x₁²+x₂²=(x₁+x₂)²-2x₁x₂=25-6=19。
