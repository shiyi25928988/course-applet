1.1 数的分类

实数分类

实数
├── 有理数
│ ├── 整数（正整数、0、负整数）
│ └── 分数（有限小数、无限循环小数）
└── 无理数（无限不循环小数）
    如：π、√2、√3

重要概念

相反数：a + (-a) = 0

倒数：a × (1/a) = 1 (a≠0)

绝对值：|a| = a (a≥0)；|a| = -a (a<0)

|a| ≥ 0，|a| = |-a|

|a| = 0 当且仅当 a = 0

1.2 有理数运算

运算法则

加法：同号相加取同号，异号相加取大号

减法：减去一个数 = 加上它的相反数：a - b = a + (-b)

乘法：同号得正，异号得负

除法：除以一个数 = 乘以它的倒数（除数≠0）

a ÷ b = a × (1/b) (b ≠ 0)

运算律

交换律：a+b = b+a，ab = ba

结合律：(a+b)+c = a+(b+c)，(ab)c = a(bc)

分配律：a(b+c) = ab + ac

乘方：aⁿ = a×a×...×a（n个a）

a⁰ = 1 (a≠0)，a⁻ⁿ = 1/aⁿ

aᵐ × aⁿ = aᵐ⁺ⁿ， (aᵐ)ⁿ = aᵐⁿ

1.3 平方根与立方根

平方根

若 x² = a，则 x 叫做 a 的平方根（二次方根）。

正数 a 有两个平方根：±√a。0 的平方根是 0。

算术平方根：正数 a 的正的平方根，记作 √a。

√a² = |a|； (√a)² = a (a ≥ 0)

性质：√(ab) = √a · √b（a≥0, b≥0），√(a/b) = √a/√b（a≥0, b>0）

立方根

若 x³ = a，则 x 叫做 a 的立方根，记作 ∛a。

每个实数都有唯一的立方根。正数的立方根为正，负数的立方根为负。

∛(-a) = -∛a； (∛a)³ = a

1.4 实数

无理数与实数

无理数：无限不循环小数。如 π ≈ 3.14159...，√2 ≈ 1.414...

实数 = 有理数 ∪ 无理数。实数与数轴上的点一一对应。

科学记数法：a × 10ⁿ，其中 1 ≤ |a| < 10。

例：300000 = 3×10⁵，0.00005 = 5×10⁻⁵
