2.1 整式的运算

基本概念

单项式：数与字母的积。如 3x²，-5ab。

多项式：几个单项式的和。如 x²+2x+1。

同类项：所含字母相同且相同字母的指数也相同的项。

合并同类项：系数相加，字母部分不变。

运算法则

加减：去括号→合并同类项

单项式×多项式：用单项式乘以多项式的每一项

m(a + b + c) = ma + mb + mc

多项式×多项式：每一项分别相乘再合并

(a + b)(m + n) = am + an + bm + bn

2.2 乘法公式

平方差公式

(a + b)(a - b) = a² - b²

例：(x+3)(x-3)=x²-9；(2a+b)(2a-b)=4a²-b²

完全平方公式

(a + b)² = a² + 2ab + b² (a - b)² = a² - 2ab + b²

例：(x+5)²=x²+10x+25；(3x-2)²=9x²-12x+4

2.3 因式分解

四种方法

方法要点示例
①提公因式法找出各项公因式ma+mb=m(a+b)
②公式法平方差或完全平方x²-9=(x+3)(x-3)
③十字相乘法x²+(p+q)x+pq=(x+p)(x+q)x²+5x+6=(x+2)(x+3)
④分组分解法分组后提取公因式ax+ay+bx+by=(a+b)(x+y)

因式分解步骤

一提（提公因式）→二套（套公式）→三交叉（十字相乘）→四分组→检查（是否分解彻底）

📝 章节练习题

1计算：(2x+3)(x-1) = ____。
✅ A。=2x²-2x+3x-3=2x²+x-3。
2完全平方公式(a-b)²展开为____。
✅ C。(a-b)²=a²-2ab+b²。
3分解因式：x²-4y² = ____。
✅ A。平方差公式：x²-(2y)²=(x+2y)(x-2y)。
4分解因式：x²+7x+12 = ____。
✅ A。十字相乘：3+4=7，3×4=12→(x+3)(x+4)。
5计算：2024² - 2023² = ____。
✅ B。平方差：=(2024+2023)(2024-2023)=4047×1=4047。
