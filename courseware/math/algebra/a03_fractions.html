3.1 分式的概念与性质

定义

形如 A/B 的式子（A、B为整式，B中含有字母，B≠0）。

分式有意义的条件：分母 ≠ 0

分式值为0的条件：分子=0且分母≠0

若 A/B = 0，则 A = 0 且 B ≠ 0

基本性质

分式的分子分母同乘（除）同一个不为0的整式，分式的值不变。

A/B = (A×C)/(B×C)，A/B = (A÷C)/(B÷C) (C≠0)

约分：分子分母同除以公因式→化为最简分式。

通分：化为同分母→找最简公分母。

3.2 分式运算

乘除法

(A/B) · (C/D) = AC/BD (A/B) ÷ (C/D) = (A/B) · (D/C) = AD/BC

（除以一个分式 = 乘以它的倒数）

加减法

同分母：A/C ± B/C = (A±B)/C

异分母：先通分，再加减

a/x + b/y = (ay+bx)/(xy)

3.3 分式方程

解法步骤

①去分母（乘以最简公分母）→化为整式方程

②解整式方程

③验根：代入最简公分母，若为0则为增根，应舍去

2/(x-1) = 3/(x+1)
→ 2(x+1) = 3(x-1) → x = 5
验：x=5时分母≠0，是原方程的解

📝 章节练习题

1若分式 (x-2)/(x+3) 的值为0，则x=____。
✅ A。分子x-2=0→x=2，分母2+3≠0→符合。
2化简：(x²-4)/(x+2) = ____。(x≠-2)
✅ A。(x²-4)/(x+2)=(x+2)(x-2)/(x+2)=x-2。
3解分式方程：2/(x-1)=3/(x+1)，x=____。
✅ A。2(x+1)=3(x-1)→2x+2=3x-3→x=5。验：分母≠0✓
4计算：(1/a+1/b)÷(1/a-1/b) 的结果为____。(a≠b,a,b≠0)
✅ B。分子=(a+b)/ab，分母=(b-a)/ab→(a+b)/(b-a)。
