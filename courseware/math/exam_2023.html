一、选择题（共6题，每题4分，满分24分）
1 下列各数中，是无理数的是（ ）
✅ C。√2是无理数，其他都是有理数。
2 下列运算中，正确的是（ ）
✅ C。(a³)² = a⁶；A应为2a²；B应为a⁵；D应为a²b²。
3 下列统计量中，表示一组数据波动程度的是（ ）
✅ B。方差和标准差表示数据的波动程度。
4 已知一次函数 y = kx + b 的图像经过第一、二、三象限，那么下列结论正确的是（ ）
✅ A。经过一、二、三象限，斜率k > 0，截距b > 0。
5 下列多边形中，是中心对称图形但不是轴对称图形的是（ ）
✅ A。平行四边形是中心对称但不一定是轴对称。
6 如图，在△ABC中，D、E分别在AB、AC上，DE∥BC，AD:DB = 2:3，那么S△ADE:S△ABC等于（ ）
✅ A。AD:AB = 2:5，面积比 = 4:25。
二、填空题（共12题，每题4分，满分48分）
7 计算：(a²)³ = ______。
a⁶。幂的乘方，指数相乘：2×3 = 6。
8 已知函数 f(x) = 2x - 1，那么 f(3) = ______。
5。f(3) = 2×3 - 1 = 5。
9 不等式 2x - 6 > 0 的解集是 ______。
x > 3。2x > 6 → x > 3。
10 方程 √(2x - 1) = 3 的解是 ______。
x = 5。两边平方：2x - 1 = 9 → 2x = 10 → x = 5。
11 从1、2、3、4、5这五个数中任选一个数，选中的数是奇数的概率是 ______。
3/5。奇数有1、3、5三个，概率 = 3/5。
12 如果关于x的一元二次方程x² - 2x + k = 0有两个相等的实数根，那么k的值是 ______。
1。Δ = 4 - 4k = 0 → k = 1。
13 某公司10月份的产值是100万元，如果该公司第四季度每个月产值的增长率相同，都为x，那么12月份的产值是 ______ 万元（用含x的代数式表示）。
100(1 + x)²。11月：100(1+x)，12月：100(1+x)²。
14 已知点A(1, 3)、B(2, 5)，那么向量AB的坐标是 ______。
(1, 2)。AB = (2-1, 5-3) = (1, 2)。
15 如图，在△ABC中，D、E分别是AB、AC的中点，设向量AB = a，BC = b，那么向量DE = ______（用a、b表示）。
b/2。DE是中位线，DE = 1/2 BC = b/2。
16 小明从家出发，先以每分钟80米的速度走了10分钟，然后以每分钟120米的速度走了5分钟到达学校，那么小明从家到学校的平均速度是每分钟 ______ 米。
280/3 ≈ 93.33。总路程 = 80×10 + 120×5 = 1400米，总时间15分钟，平均速度 = 1400/15 = 280/3。
17 如图，在矩形ABCD中，AB = 3，BC = 4，E是BC边上一点，将△ABE沿AE折叠，使点B落在点F处，如果△CEF是直角三角形，那么BE的长是 ______。
3/2 或 3。分两种情况讨论直角位置。
18 定义：如果一个数的平方等于 -1，记为 i² = -1，这个数 i 叫做虚数单位。那么 (1 + i)(2 - i) = ______。
3 + i。(1+i)(2-i) = 2 - i + 2i - i² = 2 + i + 1 = 3 + i。

19 计算：√12 - 3tan30° + (π - 2023)⁰
解：√12 = 2√3，3tan30° = 3×√3/3 = √3，(π - 2023)⁰ = 1
原式 = 2√3 - √3 + 1 = √3 + 1
20 解方程组：{ 2x + y = 5，x - 2y = 0

4y + y = 5 → y = 1, x = 2

22 如图，在平面直角坐标系中，O为原点，直线 y = kx + b 经过点A(0, 3)和点B(4, 0)。
（1）求这条直线的解析式；
（2）如果点C在x轴正半轴上，且△ABC的面积为6，求点C的坐标。
（1）代入A(0,3)得b = 3，代入B(4,0)得4k + 3 = 0 → k = -3/4
∴ y = -3/4 x + 3
（2）设C(x,0)，S△ABC = 1/2 × |x - 4| × 3 = 6 → |x - 4| = 4 → x = 8（x>0）
23 某商店销售一种商品，每件成本为50元。经市场调查，售价为60元时，可销售200件，售价每提高1元，销量减少10件。设售价为x元（x ≥ 60），销售量为y件。
（1）求y与x之间的函数关系式；
（2）当售价为多少元时，该商店销售这种商品获得的利润最大？最大利润是多少？
（1）y = 200 - 10(x - 60) = -10x + 800
（2）利润 w = (x - 50)(-10x + 800) = -10x² + 1300x - 40000
= -10(x - 65)² + 2250
∴ 当x = 65时，最大利润2250元。
24 已知抛物线 y = ax² + bx + 3 经过点A(1, 0)和点B(-3, 0)。
（1）求抛物线的解析式；
（2）点P是抛物线在第一象限内的一点，且△PAB的面积为6，求点P的坐标。
（1）设y = a(x - 1)(x + 3)，代入(0,3)得-3a = 3 → a = -1
∴ y = -(x - 1)(x + 3) = -x² - 2x + 3
（2）AB = 4，S = 1/2 × 4 × y_P = 6 → y_P = 3，代入得x = 0或x = -2，无第一象限解。
（注：此题可改为y_P = -3或调整题目）
25 如图，在正方形ABCD中，E是AB边上一点，连接DE，将△DAE绕点D逆时针旋转90°得到△DCF。
（1）求证：DE = DF，且DE⊥DF；
（2）如果正方形边长为5，AE = 2，求EF的长。
（1）证明：由旋转性质，DE = DF，∠EDF = 90°，∴ DE⊥DF
（2）DE = √(5² + 2²) = √29，EF = √2 × √29 = √58。
