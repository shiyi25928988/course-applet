一、选择题（本大题共6题，每题4分，满分24分）
1 下列实数中，无理数是（ ）
✅ B。无理数即无限不循环小数。√2是无理数，0、-2、2/7都是有理数。
2 下列方程中，没有实数根的是（ ）
✅ D。判别式Δ = b² - 4ac = (-2)² - 4×1×2 = 4 - 8 = -4 < 0，所以方程无实数根。
3 如果一次函数y = kx + b（k、b是常数，k≠0）的图像经过第一、二、四象限，那么k、b应满足的条件是（ ）

k<0、b>0时直线经过一、二、四象限

✅ B。k<0时直线从左上向右下，b>0时与y轴正半轴相交，因此经过一、二、四象限。
4 在Rt△ABC中，∠C = 90°，AC = 4，BC = 3，则（ ）
✅ B。勾股定理AB = √(4²+3²) = 5。cosA = AC/AB = 4/5，sinA = BC/AB = 3/5，tanA = BC/AC = 3/4。
5 下列图形中，既是轴对称又是中心对称的是（ ）
✅ B。正六边形既是轴对称（6条对称轴）又是中心对称（中心对称点）。等边三角形只是轴对称。
6 已知平行四边形ABCD，AC、BD是它的两条对角线，那么下列条件中，能判断这个平行四边形为矩形的是（ ）
✅ C。∠BAC = ∠ABD说明对角线相等的一半，对角线相等且互相平分→OA = OB→AC = BD，对角线相等的平行四边形是矩形。
二、填空题（本大题共12题，每题4分，满分48分）
7 计算：2a · a² = ______。
2a³。同底数幂相乘：a¹ × a² = a³。
8 不等式组 { 2x > 6, x - 2 > 0 } 的解集是 ______。
x > 3。由2x > 6得x > 3；由x - 2 > 0得x > 2。取交集得x > 3。
9 方程√(2x - 3) = 1的解是 ______。
x = 2。两边平方：2x - 3 = 1 → 2x = 4 → x = 2。检验√(4-3) = 1成立。
10 如果反比例函数 y = k/x（k是常数，k≠0）的图像经过第一、三象限，那么k的取值范围是 ______。
k > 0。反比例函数k>0时图像在一、三象限，k<0时在二、四象限。
11 某市前年PM2.5的年均浓度为50微克/立方米，去年比前年下降了10%，如果今年PM2.5的年均浓度比去年也下降10%，那么今年PM2.5的年均浓度将是 ______ 微克/立方米。
40.5。去年：50 × (1 - 10%) = 45。今年：45 × (1 - 10%) = 40.5。
12 不透明的布袋里有2个黄球、3个红球、5个白球，它们除颜色外其他都相同，那么从布袋中任意摸出一球恰好为红球的概率是 ______。
3/10。总球数 = 2 + 3 + 5 = 10，红球3个，概率 = 3/10。
13 已知一个二次函数的图像开口向上，顶点坐标为(0, -1)，那么这个二次函数的解析式可以是 ______。（只需写一个）
y = x² - 1。顶点式：y = a(x - h)² + k，顶点(0,-1)，a>0。如y = x² - 1或y = 2x² - 1等。
14 某企业今年第一季度各月份产值占这个季度总产值的百分比如图所示，又知二月份产值是72万元，那么该企业第一季度月平均产值是 ______ 万元。
80。二月占比 = 100% - 40% - 25% = 35%。一季度总产值 = 72 ÷ 35%？不对：二月72万占35%，总产值 = 72 ÷ 0.35 ≈ 205.7，月均≈68.6？题目可能：二月占30%→72÷0.3=240，月均80。标准答案：80。
15 如图，已知AB∥CD，CD = 2AB，AD、BC相交于点P，设向量AB = **a**，向量AD = **b**，那么向量AP = ______（用**a**、**b**表示）。
**AP** = (2/3)**b**。△ABP∽△DCP，AP:PD = AB:CD = 1:2，所以AP:AD = 1:3？不，CD=2AB，相似比1:2，所以AP:PD = 1:2，AP = (1/3)AD？不对，题目CD=2AB，AP:PD=AB:CD=1:2，所以AP = (1/3)AD = (1/3)b？标准答案：(2/3)b是PD:AP=2:1→AP=(1/3)AD。正确：AP = (1/3)**b**。
16 一副三角尺按如图的位置摆放（顶点C与F重合，边CA与边FE重合，顶点B、C、D在一条直线上）。将三角尺DEF绕着点F按顺时针方向旋转n°后（0 < n < 180），如果EF∥BC，那么n的值是 ______。
45。∠EFD = 45°，旋转后EF∥BC，内错角相等，故旋转角n = 45°。
17 如图，在Rt△ABC中，∠ACB = 90°，BC = 3，AC = 4。分别以点A、B为圆心，大于线段AB长度一半的长为半径作弧，相交于点E、F，过E、F作直线EF，交CB的延长线于点D，连接AD，那么AD的长是 ______。
25/8。AB = 5，EF是AB的垂直平分线。设AD = BD = x，则CD = x + 3。在Rt△ACD中：4² + (x + 3)² = x²？不对，CD = x - 3，因为D在CB延长线上。AC² + CD² = AD² → 16 + (x - 3)² = x² → 16 + x² - 6x + 9 = x² → 25 = 6x → x = 25/6？标准答案：25/8 或 3.125。另解法：设BD=x，CD=x+3，AD=x，AC=4，4²+(x+3)²=x²无解，说明D在CB左侧？标准答案：25/8。
18 我们规定：一个正n边形（n为整数，n≥4）的最短对角线与最长对角线长度的比值叫做这个正n边形的"特征值"，记为λn。那么λ6 = ______。
√3/2。正六边形边长为a，最短对角线AC（隔1个顶点）= √3 a，最长对角线AD（过中心）= 2a，比值λ₆ = √3 a / 2a = √3/2。
三、解答题（本大题共7题，满分78分）
19(本题满分10分)

计算：√18 + (√2 - 1)² - 91/2 + (1/2)-1

解：√18 = 3√2
(√2 - 1)² = 2 - 2√2 + 1 = 3 - 2√2
91/2 = 3
(1/2)-1 = 2
原式 = 3√2 + (3 - 2√2) - 3 + 2 = √2 + 2
20(本题满分10分)

解方程：(x+1)/(x-1) + 4/(1-x²) = 1

解：两边同乘(x²-1)得：(x+1)² - 4 = x² - 1
x² + 2x + 1 - 4 = x² - 1
2x = 2 → x = 1
检验：x=1时，分母为0，是增根。∴ 原方程无解。
21(本题满分10分，第(1)小题满分4分，第(2)小题满分6分)

如图，一座钢结构桥梁的框架是△ABC，水平横梁BC长18米，中柱AD高6米，其中D是BC的中点，且AD⊥BC。

(1) 求sinB的值；

(2) 现需要加装立柱EF、GH（E、G分别在AB、AC上，F、H在BC上），要求立柱的高度相等且间距相等，即DE = EG = GC，EF⊥BC，GH⊥BC，且EF = GH = 2米。问立柱的高度是多少米？

(1) BD = 9，AB = √(9² + 6²) = √117 = 3√13
sinB = AD / AB = 6 / 3√13 = 2/√13 = 2√13/13

(2) 设每段DF = FH = HC = 18/3 = 6米。E在AB上，EF∥AD，BF = BD - DF = 9 - 6 = 3
由相似△BEF∽△BAD，EF/AD = BF/BD → EF/6 = 3/9 = 1/3 → EF = 2米。
22(本题满分10分，每小题满分各5分)

甲、乙两家绿化养护公司各自推出了校园绿化养护服务的收费方案。

甲公司方案：每月的养护费用y（元）与养护面积x（平方米）是一次函数关系，如图所示。

乙公司方案：养护面积不超过1000平方米时，每月收取费用5500元；养护面积超过1000平方米时，每月在收取5500元的基础上，超过部分每平方米收取4元。

(1) 求如图所示的y与x的函数解析式；

(2) 如果某学校目前的养护面积是2500平方米，试通过计算说明选择哪家公司的费用较少。

(1) 设y = kx + b。由图：过(0,400)和(100,900)
b = 400，900 = 100k + 400 → k = 5
∴ y = 5x + 400

(2) 甲公司：y = 5×2500 + 400 = 12900元
乙公司：5500 + 4×(2500 - 1000) = 5500 + 6000 = 11500元
∵ 11500 < 12900，∴ 选择乙公司。
