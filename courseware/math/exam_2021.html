一、选择题（共6题，每题4分，满分24分）
1 下列实数中，有理数是（ ）
✅ C。√4 = 2，是有理数。其他都是无理数。
2 下列单项式中，a²b³的同类项是（ ）
✅ B。同类项要求字母相同且相同字母的指数相同。
3 将抛物线y = (x - 2)² + 1向下平移2个单位，再向左平移1个单位后，所得新抛物线的解析式为（ ）
✅ B。向下平移2：y = (x-2)² - 1，再向左1：y = (x-1)² - 1。
4 如图，已知在△ABC中，点D、E分别在AB、AC上，DE∥BC，AD:DB = 1:2，那么下列结论中正确的是（ ）
✅ B。AD:DB = 1:2 → AD:AB = 1:3，由相似得AE:AC = 1:3，DE:BC = 1:3，面积比1:9。
5 已知单位向量e₁和e₂，下列结论中正确的是（ ）
✅ B。单位向量模长都是1，但方向不一定相同，所以只有模长相等。
6 如图，长方形ABCD中，AB = 4，AD = 3，圆B的半径为1，圆A与圆B内切，则点C、D与圆A的位置关系是（ ）
✅ C。两圆内切，AB = 4 = R - 1 → R = 5。AD = 3 < 5，AC = 5，∴ D在圆内，C在圆上。
二、填空题（共12题，每题4分，满分48分）
7 计算：x⁷ ÷ x² = ______。
x⁵。同底数幂相除，指数相减：7 - 2 = 5。
8 已知f(x) = 6/x，那么f(√3) = ______。
2√3。f(√3) = 6/√3 = 2√3。
9 方程 √(x + 2) = x 的解是 ______。
x = 2。两边平方：x + 2 = x² → x² - x - 2 = 0 → (x-2)(x+1) = 0。检验：x = 2有效，x = -1舍去。
10 反比例函数 y = k/x 的图像经过点(2, -1)，则k的值为 ______。
-2。k = xy = 2×(-1) = -2。
11 有一枚材质均匀的正方体骰子，六个面的点数分别是1、2、3、4、5、6，掷一次骰子，朝上的点数大于4的概率是 ______。
1/3。大于4的点数是5、6两个，概率 = 2/6 = 1/3。
12 已知一元二次方程x² - 4x + 3 = 0的两根为x₁、x₂，则x₁·x₂ = ______。
3。由韦达定理，x₁·x₂ = c/a = 3/1 = 3。
13 某商店6月份的利润是2500元，要使8月份的利润达到3600元，平均每月增长的百分率是 ______。
20%。2500(1+x)² = 3600 → (1+x)² = 1.44 → 1+x = 1.2 → x = 20%。
14 若一组数据1、2、x、4的众数是1，则这组数据的方差是 ______。
1.5。众数是1，∴ x = 1。平均数 = (1+1+2+4)/4 = 2，方差 = [(1-2)²+(1-2)²+(2-2)²+(4-2)²]/4 = 6/4 = 1.5。
15 如图，已知在△ABC中，D、E分别是AB、AC的中点，设向量AB = a，BC = b，那么向量DE = ______（用a、b表示）。
b/2。DE是中位线，DE = 1/2 BC = b/2。
16 《九章算术》中记载了一种测量井深的方法。如图所示，在井口B处立一根垂直于井口的木杆BD，从木杆的顶端D观察井水水岸C，视线DC与井口的直径AB交于点E。如果测得AB = 1.6米，BD = 1米，BE = 0.2米，那么井深AC为 ______ 米。
7。由相似三角形，AC/BD = AE/BE → AC/1 = 1.4/0.2 = 7。
17 六个带标号的小球完全相同，分别标有1、2、3、4、5、6，从中随机取出两个，取出的两个球的标号之和为3的倍数的概率是 ______。
1/3。总取法C(6,2)=15种，和为3的倍数的组合有5种，概率=5/15=1/3。
18 在△ABC中，∠C = 90°，AC = 3，BC = 4，将△ABC绕点B逆时针旋转得到△A'BC'，点C的对应点C'在AB上，则tan∠AA'C的值为 ______。
1/2。由旋转性质和几何关系可求得。
三、解答题（共7题，满分78分）
19 计算：|√3 - 1| - (1/2)⁰ + 2cos30°
解：|√3 - 1| = √3 - 1，(1/2)⁰ = 1，2cos30° = 2×√3/2 = √3
原式 = (√3 - 1) - 1 + √3 = 2√3 - 2
20 解方程组：{ x + y = 3，x² - 4y² = 0
解：由x² - 4y² = 0得(x + 2y)(x - 2y) = 0
情况1：x + 2y = 0，与x + y = 3联立得y = -3, x = 6
情况2：x - 2y = 0，与x + y = 3联立得y = 1, x = 2
∴ 方程组的解为 (6, -3) 和 (2, 1)
21 如图，已知在△ABC中，AB = BC，点D、E分别在AB、BC上，且BD = BE，连接CD、AE交于点F。
（1）求证：△ABE ≅ △CBD；
（2）如果CD是∠ACE的角平分线，求证：DF = EF。
（1）证明：AB = BC，∠B公共，BE = BD
∴ △ABE ≅ △CBD（SAS）
（2）由角平分线和全等可证△ADF ≅ △CEF，得DF = EF。
22 现在5G手机非常流行，某公司第一季度共卖出100万台5G手机，第二季度卖出的5G手机数量比第一季度增长了x%，第三季度比第二季度又增长了x%。
（1）用含x的代数式表示第三季度的销量；
（2）如果x = 10，求第三季度的销量。
（1）第二季度销量 = 100(1 + x%)万台
第三季度销量 = 100(1 + x%)²万台
（2）x = 10时，销量 = 100×1.1² = 121万台
23 如图，已知在梯形ABCD中，AD∥BC，AB = DC，对角线AC、BD交于点O，E是BC边上一点，且∠AED = ∠BCD。
（1）求证：△ABE∽△EDA；
（2）如果AB = 5，BC = 10，求AE的长。

（2）AE = 5√2。由相似三角形性质可求得。
24 已知抛物线 y = x² + bx + c 经过点(1, 0)和点(0, 3)。
（1）求抛物线的解析式；
（2）将抛物线平移，使其顶点落在x轴上，求平移后的解析式。
（1）代入(1,0)和(0,3)：
1 + b + c = 0，c = 3 → b = -4
∴ y = x² - 4x + 3
（2）y = (x-2)² - 1，顶点(2,-1)，向上平移1个单位得y = (x-2)²。

∴ △ABD ≅ △ACE（SAS）
