📐 计算题
1 运动学问题 （10分）

一质点从静止开始做直线运动，第一秒内加速度为 a，第二秒内加速度为 -a，第三秒内加速度为 a，第四秒内加速度为 -a，如此反复。求 100 秒末质点的速度和位移。

答案：速度 v = 0；位移 s = 50a
解析：
每两秒为一个周期：
- 第1秒末速度：v₁ = a×1 = a
- 第2秒末速度：v₂ = a - a×1 = 0
- 一个周期（2秒）内的位移：s = ½a×1² + (a×1 - ½a×1²) = ½a + ½a = a

100秒共有 50 个周期，所以：
速度：v = 0（每个周期末速度都回到0）
位移：s = 50a
2 能量守恒问题 （10分）

质量为 m 的小球从高度 h 处由静止释放，落入沙坑，进入深度为 d 后停止。求沙坑对小球的平均阻力。（忽略空气阻力）

答案：f = mg(h+d)/d
解析：
对整个过程用动能定理：
mg(h+d) - f d = 0 - 0
（重力做功 mg(h+d)，阻力做功 -f d，初末动能都为0）

解得：f = mg(h+d)/d
3 静电场 （15分）

两个点电荷 +q 和 -q 相距为 d，构成电偶极子。求在两电荷中垂线上、距离中点 r 处（r>>d）的电场强度。

答案：E = kqd/r³ 方向沿电偶极矩反方向
解析：
电偶极矩 p = qd
两电荷在该点的电场矢量叠加，垂直分量抵消，水平分量相加：
E = 2×(kq/r²)×(d/2r) = kqd/r³ = kp/r³
方向沿电偶极矩的反方向
4 电路计算 （12分）

如图所示电路，电源电动势 E = 12V，内阻 r = 1Ω，R₁ = 2Ω，R₂ = 3Ω，R₃ = 6Ω。求通过 R₃ 的电流。

答案：I₃ = 1A
解析：
R₂与R₃并联：R23 = (3×6)/(3+6) = 2Ω
总电阻：R总 = R₁ + R23 + r = 2 + 2 + 1 = 5Ω
总电流：I = E/R总 = 12/5 = 2.4A
并联部分电压：U23 = I×R23 = 2.4×2 = 4.8V
I₃ = U23/R₃ = 4.8/6 = 0.8A
5 热力学第一定律 （15分）

一定质量的理想气体从状态 A 经等压过程到状态 B，体积加倍；再经等容过程到状态 C，压强减半；最后经等温过程回到状态 A。求整个循环中气体对外做的功。（设 A 状态：p, V, T）

答案：W = pV(1 - 2ln2)
解析：
A→B 等压过程：W₁ = p(2V - V) = pV
B→C 等容过程：W₂ = 0
C→A 等温过程：W₃ = -∫pdV = -2pVln2
总功：W = W₁ + W₂ + W₃ = pV(1 - 2ln2)
6 光电效应 （15分）

用波长 λ = 400nm 的光照射某金属，测得遏止电压为 1V。求该金属的逸出功和截止频率。

答案：W₀ ≈ 2.1eV, ν₀ ≈ 5.1×10¹⁴Hz
解析：
爱因斯坦光电方程：½mv² = hν - W₀ = eUc
光子能量：E = hc/λ ≈ (1240eV·nm)/(400nm) = 3.1eV
逸出功：W₀ = E - eUc = 3.1 - 1 = 2.1eV
截止频率：ν₀ = W₀/h ≈ 5.1×10¹⁴Hz
