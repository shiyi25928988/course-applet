📚 物理竞赛知识点总结

竞赛核心知识点、重要公式、推荐书目全面梳理

📐 力学

1. 运动学

  • 质点运动的描述（位矢、速度、加速度）
  • 相对运动、牵连运动
  • 平面极坐标系、自然坐标系
  • 抛体运动、圆周运动

2. 牛顿运动定律

  • 惯性系与非惯性系
  • 惯性力（离心力、科里奥利力）
  • 连接体问题、约束条件
  • 临界问题分析

3. 动量与能量

  • 动量定理、动量守恒定律
  • 动能定理、机械能守恒
  • 功能原理、能量守恒
  • 碰撞问题（弹性、非弹性、完全非弹性）

4. 角动量

  • 角动量定理、角动量守恒
  • 有心力场、开普勒定律
  • 天体运动、宇宙速度
  • 二体问题、约化质量

5. 刚体力学

  • 转动惯量计算（平行轴、垂直轴定理）
  • 定轴转动定律
  • 刚体的平面平行运动
  • 纯滚动、进动

6. 振动与波

  • 简谐运动（微分方程、能量）
  • 阻尼振动、受迫振动、共振
  • 简正模、耦合振动
  • 机械波、多普勒效应、冲击波
刚体转动惯量：I = Σmᵢrᵢ²    平行轴定理：I = I_c + md²    角动量：L = r × p

⚡ 电磁学

1. 静电场

  • 库仑定律、电场强度
  • 高斯定理及其应用
  • 电势、电势能、电势差
  • 电像法、静电场唯一性定理
  • 电介质、极化强度、电位移矢量

2. 稳恒电流

  • 欧姆定律、电阻的串并联
  • 基尔霍夫定律
  • 复杂直流电路（节点电压、回路电流）
  • 戴维南定理、诺顿定理
  • Δ-Y变换

3. 稳恒磁场

  • 毕奥-萨伐尔定律
  • 安培环路定理
  • 安培力、洛伦兹力
  • 带电粒子在磁场中运动
  • 磁介质、磁化强度、H矢量

4. 电磁感应

  • 法拉第电磁感应定律
  • 楞次定律、动生电动势、感生电动势
  • 自感、互感、磁场能量
  • 暂态过程（RC、RL、RLC电路）

5. 交流电

  • 正弦交流电的描述
  • 交流电路的阻抗、相位
  • 交流电功率、功率因数
  • 变压器原理、三相交流电
  • 谐振电路（串联、并联谐振）

6. 电磁波

  • 位移电流
  • 麦克斯韦方程组
  • 电磁波的产生与传播
  • 电磁波的能量和动量
麦克斯韦方程组：∮E·dS = q/ε₀    ∮B·dS = 0    ∮E·dl = -dΦ/dt    ∮B·dl = μ₀(I + ε₀dΦ_E/dt)

🌡️ 热学

1. 分子动理论

  • 理想气体状态方程
  • 压强的微观解释
  • 麦克斯韦速度分布律
  • 玻尔兹曼分布
  • 能量按自由度均分定理

2. 热力学第一定律

  • 功、热量、内能
  • 热力学第一定律表达式
  • 准静态过程（等容、等压、等温、绝热）
  • 多方过程
  • 热机、制冷机

3. 热力学第二定律

  • 两种表述（开尔文、克劳修斯）
  • 卡诺循环、卡诺定理
  • 熵的定义、熵增加原理
  • 热力学概率、玻尔兹曼熵公式
  • 自由能、热力学势

4. 气体与相变

  • 真实气体、范德瓦尔斯方程
  • 平均自由程、输运过程
  • 气液相变、克拉佩龙方程
  • 相变潜热、蒸气凝结
  • 热传递：传导、对流、辐射
理想气体状态方程：PV = νRT    卡诺效率：η = 1 - T₂/T₁    玻尔兹曼熵：S = k lnΩ

🔬 光学

1. 几何光学

  • 费马原理
  • 反射定律、折射定律
  • 球面镜成像、薄透镜成像
  • 透镜组、光具组
  • 光学仪器（望远镜、显微镜）
  • 变折射率光学

2. 波动光学

  • 光波的叠加原理
  • 双缝干涉、薄膜干涉
  • 劈尖、牛顿环、迈克尔逊干涉仪
  • 法布里-珀罗干涉
  • 单缝衍射、圆孔衍射
  • 光栅衍射、X射线衍射

3. 偏振光学

  • 自然光与偏振光
  • 马吕斯定律
  • 布儒斯特定律
  • 双折射现象（o光、e光）
  • 波片、偏振光干涉
  • 旋光性
透镜成像公式：1/u + 1/v = 1/f    双缝干涉：Δx = λD/d    光栅方程：d sinθ = kλ

⚛️ 近代物理

1. 狭义相对论

  • 伽利略变换、经典时空观
  • 狭义相对论基本假设
  • 洛伦兹变换
  • 相对论运动学（长度收缩、时间膨胀）
  • 相对论动力学（质速关系、质能关系）
  • 相对论能量动量关系

2. 原子物理

  • 氢原子光谱、玻尔模型
  • 能级跃迁、选择定则
  • 夫兰克-赫兹实验
  • 电子自旋、泡利不相容原理
  • 原子的壳层结构
  • X射线、康普顿散射

3. 原子核物理

  • 原子核的组成与性质
  • 放射性衰变（α、β、γ衰变）
  • 衰变规律、半衰期
  • 核反应、核裂变、核聚变
  • 结合能、质量亏损

4. 量子力学初步

  • 波粒二象性、德布罗意波
  • 不确定关系
  • 波函数、薛定谔方程
  • 一维定态问题（无限深势阱、势垒穿透）
  • 隧道效应
洛伦兹变换：x' = γ(x-vt)    质能方程：E = mc²    德布罗意波长：λ = h/p
