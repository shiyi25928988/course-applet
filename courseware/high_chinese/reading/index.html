📖 现代文阅读专项（36分）

论述类 + 实用类 + 文学类 · 三大阅读模块 · 答题模板 + 真题训练

📕 论述类文本阅读（9分）

🎯 核心考点

  • 理解文中重要概念的含义
  • 理解文中重要句子的含意
  • 筛选并整合文中信息
  • 分析文章结构，归纳内容要点
  • 分析论点、论据和论证方法
  • 分析概括作者在文中的观点态度

📝 常见设题陷阱

偷换概念：选项中的词语与原文概念内涵不同

无中生有：选项内容原文没有涉及

因果倒置：选项把原因和结果说反了

曲解文意：选项对原文的理解有偏差

以偏概全：选项把部分说成整体

绝对化表述："一定""必然""只要...就..."等

📝 论证思路题答题模板

本文首先提出了……的中心论点，然后从……几个方面/用……的论证方法展开论述， 最后总结全文，得出……的结论/强调……的观点。

📝 论证方法题答题模板

运用了……的论证方法，举了……的例子/引用了……的名言/将……与……进行对比， 具体有力/生动形象地论证了……的观点，增强了文章的说服力。

📗 实用类文本阅读（12分）

🎯 核心考点

  • 非连续性文本阅读（多则材料组合）
  • 图表信息筛选与转换
  • 新闻、传记、报告、科普文章阅读
  • 比较不同材料的侧重点
  • 探究文本中的某些问题，提出自己的见解

📝 图表题答题技巧

看标题：明确图表说明的对象和内容

看纵横：理清图表的横向和纵向项目

看数据：注意数据的变化趋势和极值

看注释：不要忽略图表下方的注释说明

📝 材料比较题答题模板

材料一侧重于从……角度介绍/报道……，内容包括……； 材料二侧重于从……角度介绍/报道……，内容包括……。 两则材料的共同点是……，不同点是……。

📘 文学类文本阅读（15分）

🎯 核心考点

  • 小说三要素：人物、情节、环境
  • 散文的形散神聚特点
  • 重要语句的丰富含意与品味
  • 分析作品的体裁特征和表现手法
  • 鉴赏作品的文学形象，领悟作品的艺术魅力
  • 对作品进行个性化阅读和有创意的解读

📝 人物形象分析题答题模板

×××是一个……（概括性评价）的人物形象。

①他……（具体表现1），体现了他……的性格特点；

②他……（具体表现2），体现了他……的性格特点；

③他……（具体表现3），体现了他……的性格特点。

总结：这个人物形象反映了……/具有……的社会意义。

📝 句子赏析题答题模板

这句话运用了……的修辞手法/表现手法/描写方法，……（具体分析手法的运用）， 生动形象/准确传神地写出了……（内容），表达了作者/人物……的情感， 在结构上起到了……的作用（照应题目、承上启下、总结全文等）。

📝 标题作用题答题模板

①内容上：概括了文章的主要内容/点明了文章的中心事件；

②主题上：点明/暗示了文章的主旨，抒发了作者……的情感；

③结构上：是文章的线索，贯穿全文；

④手法上：运用了……的手法，新颖别致，吸引读者阅读兴趣；

⑤人物上：突出了人物……的形象特点。

📝 环境描写作用题答题模板

②烘托了人物……的心情/形象；

③推动了故事情节的发展，为下文……作铺垫；

④暗示了社会环境/时代背景；

⑤深化了文章……的主题。
